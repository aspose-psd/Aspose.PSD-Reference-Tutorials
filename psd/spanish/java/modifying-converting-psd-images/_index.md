- **¿Qué hace “add diagonal watermark”?** Superpone una marca de agua de texto o imagen inclinada a lo largo de todo el lienzo PSD.

- **¿Necesito una licencia?** Una prueba gratuita funciona para evaluación; se requiere una licencia comercial para uso en producción.

- **¿Es la API compatible con Java 8+?** Absolutamente, soporta versiones modernas de Java y se ejecuta en cualquier JVM.

## ¿Qué es agregar una marca de agua diagonal?
Agregar una marca de agua diagonal significa colocar un texto o imagen semitransparente en un ángulo de 45 grados de modo que abarque toda la composición. Esta técnica disuade el uso no autorizado mientras mantiene visible el diseño subyacente.

## ¿Por qué usar Aspose.PSD for Java?
- **Fidelidad total de PSD:** Conserva la información de capas, máscaras y ajustes.  
- **Procesamiento del lado del servidor:** No necesitas una GUI; automatiza trabajos por lotes en servicios back‑end.  
- **Suite de ajustes rica:** Combina la marca de agua con capas de tono‑saturación, capas de relleno de color, ajustes de curvas y cambios de modo de fusión.

## Preguntas frecuentes

**Q: ¿Puedo agregar una marca de agua diagonal a un PSD que ya contiene capas de ajuste?**  
A: Sí. Aspose.PSD te permite apilar una marca de agua sobre capas existentes, incluidas hue saturation, curves y capas de relleno de color, sin aplanar el archivo.

**Q: ¿Es posible procesar por lotes varios archivos PSD para agregar la misma marca de agua?**  
A: Absolutamente. Puedes iterar sobre un directorio, cargar cada PSD, aplicar la marca de agua y guardar el resultado, todo dentro de un solo programa Java.

**Q: ¿Los tutoriales de capas de ajuste (hue saturation, curves, color fill) afectan la visibilidad de la marca de agua?**  
A: Las capas de ajuste afectan los datos de imagen subyacentes pero no la capa de marca de agua rasterizada si agregas la marca de agua después de esos ajustes. También puedes colocar la marca de agua en su propia capa para un control total.

**Q: ¿Qué consideraciones de memoria debo tener en cuenta al trabajar con archivos PSD grandes?**  
A: Utiliza los controles de reasignación de caché descritos en el tutorial “Control Cache Reallocation” para limitar el uso de RAM, especialmente al procesar archivos de alta resolución por lotes.

**Q: ¿Qué versiones de Aspose.PSD son compatibles con Java 11 y posteriores?**  
A: Todas las versiones recientes (2023‑2025) soportan Java 8+, incluyendo Java 11, 17 y posteriores. Consulta las notas de la versión del producto para el número exacto que estés usando.
