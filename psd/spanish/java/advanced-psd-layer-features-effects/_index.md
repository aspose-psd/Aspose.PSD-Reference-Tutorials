## Preguntas frecuentes

**P:** ¿Necesito una licencia de pago para usar Aspose.PSD for Java?  
**R:** Una prueba gratuita está disponible para desarrollo y pruebas, pero se requiere una licencia comercial para implementaciones en producción.

**P:** ¿Puedo aplicar múltiples efectos, como sombra interna y resplandor externo, a la misma capa?  
**R:** Sí. Los efectos pueden apilarse añadiéndolos secuencialmente a la colección de efectos de la capa.

**P:** ¿Es posible editar un archivo PSD sin perder la información de capas?  
**R:** Absolutamente. Aspose.PSD conserva todos los metadatos de capa, máscaras y capas de ajuste al guardar los cambios.

**P:** ¿Qué versiones de Java son compatibles con la última biblioteca Aspose.PSD?  
**R:** Java 8, 11 y versiones LTS más recientes son totalmente compatibles.

**P:** ¿Cómo manejo archivos PSD grandes para evitar problemas de memoria?  
**R:** Utiliza la sobrecarga `Load` de la clase `PsdImage` con un `FileStream` y habilita la carga diferida (lazy loading) donde sea apropiado.
