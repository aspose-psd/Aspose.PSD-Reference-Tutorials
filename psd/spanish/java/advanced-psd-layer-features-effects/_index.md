## Preguntas frecuentes

**Q: ¿Necesito una licencia paga para usar Aspose.PSD for Java?**  
A: Hay una prueba gratuita disponible para desarrollo y pruebas, pero se requiere una licencia comercial para implementaciones en producción.

**Q: ¿Puedo aplicar múltiples efectos, como sombra interna y resplandor externo, a la misma capa?**  
A: Sí. Los efectos pueden apilarse añadiéndolos secuencialmente a la colección de efectos de la capa.

**Q: ¿Es posible editar un archivo PSD sin perder la información de capas?**  
A: Absolutamente. Aspose.PSD preserva todos los metadatos de capa, máscaras y capas de ajuste cuando guardas los cambios.

**Q: ¿Qué versiones de Java son compatibles con la última biblioteca Aspose.PSD?**  
A: Java 8, 11 y versiones LTS más recientes son totalmente compatibles.

**Q: ¿Cómo manejo archivos PSD grandes para evitar problemas de memoria?**  
A: Usa la sobrecarga `Load` de la clase `PsdImage` con un `FileStream` y habilita la carga diferida (lazy loading) donde sea apropiado.
