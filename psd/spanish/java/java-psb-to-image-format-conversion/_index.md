## ¿Qué es PSB y por qué convertirlo?
PSB (Photoshop Big) es el formato de Adobe para archivos que superan el límite de 2 GB de un PSD normal. Diseños a gran escala, composiciones de alta resolución o trabajos con múltiples capas suelen usar PSB. Convertir PSB a formatos más ligeros como JPG o PDF facilita compartir, incrustar en páginas web o archivar sin perder información visual esencial.

## ¿Por qué usar Aspose.PSD para Java?

- **Renderizado de alto rendimiento** – optimizado para archivos grandes, ideal para PSB.  
- **Compatibilidad multiplataforma** – una llamada a la API puede exportar a JPG, PDF, PNG, TIFF y más.  
- **Sin dependencia nativa de Photoshop** – funciona en cualquier entorno Java del lado del servidor.

## Preguntas frecuentes

**Q: ¿Puedo convertir un archivo PSB protegido con contraseña?**  
A: Sí. Aspose.PSD permite abrir archivos PSB encriptados proporcionando la contraseña durante la operación de carga.

**Q: ¿La conversión de PSB a JPG pierde información de capas?**  
A: Convertir a JPG aplana la imagen, por lo que los datos de capas se fusionan. Usa PDF o PSD si necesitas conservar las capas.

**Q: ¿Existe un límite de tamaño para los archivos PSB al usar Aspose.PSD?**  
A: La biblioteca está diseñada para manejar archivos PSB muy grandes (varios gigabytes) siempre que tu JVM disponga de suficiente memoria heap.

**Q: ¿Necesito una licencia separada para cada formato de salida?**  
A: No. Una única licencia de Aspose.PSD for Java cubre todos los formatos de exportación compatibles, incluidos JPG, PDF y PSD.

**Q: ¿Cómo mejoro el rendimiento al convertir muchos archivos PSB en lote?**  
A: Reutiliza la misma instancia de `PsdImage` cuando sea posible y considera procesar los archivos en flujos paralelos con configuraciones de memoria adecuadas.
