## Domande Frequenti

**D: Posso usare Aspose.PSD per applicazioni Java commerciali?**  
R: Sì, con una licenza Aspose valida. È disponibile una prova gratuita per la valutazione.

**D: La libreria supporta file AI con font incorporati?**  
R: Assolutamente. Aspose.PSD legge i font incorporati e preserva il rendering del testo quando possibile.

**D: Cosa fare se devo convertire un grande batch di file AI?**  
R: Usa un ciclo per caricare ogni file e chiamare il metodo `save`. La libreria è ottimizzata per l’elaborazione batch.

**D: Esistono limitazioni di dimensione per i file AI?**  
R: La libreria gestisce file fino a diverse centinaia di megabyte, limitati solo dalla dimensione dell’heap JVM.

**D: Come preservare la trasparenza quando converto in PNG?**  
R: Assicurati di utilizzare le `PngOptions` con `ColorType = PngColorType.TrueColorWithAlpha`.
