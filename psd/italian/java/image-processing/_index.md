## Risposte rapide
- **Che cos'è un filtro mediano?** Un filtro non lineare che sostituisce ogni pixel con il valore mediano dei pixel vicini, riducendo efficacemente il rumore mantenendo i bordi.  
- **Perché scegliere Aspose.PSD per Java?** Fornisce un'API completa per PSD, PNG, JPEG e molti altri formati senza richiedere librerie native.  
- **Posso combinare il filtro mediano con altri filtri?** Sì—mescola il filtro mediano con filtri Gaussiani o Wiener per affinare la riduzione del rumore e la conservazione dei dettagli.  
- **È necessaria una licenza?** È disponibile una prova gratuita; è richiesta una licenza commerciale per l'uso in produzione.  
- **Quali versioni di Java sono supportate?** Java 8 e successive, inclusi Java 17 LTS.

## Domande frequenti

**D: Posso usare il filtro mediano su file PSD di grandi dimensioni senza esaurire la memoria?**  
R: Sì. Aspose.PSD elabora i livelli in modo sequenziale e offre opzioni di streaming per mantenere basso l'utilizzo della memoria.

**D: In che modo un filtro mediano differisce da un filtro Gaussiano?**  
R: Un filtro mediano è non lineare e eccelle nella rimozione del rumore sale‑e‑pepe preservando i bordi, mentre un filtro Gaussiano è lineare e sfuma uniformemente.

**D: È possibile combinare **apply gaussian filter** e filtraggio mediano in un unico pipeline?**  
R: Assolutamente. Puoi concatenare i filtri—applica prima una sfocatura Gaussiana per ridurre il rumore ad alta frequenza, poi un filtro mediano per una pulizia che preserva i bordi.

**D: Aspose.PSD supporta operazioni **crop psd file** su oggetti smart?**  
R: Sì. Il ritaglio funziona su qualsiasi livello raster, inclusi quelli incorporati negli oggetti smart.

**D: Quale versione di Java è richiesta per questi tutorial?**  
R: È richiesta Java 8 o successiva; la libreria è pienamente compatibile con Java 11, 17 e le successive versioni LTS.
