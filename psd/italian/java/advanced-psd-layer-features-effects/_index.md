## Domande frequenti

**Q:** Devo avere una licenza a pagamento per usare Aspose.PSD per Java?  
**A:** È disponibile una prova gratuita per sviluppo e test, ma è necessaria una licenza commerciale per le distribuzioni in produzione.

**Q:** Posso applicare più effetti, come ombra interna e bagliore esterno, allo stesso livello?  
**A:** Sì. Gli effetti possono essere impilati aggiungendoli sequenzialmente alla collezione di effetti del livello.

**Q:** È possibile modificare un file PSD senza perdere le informazioni dei livelli?  
**A:** Assolutamente. Aspose.PSD preserva tutti i metadati dei livelli, le maschere e i livelli di regolazione quando si salvano le modifiche.

**Q:** Quali versioni di Java sono supportate dall'ultima libreria Aspose.PSD?  
**A:** Java 8, 11 e le versioni LTS più recenti sono pienamente supportate.

**Q:** Come gestire file PSD di grandi dimensioni per evitare problemi di memoria?  
**A:** Usa la sovraccarico `Load` della classe `PsdImage` con un `FileStream` e abilita il caricamento lazy dove appropriato.
