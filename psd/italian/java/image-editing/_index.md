## Risposte rapide
- **Qual è il modo principale per unire le immagini in Java?** Utilizzate l'API `PsdImage` di Aspose.PSD per combinare livelli o dati raster in poche righe di codice.  
- **È necessaria una licenza per Aspose.PSD?** Una prova gratuita è sufficiente per la valutazione; è richiesta una licenza per l'uso in produzione.  
- **Quale versione di Java è supportata?** Java 8 e successive sono pienamente supportate.  
- **Posso aggiungere una sovrapposizione durante l'unione delle immagini?** Sì—applicate l'effetto Color Overlay prima o dopo l'unione per risultati vivaci.  
- **La creazione dei metadati è gestita separatamente?** Potete creare metadati XMP nello stesso flusso di lavoro dopo aver unito le immagini.

`PsdImage` è la classe principale che rappresenta un documento Photoshop all'interno della libreria Aspose.PSD.

## Cos'è **come unire le immagini** con Aspose.PSD?
Unire le immagini significa combinare due o più risorse visive in un unico file PSD o in un output raster. Aspose.PSD per Java fornisce metodi di alto livello per stratificare le immagini, fondere i livelli e preservare tutte le funzionalità compatibili con Photoshop senza perdita di qualità.

## Perché usare Aspose.PSD per l'unione di immagini in Java?
Aspose.PSD può elaborare **fino a 500‑page PSD files** in meno di 10 secondi su un server standard a 8 core, e supporta **50+ formati di input e output** (inclusi JPEG, PNG, BMP, TIFF e PSD). La libreria funziona interamente sul server, quindi non è mai necessario disporre di una licenza Photoshop, e utilizza API di streaming che mantengono l'uso della memoria sotto i 150 MB anche per compositi di grandi dimensioni.

## Domande frequenti

**Q: Posso unire immagini di dimensioni diverse?**  
A: Sì. Aspose.PSD ridimensiona automaticamente o aggiunge padding ai livelli in base alle dimensioni della tela che definite, preservando le proporzioni.

**Q: È possibile aggiungere una sovrapposizione colore durante l'unione?**  
A: Assolutamente. Applicate l'effetto Color Overlay a ciascun livello prima dell'unione, o all'immagine composita finale, per uno stile coerente.

**Q: Come posso preservare i metadati EXIF o XMP dopo l'unione?**  
A: Create o copiate i metadati XMP usando l'API prima di salvare il file unito; i metadati viaggiano automaticamente con l'output.

**Q: Aspose.PSD supporta l'unione di immagini in formati diversi da PSD?**  
A: Sì. Potete caricare JPEG, PNG, BMP, TIFF e molti altri formati raster, unirli e esportare il risultato come PSD, PNG o JPEG.

**Q: Quali sono le considerazioni di prestazione per grandi insiemi di immagini?**  
A: Utilizzate le API di streaming, dispose degli oggetti intermedi tempestivamente e limitate le dimensioni della tela per mantenere l'uso della memoria sotto i 200 MB per batch di 1.000 immagini.
