## Frequently Asked Questions

**Q: Can I flatten only selected layers while keeping others intact?**  
A: Yes, Aspose.PSD lets you specify which layers to flatten, giving you full control over the final composition.

**Q: Does flattening affect adjustment layers like curves or levels?**  
A: When you flatten, adjustment layers are rasterized into the underlying pixels, preserving their visual effect in the final image.

**Q: Is there a performance impact when flattening very large PSD files?**  
A: Flattening large files consumes memory proportional to the image size, but Aspose.PSD optimizes memory usage and supports streaming for massive documents.

**Q: Can I flatten PSD files to formats other than PNG/JPEG?**  
A: Absolutely—Aspose.PSD can export the flattened result to BMP, TIFF, GIF, and many other raster formats.

**Q: Do I need a license to use the flattening features in development?**  
A: A trial license works for evaluation, but a full commercial license is required for production deployments.
