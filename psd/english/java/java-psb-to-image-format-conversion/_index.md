## What is PSB and Why Convert It?
PSB (Photoshop Big) is Adobe’s format for files exceeding the 2 GB limit of a regular PSD. Large‑scale designs, high‑resolution composites, or multi‑layered artwork often use PSB. Converting PSB to lighter formats like JPG or PDF makes it easier to share, embed in web pages, or archive without losing essential visual information.

## Frequently Asked Questions

**Q: Can I convert a password‑protected PSB file?**  
A: Yes. Aspose.PSD allows you to open encrypted PSB files by providing the password during the load operation.

**Q: Will converting PSB to JPG lose layer information?**  
A: Converting to JPG flattens the image, so layer data is merged. Use PDF or PSD if you need to preserve layers.

**Q: Is there a size limit for PSB files when using Aspose.PSD?**  
A: The library is designed to handle very large PSB files (multiple gigabytes) as long as your JVM has sufficient heap memory.

**Q: Do I need a separate license for each output format?**  
A: No. A single Aspose.PSD for Java license covers all supported export formats, including JPG, PDF, and PSD.

**Q: How do I improve performance when converting many PSB files in a batch?**  
A: Reuse the same `PsdImage` instance when possible, and consider processing files in parallel streams with appropriate memory settings.
