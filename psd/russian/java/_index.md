## Что такое чёрно‑белое изображение в Java?  

Чёрно‑белое изображение хранит один байт на пиксель, представляющий яркость, в диапазоне от 0 (чёрный) до 255 (белый). Преобразование полноцветного PSD в чёрно‑белый позволяет сосредоточиться на светимости, что полезно для художественных эффектов, предобработки в алгоритмах компьютерного зрения или создания лёгких ресурсов.

## Почему оптимизировать PNG в Java с помощью Aspose.PSD?  

PNG — формат без потерь, но неоптимизированные файлы могут быть избыточно большими. С помощью Aspose.PSD вы можете:

1. Уменьшить битовую глубину, где это возможно.  
2. Применить фильтры сжатия, подходящие содержимому изображения.  
3. Удалить ненужные метаданные.  

Эти шаги сохраняют визуальное качество, одновременно уменьшая размер файлов — идеально для веб‑ и мобильных развертываний.

## Часто задаваемые вопросы

**Q: Как конвертировать PSD файл в TIFF на Java?**  
A: Загрузите PSD с помощью `PsdImage` и вызовите `save("output.tiff", new TiffOptions())`. API автоматически обрабатывает цветовые профили и слои.

**Q: Могу ли я рисовать фигуры на Java перед конвертацией в TIFF?**  
A: Да. Используйте объекты `Graphics`, предоставляемые Aspose.PSD, для рисования линий, прямоугольников или пользовательских путей, затем сохраните результат как TIFF.

**Q: Можно ли заменить шрифты в PSD перед конвертацией?**  
A: Абсолютно. Библиотека позволяет заменять отсутствующие шрифты системными, обеспечивая корректный рендеринг текста в финальном TIFF.

**Q: Как конвертировать AI файлы в изображения с помощью Aspose.PSD?**  
A: Хотя Aspose.PSD ориентирован на PSD, вы можете сначала конвертировать AI в PDF (используя Aspose.Slides или другие инструменты), а затем загрузить страницы PDF как изображения для дальнейшей обработки.

**Q: Какие шаги оптимизируют PNG файлы в Java?**  
A: Уменьшите битовую глубину, примените подходящие фильтры сжатия и удалите метаданные — всё это доступно через параметры PNG в Aspose.PSD.
