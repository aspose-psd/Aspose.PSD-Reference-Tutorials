# Как конвертировать PSD в PNG и изменять размер пропорционально с помощью Aspose.PSD для Java

## Введение

Если вам нужно **конвертировать PSD в PNG**, сохраняя исходное соотношение сторон, вы попали в нужное место. В этом руководстве мы пройдем полный **Aspose.PSD tutorial**, показывающий, **как изменять размер изображения** пропорционально в Java, кэшировать изображение для оптимальной производительности и, наконец, сохранять результат в PNG. К концу вы сможете интегрировать этот рабочий процесс в любой Java‑ориентированный конвейер обработки изображений.

## Быстрые ответы
- **Можно ли Aspose.PSD конвертировать PSD в PNG?** Да — просто загрузите PSD и сохраните его с помощью `PngOptions`.
- **Поддерживает ли библиотека пропорциональное изменение размера?** Абсолютно; используйте `resizeWidthProportionally` и `resizeHeightProportionally`.
- **Нужно ли сначала кэшировать изображение?** Кэширование (`image.cacheData()`) улучшает производительность при работе с большими PSD‑файлами.
- **Какая версия Java требуется?** Полностью поддерживается Java 8 и новее.
- **Нужна ли лицензия для продакшна?** Да, коммерческая лицензия убирает водяные знаки оценки.

## Что такое «конвертировать PSD в PNG» и почему это важно?

Конвертация Photoshop‑документа (PSD) в файл portable network graphics (PNG) позволяет делиться многослойными дизайнами в лёгком, веб‑дружественном формате. PNG сохраняет прозрачность и без потерь, что делает его идеальным для UI‑элементов, миниатюр или любой ситуации, где важна точность изображения.

## Почему изменять размер изображения пропорционально в Java?

Пропорциональное **how to resize image** без искажения соотношения сторон гарантирует естественный вид графики на разных экранах. API `Aspose.PSD` предоставляет специальные методы для пропорционального масштабирования, которые автоматически рассчитывают соответствующую высоту или ширину, избавляя от ручных вычислений и ошибок.

## Предварительные требования

Прежде чем перейти к коду, убедитесь, что у вас есть:

1. **Java Development Kit (JDK)** — установлен Java 8 или новее.
2. **Aspose.PSD for Java** — скачайте последнюю JAR‑файл с официального сайта [здесь](https://releases.aspose.com/psd/java/).
3. Пример PSD‑файла (`sample.psd`), размещённый в каталоге, доступном из вашего проекта.

## Импорт пакетов

Добавьте необходимые импорты в ваш Java‑класс. Эти классы дают доступ к загрузке изображений, кэшированию, изменению размера и параметрам вывода PNG.

```java
import com.aspose.psd.Image;
import com.aspose.psd.imageoptions.PngOptions;
```

## Шаг 1: Загрузка PSD‑изображения

Сначала загрузите исходный PSD‑файл. Если файл большой, кэширование данных изображения уменьшит нагрузку на память при последующих операциях.

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";

Image image = Image.load(sourceFile);

if (!image.isCached()) {
    // Image caching java – improves performance for large files
    image.cacheData();
}
```

## Шаг 2: Пропорциональное изменение размера изображения (Java)

Теперь задайте новые размеры. В этом примере мы уменьшаем изображение до **половины** его оригинального размера, сохраняя соотношение сторон. При необходимости можете изменить целевую ширину или высоту.

```java
int newWidth = image.getWidth() / 2;
image.resizeWidthProportionally(newWidth);

int newHeight = image.getHeight() / 2;
image.resizeHeightProportionally(newHeight);
```

> **Pro tip:** Используйте либо `resizeWidthProportionally`, *либо* `resizeHeightProportionally` — библиотека автоматически рассчитывает другую размерность, чтобы изображение оставалось пропорциональным.

## Шаг 3: Сохранение изменённого изображения в PNG

Наконец, экспортируйте изменённое изображение в формат PNG. Этот шаг завершает рабочий процесс **convert PSD to PNG**.

```java
String destName = dataDir + "SimpleResizeImageProportionally_out.png";
image.save(destName, new PngOptions());
```

Когда код выполнится, вы получите PNG‑файл, который в два раза меньше оригинального PSD, с тем же визуальным качеством и без искажений.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| **OutOfMemoryError** | Большой PSD не кэширован | Вызовите `image.cacheData()` перед изменением размера |
| **Blank PNG output** | Неправильный путь к файлу | Проверьте `dataDir` и права записи |
| **Aspect ratio looks off** | Одновременное указание ширины и высоты вручную | Используйте пропорциональные методы, предоставляемые Aspose.PSD |
