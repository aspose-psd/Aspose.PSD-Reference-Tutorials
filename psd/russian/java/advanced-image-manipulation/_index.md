- **Можно ли повернуть изображение на определённый угол?** Да — библиотека поддерживает произвольные углы через перегрузку `rotate(double angle)`.

Загрузите ваш PSD‑файл, затем вызовите метод `rotate(double angle)` — он повернёт изображение на указанный угол в градусах — после чего сохраните результат. Операция переориентирует данные битмапа, сохраняя каждый слой, маску и корректирующий эффект, гарантируя, что визуальный результат будет идентичен оригинальной композиции.

## Часто задаваемые вопросы

**Q: Можно ли повернуть PSD‑изображение без потери информации о слоях?**  
A: Да. Aspose.PSD вращает изображение на уровне слоев, сохраняя все корректирующие слои, маски и эффекты.

**Q: Как повернуть изображение на угол, не кратный 90°?**  
A: Используйте метод `rotate(double angle)`, который принимает любой угол с плавающей точкой (например, 45.0) для плавного вращения.

**Q: Влияет ли вращение больших PSD‑файлов на производительность?**  
A: Минимально. Включение кэширования шрифтов (см. руководство «Force Font Cache») и использование последней версии библиотеки снижает нагрузку на память.

**Q: Нужна ли отдельная библиотека для замены шрифтов?**  
A: Нет. Aspose.PSD включает встроенные API для замены шрифтов, что упрощает реализацию **how to replace fonts java** без дополнительных зависимостей.

**Q: Какие версии Java официально поддерживаются?**  
A: Aspose.PSD for Java поддерживает Java 8, 11, 17 и более новые LTS‑версии.
