## Часто задаваемые вопросы

**В: Можно ли повернуть PSD‑изображение без потери информации о слоях?**  
**О:** Да. Aspose.PSD вращает изображение на уровне слоёв, сохраняя все корректирующие слои, маски и эффекты.

**В: Как повернуть изображение на угол, не кратный 90°?**  
**О:** Используйте метод `rotate(double angle)`, который принимает любой угол с плавающей точкой (например, 45.0) для плавного вращения.

**В: Влияет ли вращение больших PSD‑файлов на производительность?**  
**О:** Минимально. Включение кэша шрифтов (см. учебник «Force Font Cache») и использование последней версии библиотеки снижают нагрузку на память.

**В: Нужна ли отдельная библиотека для замены шрифтов?**  
**О:** Нет. Aspose.PSD включает встроенные API для замены шрифтов, позволяя реализовать **how to replace fonts java** без дополнительных зависимостей.

**В: Какие версии Java официально поддерживаются?**  
**О:** Aspose.PSD for Java поддерживает Java 8, 11, 17 и более новые LTS‑версии.
