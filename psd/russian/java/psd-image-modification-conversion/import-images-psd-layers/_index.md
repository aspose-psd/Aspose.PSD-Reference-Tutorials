## Что такое «how to import psd»?
Импорт PSD означает программную загрузку документа Photoshop, изменение его слоёв (например, добавление изображения) и последующее сохранение обновлённого файла. Такой подход идеален для пакетной обработки, автоматической генерации графики или интеграции дизайнерских ресурсов в более крупные приложения.

## Почему стоит использовать Aspose.PSD для Java?
Aspose.PSD предоставляет полностью управляемый, бесплатный API, который скрывает сложность формата PSD. Вы получаете:

## Как импортировать изображения PSD в слои
Ниже представлена чёткая пошаговая инструкция, объясняющая **как добавить изображение** в слой PSD, **задать координаты слоя** и **заполнить цветом слой PSD**.

### Шаг 1: Импортировать необходимые пакеты
Сначала подключаем классы Aspose.PSD, которые понадобятся. Это задаёт основу для остального кода.

```java
import com.aspose.psd.Color;
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.Point;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
```

### Шаг 2: Установить каталог документа
Определите, где находится ваш исходный PSD и куда будет сохранён результат.

```java
String dataDir = "Your Document Directory";
```

Замените `"Your Document Directory"` на реальный путь в вашей файловой системе, где находятся PSD‑файлы.

### Шаг 3: Загрузить ваш PSD‑файл
Откройте PSD, чтобы работать с его слоями.

```java
PsdImage image = (PsdImage) Image.load(dataDir + "sample.psd");
```

Убедитесь, что `"sample.psd"` соответствует имени файла, который вы хотите отредактировать.

### Шаг 4: Извлечь целевой слой
Выберите слой, который получит новое изображение. Здесь используется второй слой (индекс 1).

```java
Layer layer = image.getLayers()[1];
```

Если нужен другой слой, просто измените индекс.

### Шаг 5: Создать новое изображение для импорта
Теперь **add image psd layer** создадим новый `PsdImage`, на который будем рисовать.

```java
PsdImage drawImage = new PsdImage(200, 200);
```

Вы можете скорректировать ширину и высоту, чтобы они соответствовали вашему исходному изображению.

### Шаг 6: Заполнить поверхность изображения (установить цвет слоя)
Давайте **fill psd layer color** ярко‑желтым фоном. Это демонстрирует, как задать сплошной цвет перед рисованием.

```java
Graphics g = new Graphics(drawImage);
g.clear(Color.getYellow());
```

При желании замените `Color.getYellow()` на любой другой `Color`, который вам нужен.

### Шаг 7: Нарисовать изображение на слое (задать координаты слоя)
Вот ядро **how to add image** — размещаем только что созданное изображение на выбранном слое в заданных координатах.

```java
layer.drawImage(new Point(10, 10), drawImage);
```

Вызов `Point(10, 10)` **устанавливает координаты слоя** (X = 10, Y = 10). Измените эти значения, чтобы позиционировать изображение точно там, где нужно.

### Шаг 8: Сохранить обновлённый PSD‑файл
Наконец, запишем изменения на диск.

```java
image.save(dataDir + "ImportImageToPSDLayer_out.psd");
```

Дайте выходному файлу осмысленное имя; в примере к оригиналу добавляется суффикс `_out`, чтобы сохранить исходный файл нетронутым.

## Распространённые проблемы и их решения
- **Изображение отображается пустым** — Убедитесь, что вызвали `Graphics.clear()` перед рисованием; иначе холст может быть прозрачным.  
- **Изменяется неверный слой** — Помните, что индексы слоёв начинаются с 0. Проверьте используемый индекс в `getLayers()`.  
- **Неподдерживаемый цветовой профиль** — Aspose.PSD обрабатывает большинство профилей, но если наблюдаются смещения цветов, попробуйте конвертировать исходное изображение в sRGB перед импортом.
