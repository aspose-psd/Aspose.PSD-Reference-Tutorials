# Конвертировать PSD в RAW с помощью Java (не сжатые файлы изображений)

## Введение
Когда речь идет о работе с документами Photoshop (PSD) в Java, понимание того, как **convert PSD to RAW** и экспортировать PSD без сжатия, является необходимым для сохранения точности изображения. В этом руководстве мы рассмотрим, как Aspose.PSD упрощает процесс работы с несжатыми файлами изображений, от загрузки PSD до сохранения его как RAW‑подобного несжатого файла. Независимо от того, создаете ли вы графически интенсивное приложение или вам нужны без потерь экспортированные изображения, здесь вы найдете всё необходимое. Готовы погрузиться? Давайте начнём!

## Быстрые ответы
- **What does “convert PSD to RAW” mean?** Он сохраняет данные PSD без какого‑либо сжатия, оставляя пиксельные данные в их исходной форме.  
- **Which library handles this?** Aspose.PSD for Java предоставляет простой API для несжатого сохранения.  
- **Do I need a license?** Бесплатная пробная версия подходит для тестирования; коммерческая лицензия требуется для продакшн‑использования.  
- **What Java version is required?** JDK 8 или выше.  
- **Can I still edit the file after saving?** Да — вы можете загрузить несжатый PSD и продолжить рисовать или работать со слоями.

## Что означает “convert PSD to RAW”?
Конвертация PSD в RAW означает экспорт документа Photoshop **без какого‑либо сжатия**. Полученный файл сохраняет полные пиксельные данные, что идеально подходит для сценариев, где качество изображения не может быть скомпрометировано — например, архивное хранение, научные изображения или высококачественные печатные процессы.

## Почему экспортировать PSD без сжатия?
- **Maximum quality:** Нет потери деталей из‑за артефактов сжатия.  
- **Predictable file size:** Размер RAW‑файлов напрямую отражает размеры изображения и глубину цвета.  
- **Simplified downstream processing:** Другие инструменты могут читать пиксельные данные без необходимости предварительной декомпрессии.

## Импорт пакетов
Чтобы начать, нам нужно импортировать необходимые пакеты для нашего кода. Ниже перечислены импорты, которые вам потребуются:

```java
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.CompressionMethod;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PsdOptions;
```

Эти импорты добавят необходимые классы и методы в ваш проект, позволяя без проблем работать с PSD‑файлами. Разобьём процесс на управляемые шаги.

## Шаг 1: Настройка каталога файлов
Сначала укажите, где находится ваш PSD‑файл и куда сохранять результат. В примере кода мы создадим переменную, содержащую путь к каталогу.

```java
String dataDir = "Your Document Directory";
```

Замените `"Your Document Directory"` реальным путём, где хранится ваш PSD‑файл (`layers.psd`). Это позволит программе знать, где искать файл.

## Шаг 2: Загрузка PSD файла
Теперь загрузим PSD‑файл с помощью метода `Image.load()`, приведя его к типу `PsdImage`.

```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "layers.psd");
```

Эта строка вызывает метод `load` класса `Image`, загружая ваш PSD‑файл в память. Приведение к `PsdImage` сообщает Java, что это PSD‑изображение со специфическими возможностями работы с PSD.

## Шаг 3: Настройка параметров сохранения
Далее нам нужно задать параметры сохранения файла, в частности указать, что вывод должен быть **uncompressed** (т.е. convert PSD to RAW).

```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw);
```

Класс `PsdOptions` позволяет указать различные параметры сохранения PSD‑файла. Установка `setCompressionMethod` в `CompressionMethod.Raw` гарантирует, что сохраняемый файл будет несжатым и сохранит высокое качество.

## Шаг 4: Сохранение несжатого PSD файла
Пришло время сохранить только что сконфигурированное PSD‑изображение.

```java
psdImage.save(dataDir + "uncompressed_out.psd", saveOptions);
```

Эта строка выполняет функцию сохранения на экземпляре `PsdImage` (`psdImage`). Файл сохраняется как `uncompressed_out.psd` в указанном каталоге с ранее заданными параметрами.

## Шаг 5: Повторное открытие только что созданного изображения
После сохранения загрузим наш результат, чтобы убедиться, что всё прошло успешно.

```java
PsdImage img = (PsdImage) Image.load(dataDir + "uncompressed_out.psd");
```

Вызвав `load` ещё раз, мы создаём новый экземпляр `PsdImage`, соответствующий сохранённому файлу. Этот шаг важен, если вы планируете дальше манипулировать или отображать изображение.

## Шаг 6: Рисование или манипуляция изображением
Наконец, вы можете рисовать или изменять только что открытое изображение.

```java
Graphics graphics = new Graphics(img);
```

Здесь мы инициализируем объект `Graphics`, который позволяет выполнять различные графические операции над нашим `img`. Вы можете рисовать фигуры, добавлять текст или даже изменять слои, если захотите!
