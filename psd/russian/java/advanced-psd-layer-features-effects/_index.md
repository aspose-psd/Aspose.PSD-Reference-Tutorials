## Часто задаваемые вопросы

**В:** Нужна ли платная лицензия для использования Aspose.PSD for Java?  
**О:** Доступна бесплатная пробная версия для разработки и тестирования, но для продакшн‑развёртываний требуется коммерческая лицензия.

**В:** Можно ли применить несколько эффектов, например внутреннюю тень и внешнее свечение, к одному слою?  
**О:** Да. Эффекты можно накладывать, последовательно добавляя их в коллекцию эффектов слоя.

**В:** Можно ли редактировать файл PSD без потери информации о слоях?  
**О:** Конечно. Aspose.PSD сохраняет все метаданные слоёв, маски и корректирующие слои при сохранении изменений.

**В:** Какие версии Java поддерживает последняя библиотека Aspose.PSD?  
**О:** Полностью поддерживаются Java 8, 11 и более новые LTS‑версии.

**В:** Как работать с большими файлами PSD, чтобы избежать проблем с памятью?  
**О:** Используйте перегрузку `Load` класса `PsdImage` с `FileStream` и включайте отложенную загрузку там, где это уместно.
