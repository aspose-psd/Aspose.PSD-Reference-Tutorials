## Часто задаваемые вопросы

**В: Нужна ли платная лицензия для использования Aspose.PSD for Java?**  
О: Бесплатная пробная версия доступна для разработки и тестирования, но для продакшн‑развёртываний требуется коммерческая лицензия.

**В: Можно ли применить несколько эффектов, например внутреннюю тень и внешнее свечение, к одному слою?**  
О: Да. Эффекты можно накладывать, последовательно добавляя их в коллекцию эффектов слоя.

**В: Возможно ли редактировать PSD‑файл без потери информации о слоях?**  
О: Абсолютно. Aspose.PSD сохраняет все метаданные слоёв, маски и корректирующие слои при сохранении изменений.

**В: Какие версии Java поддерживаются последней библиотекой Aspose.PSD?**  
О: Java 8, 11 и более новые LTS‑выпуски полностью поддерживаются.

**В: Как работать с большими PSD‑файлами, чтобы избежать проблем с памятью?**  
О: Используйте перегрузку метода `Load` класса `PsdImage` с `FileStream` и включайте отложенную загрузку там, где это уместно.
