## Краткие ответы
- **What does “add diagonal watermark” do?** It overlays a slanted text or image watermark across the entire PSD canvas.  
- **Which library supports this?** Aspose.PSD for Java provides a dedicated API for watermarking and other PSD manipulations.  
- **Do I need a license?** A free trial works for evaluation; a commercial license is required for production use.  
- **Can I combine watermarking with other adjustments?** Yes – you can chain operations like hue‑saturation changes, color fill layers, and blend‑mode tweaks in a single workflow.  
- **Is the API compatible with Java 8+?** Absolutely, it supports modern Java versions and runs on any JVM.

## Что такое add diagonal watermark?
Добавление диагонального водяного знака подразумевает размещение полупрозрачного текста или изображения под углом 45 градусов, чтобы он охватывал всю композицию. Эта техника препятствует несанкционированному использованию, при этом оставляя видимым основной дизайн.

## Почему использовать Aspose.PSD for Java?
- **Full PSD fidelity:** Preserve layer information, masks, and adjustment settings.  
- **Server‑side processing:** No need for a GUI; automate batch jobs on back‑end services.  
- **Rich adjustment suite:** Combine watermarking with hue‑saturation layers, color fill layers, curves adjustments, and blend‑mode changes.

## Часто задаваемые вопросы

**Q: Могу ли я добавить диагональный водяной знак в PSD, который уже содержит слои коррекции?**  
A: Да. Aspose.PSD позволяет разместить водяной знак поверх существующих слоёв, включая hue saturation, curves и слои заливки цветом, без объединения слоёв в один.

**Q: Можно ли пакетно обработать несколько PSD‑файлов, чтобы добавить один и тот же водяной знак?**  
A: Абсолютно. Вы можете пройтись по каталогу, загрузить каждый PSD, применить водяной знак и сохранить результат — всё в рамках одной Java‑программы.

**Q: Влияют ли руководства по слоям коррекции (hue saturation, curves, color fill) на видимость водяного знака?**  
A: Слои коррекции влияют на данные базового изображения, но не на растровый слой водяного знака, если вы добавляете знак после этих коррекций. Вы также можете разместить водяной знак на отдельном слое для полного контроля.

**Q: Какие соображения по памяти следует учитывать при работе с большими PSD‑файлами?**  
A: Используйте настройки перераспределения кеша, описанные в руководстве «Control Cache Reallocation», чтобы ограничить использование ОЗУ, особенно при пакетной обработке файлов высокого разрешения.

**Q: Какие версии Aspose.PSD совместимы с Java 11 и новее?**  
A: Все последние выпуски (2023‑2025) поддерживают Java 8+, включая Java 11, 17 и более новые версии. Проверьте примечания к выпуску продукта, чтобы узнать точную версию, которую вы используете.
