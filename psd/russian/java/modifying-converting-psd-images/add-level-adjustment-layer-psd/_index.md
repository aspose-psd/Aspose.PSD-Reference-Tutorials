# Добавить слой коррекции уровней в PSD

## Введение
Если вы ищете, **как настроить уровни** в ваших документах Photoshop, слой коррекции уровней — идеальный инструмент. Он позволяет точно подрегулировать тени, средние тона и светлые участки без постоянного изменения оригинальных пикселей. В этом руководстве мы пройдем процесс добавления слоя коррекции уровней в файл PSD с помощью Aspose.PSD for Java, чтобы вы могли достичь профессионального тонального контроля за несколько шагов.

## Быстрые ответы
- **Что делает слой коррекции уровней?** Он изменяет тональный диапазон изображения без разрушения.  
- **Какая библиотека используется?** Aspose.PSD for Java.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; лицензия требуется для продакшн.  
- **Сколько времени занимает реализация?** Около 10‑15 минут для базовой коррекции.  
- **Можно ли настроить несколько каналов?** Да, вы можете задавать входные/выходные уровни для каждого цветового канала отдельно.

## Что такое слой коррекции уровней?
Слой коррекции уровней позволяет исправить тональный баланс изображения, регулируя входные тени, средние тона и светлые участки, а также выходные уровни. Поскольку он находится на отдельном слое, вы можете включать/выключать его видимость или удалять его, не влияя на нижележащие элементы.

## Зачем добавлять слой коррекции уровней с помощью Aspose.PSD?
- **Автоматизация:** Интегрировать корректировки уровней в конвейеры пакетной обработки.  
- **Кросс‑платформенность:** Работает на любой ОС, поддерживающей Java.  
- **Точность:** Программно получать доступ к настройкам каждого канала для точных результатов.  

## Требования
1. Java Development Kit (JDK). Если у вас его нет, скачайте с [Oracle website](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) или используйте OpenJDK.  
2. Библиотека Aspose.PSD for Java — получите последнюю JAR по этой [download link](https://releases.aspose.com/psd/java/).  
3. Базовые знания программирования на Java.  
4. IDE, например IntelliJ IDEA, Eclipse или NetBeans, с добавленным в classpath проекта JAR Aspose.PSD.  

## Импорт пакетов
Прежде чем начать писать код, нам нужно импортировать необходимые пакеты из библиотеки Aspose.PSD. Вот как это сделать:
```java
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.adjustmentlayers.LevelsLayer;
import com.aspose.psd.fileformats.psd.layers.layerresources.LevelChannel;
```
Эти импорты дают нам доступ к классам для загрузки PSD‑файлов, работы со слоями коррекции уровней и управления настройками отдельных каналов.

## Как настроить уровни в файле PSD
Ниже пошаговое руководство, показывающее, как **настроить уровни** программно.

### Шаг 1: Настройте пути к файлам
Укажите, где находится исходный PSD и куда будет сохранён отредактированный файл.
```java
String dataDir = "Your Document Directory";
String sourceFileName = dataDir + "LevelsAdjustmentLayer.psd";
String psdPathAfterChange = dataDir + "LevelsAdjustmentLayerChanged.psd";
```
Замените `"Your Document Directory"` на фактическую папку на вашем компьютере.

### Шаг 2: Загрузите PSD‑файл
Создайте экземпляр `PsdImage` из исходного файла.
```java
PsdImage im = (PsdImage) Image.load(sourceFileName);
```
Теперь у вас есть полный доступ ко всем слоям внутри PSD.

### Шаг 3: Пройдите по слоям
Найдите слой коррекции уровней, который хотите изменить.
```java
for (int i = 0; i < im.getLayers().length; i++) {
    if (im.getLayers()[i] instanceof LevelsLayer) {
        LevelsLayer levelsLayer = (LevelsLayer) im.getLayers()[i];
        // Further manipulation will go here...
    }
}
```
Проверка `instanceof LevelsLayer` гарантирует, что мы работаем только со слоями коррекции уровней.

### Шаг 4: Настройте параметры каналов уровня
Отрегулируйте входные и выходные значения для выбранного канала.
```java
LevelChannel channel = levelsLayer.getChannel(0);
channel.setInputMidtoneLevel(2.0f);
channel.setInputShadowLevel((short) 10);
channel.setInputHighlightLevel((short) 230);
channel.setOutputShadowLevel((short) 20);
channel.setOutputHighlightLevel((short) 200);
```
- **Input Midtone Level:** Смещает диапазон средних тонов.  
- **Input Shadow Level:** Затемняет или осветляет тени.  
- **Input Highlight Level:** Управляет самыми яркими участками.  
- **Output Shadow/Highlight Levels:** Определяют окончательный диапазон вывода.  

Не стесняйтесь экспериментировать с разными значениями, чтобы увидеть, как они влияют на изображение.

### Шаг 5: Сохраните изменённый PSD‑файл
Сохраните изменения в новый файл.
```java
im.save(psdPathAfterChange);
```
Обновлённый PSD будет находиться по пути, указанному в `psdPathAfterChange`.

## Распространённые проблемы и решения
- **File not found:** Убедитесь, что `dataDir` указывает на правильную папку и исходный PSD существует.  
- **ClassCastException:** Убедитесь, что загружаемый файл действительно PSD; другие форматы требуют других классов.  
- **License errors:** Используйте действующую лицензию Aspose.PSD для продакшн‑сборок; пробная версия подходит для разработки.
