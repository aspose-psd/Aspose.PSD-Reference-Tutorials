## Veelgestelde vragen

**Q: Hoe converteer ik PSD naar TIFF met Aspose.PSD voor Java?**  
A: Gebruik de `PsdImage`‑klasse om het PSD‑bestand te laden en roep de `save`‑methode aan met `TiffOptions`. De tutorial “Convert CMYK PSD to CMYK TIFF” toont de exacte code.

**Q: Kan ik afbeeldingen op een multi‑threaded manier exporteren?**  
A: Ja. De gids “Export Images in Multi‑Threaded Environment” laat zien hoe je conversies veilig op afzonderlijke threads kunt uitvoeren.

**Q: Hoe converteer ik GIF‑afbeeldingslagen naar TIFF?**  
A: Laad de GIF met `Image.load`, doorloop de frames en sla elk frame op als een TIFF‑pagina met `TiffOptions`. Zie de tutorial “Convert GIF Image Layers to TIFF”.

**Q: Ondersteunt Aspose.PSD java‑beeldverwerking voor kleurbeheer?**  
A: Absoluut. Je kunt standaardprofielen gebruiken of aangepaste ICC‑profielen leveren, zoals uitgelegd in de secties “Color Conversion using Default Profiles” en “Color Conversion using ICC Profiles”.

**Q: Waar kan ik meer voorbeelden vinden over het exporteren van afbeeldingen?**  
A: De pagina “Export Images in Multi‑Threaded Environment” bevat gedetailleerde voorbeelden over hoe je afbeeldingen efficiënt kunt exporteren.
