Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων όπου θα αποθηκευτεί η εικόνα εξόδου.

```csharp
string dataDir = "Your Document Directory";
```

## Βήμα 2: Σχεδιάζοντας ορθογώνια

Τώρα, ας εμβαθύνουμε στη διαδικασία σχεδίασης ορθογωνίων χρησιμοποιώντας το Aspose.PSD.

### Βήμα 2.1: Δημιουργήστε μια παρουσία του BmpOptions

```csharp
string outpath = dataDir + "Rectangle.bmp";
BmpOptions saveOptions = new BmpOptions();
saveOptions.BitsPerPixel = 32;
```

### Βήμα 2.2: Δημιουργήστε μια παρουσία εικόνας

```csharp
using (Image image = new PsdImage(100, 100))
{
    // Βήμα 2.3: Αρχικοποιήστε την κατηγορία γραφικών και καθαρίστε την επιφάνεια γραφικών
    Graphics graphic = new Graphics(image);
    graphic.Clear(Color.Yellow);

    // Βήμα 2.4: Σχεδιάστε ορθογώνια
    graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
    graphic.DrawRectangle(new Pen(new SolidBrush(Color.Blue)), new Rectangle(10, 30, 80, 40));

    // Βήμα 2.5: Εξαγωγή εικόνας σε μορφή αρχείου BMP
    image.Save(outpath, saveOptions);
}
```
