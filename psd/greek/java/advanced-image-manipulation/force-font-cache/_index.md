# Αποθήκευση PSD με γραμματοσειρές και εξαναγκασμός προσωρινής μνήμης γραμματοσειρών – Aspose.PSD Java

## Εισαγωγή

Αν χρειάζεστε να **αποθηκεύσετε PSD με γραμματοσειρές** γρήγορα ενώ εξασφαλίζετε ότι οι σωστές γραμματοσειρές είναι διαθέσιμες, βρίσκεστε στο σωστό μέρος. Η προσωρινή μνήμη γραμματοσειρών μπορεί να βελτιώσει δραματικά **την απόδοση της εικόνας**, ειδικά όταν επεξεργάζεστε επανειλημμένα μεγάλα έγγραφα Photoshop. Σε αυτό το tutorial θα περάσουμε βήμα προς βήμα τις ακριβείς ενέργειες για να εξαναγκάσουμε την προσωρινή μνήμη γραμματοσειρών, **να φορτώσουμε αρχεία PSD image**, και τελικά **να αποθηκεύσουμε PSD με γραμματοσειρές** χρησιμοποιώντας το Aspose.PSD για Java.

## Γρήγορες απαντήσεις
- **Τι κάνει η εξαναγκασμένη προσωρινή μνήμη γραμματοσειρών;** Αναγκάζει το Aspose.PSD να ξανασκανάρει τις γραμματοσειρές του συστήματος ώστε οι νεοεγκατεστημένες γραμματοσειρές να αναγνωρίζονται άμεσα.  
- **Πόσο χρόνο πρέπει να περιμένω για την εγκατάσταση μιας γραμματοσειράς;** Ο κώδικας δείγματος κάνει παύση για **2 λεπτά**, που συνήθως είναι αρκετά για χειροκίνητη εγκατάσταση.  
- **Μπορώ να το χρησιμοποιήσω με οποιοδήποτε αρχείο PSD;** Ναι – εφόσον το αρχείο είναι προσβάσιμο και οι απαιτούμενες γραμματοσειρές είναι εγκατεστημένες στο σύστημα.  
- **Χρειάζομαι άδεια για την εκτέλεση αυτού του κώδικα;** Απαιτείται προσωρινή ή πλήρης άδεια για χρήση σε παραγωγή· μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση.  
- **Ποιες εκδόσεις της Java υποστηρίζονται;** Το Aspose.PSD για Java λειτουργεί με JDK 8 και νεότερες.

## Τι είναι η “αποθήκευση PSD με γραμματοσειρές” και γιατί είναι σημαντική;

Η αποθήκευση ενός αρχείου PSD μετά την επεξεργασία των στρωμάτων, του κειμένου ή των γραμματοσειρών είναι το τελικό βήμα που διατηρεί τις αλλαγές σας. Όταν η προσωρινή μνήμη γραμματοσειρών δεν είναι ενημερωμένη, το αποθηκευμένο αρχείο μπορεί να επιστρέψει σε προεπιλεγμένες γραμματοσειρές, προκαλώντας οπτικές ασυνέπειες. Εξαναγκάζοντας πρώτα τη μνήμη, εξασφαλίζετε ότι η λειτουργία **αποθήκευσης PSD με γραμματοσειρές** ενσωματώνει τις σωστές γραμματοσειρές.

## Γιατί να εξαναγκάσετε τη προσωρινή μνήμη γραμματοσειρών με το Aspose.PSD;

- **Βελτίωση απόδοσης** – Μειώνει την ανάγκη για επαναλαμβανόμενες αναζητήσεις γραμματοσειρών.  
- **Αξιοπιστία** – Εγγυάται ότι τα αποθηκευμένα αρχεία PSD αποδίδουν ακριβώς όπως προορίζεται σε οποιονδήποτε υπολογιστή.  
- **Απλότητα** – Μία κλήση μεθόδου (`OpenTypeFontsCache.updateCache()`) διαχειρίζεται το βάρος της εργασίας.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- Εγκατεστημένο Java Development Kit (JDK) 8 ή νεότερο.  
- Βιβλιοθήκη Aspose.PSD for Java (κατεβάστε από τον επίσημο [download link](https://releases.aspose.com/psd/java/)).  
- Ένα δείγμα αρχείου PSD (`sample.psd`) τοποθετημένο σε φάκελο που μπορείτε να αναφέρετε από τον κώδικά σας.  

Τώρα που όλα είναι έτοιμα, ας βυθιστούμε στην υλοποίηση.

## Εισαγωγή πακέτων

Πρώτα, εισάγετε τις κλάσεις που απαιτούνται για εργασία με εικόνες PSD και την προσωρινή μνήμη γραμματοσειρών. Τοποθετήστε αυτές τις δηλώσεις στην κορυφή της κλάσης Java:

```java
import com.aspose.psd.Image;
import com.aspose.psd.OpenTypeFontsCache;

import com.aspose.psd.fileformats.psd.PsdImage;
import java.io.Console;
import java.util.concurrent.TimeUnit;
```

## Οδηγός βήμα‑βήμα

### Βήμα 1: Φόρτωση της εικόνας PSD (Πώς να φορτώσετε εικόνα PSD)

Ξεκινάμε φορτώνοντας το αρχικό αρχείο PSD. Αυτό μας παρέχει ένα αντικείμενο εικόνας βάσης που μπορούμε αργότερα να αποθηκεύσουμε ξανά μετά την ενημέρωση της προσωρινής μνήμης γραμματοσειρών.

```java
String dataDir = "Your Document Directory";

PsdImage image = (PsdImage)Image.load(dataDir + "sample.psd");
image.save(dataDir + "NoFont.psd");
```

- **Εξήγηση:**  
  - `Image.load` διαβάζει το αρχείο στη μνήμη.  
  - `image.save` δημιουργεί ένα αντίγραφο με όνομα **NoFont.psd** που αντανακλά την κατάσταση **πριν** να είναι διαθέσιμες νέες γραμματοσειρές.  
  - Αυτό το βήμα είναι χρήσιμο για σύγκριση και διασφαλίζει ότι η επακόλουθη ενημέρωση της μνήμης αλλάζει πραγματικά το αποτέλεσμα.

### Βήμα 2: Αναμονή για εγκατάσταση γραμματοσειράς (java thread sleep – βελτίωση απόδοσης εικόνας)

Τώρα δίνουμε στον χρήστη χρόνο να εγκαταστήσει τη ζητούμενη γραμματοσειρά χειροκίνητα. Σε πραγματικό σενάριο μπορεί να αυτοματοποιηθεί αυτό το βήμα, αλλά η παύση δείχνει τον μηχανισμό ανανέωσης της μνήμης.

```java
System.out.println("You have 2 minutes to install the font");
Thread.sleep(2 * 60 * 1000);
OpenTypeFontsCache.updateCache();
```

- **Εξήγηση:**  
  - `Thread.sleep` (ένα **java thread sleep**) παύει την εκτέλεση για **2 λεπτά** (2 × 60 × 1000 ms).  
  - `OpenTypeFontsCache.updateCache()` εξαναγκάζει το Aspose.PSD να ξανασκανάρει τις OpenType γραμματοσειρές του συστήματος, καθιστώντας τυχόν νεοεγκατεστημένες γραμματοσειρές άμεσα διαθέσιμες για απόδοση.  
  - Αυτή η παύση βοηθά στη **βελτίωση απόδοσης εικόνας** διασφαλίζοντας ότι η μνήμη αντανακλά τις τελευταίες γραμματοσειρές πριν από την επόμενη φόρτωση.

### Βήμα 3: Φόρτωση της ενημερωμένης εικόνας PSD και **αποθήκευση PSD με γραμματοσειρές**

Μετά την ανανέωση της μνήμης, φορτώνουμε ξανά το αρχικό PSD. Αυτή τη φορά οι πληροφορίες γραμματοσειράς είναι ενημερωμένες και μπορούμε να **αποθηκεύσουμε PSD με γραμματοσειρές** σωστά.

```java
PsdImage image1 = (PsdImage)Image.load(dataDir + "sample.psd");
image1.save(dataDir + "HasFont.psd");
```

- **Εξήγηση:**  
  - Η δεύτερη φόρτωση εντοπίζει τη νεοεγκατεστημένη γραμματοσειρά.  
  - Η αποθήκευση σε **HasFont.psd** παράγει ένα αρχείο που τώρα περιέχει τη σωστή απόδοση γραμματοσειράς, επιβεβαιώνοντας ότι η ενημέρωση της μνήμης λειτούργησε.

## Κοινά προβλήματα και λύσεις

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| Το αποθηκευμένο PSD εξακολουθεί να εμφανίζει προεπιλεγμένη γραμματοσειρά | Η γραμματοσειρά δεν είναι εγκατεστημένη σε θέση που σαρώνεται από το λειτουργικό σύστημα | Εγκαταστήστε τη γραμματοσειρά στο φάκελο γραμματοσειρών του συστήματος (π.χ., `C:\Windows\Fonts` στα Windows) και ξανατρέξτε `updateCache()`. |
| `Thread.sleep` throws `InterruptedException` | Η υπογραφή της μεθόδου δεν διαχειρίζεται ελεγχόμενες εξαιρέσεις | Προσθέστε `throws InterruptedException` στη μέθοδο `main` ή τυλίξτε την κλήση σε μπλοκ try‑catch. |
| `OpenTypeFontsCache.updateCache()` δεν κάνει τίποτα | Εκτέλεση σε server χωρίς γραφικό περιβάλλον χωρίς διαμόρφωση γραμματοσειρών | Βεβαιωθείτε ότι ο server έχει εγκατεστημένες τις απαιτούμενες γραμματοσειρές και ότι η διαδικασία Java έχει άδεια ανάγνωσης. |
| **διαχείριση ελλιπών γραμματοσειρών** – PSD αποδίδει με εναλλακτική γραμματοσειρά | Τα αρχεία γραμματοσειρών είναι κατεστραμμένα ή μη προσβάσιμα | Επαληθεύστε την ακεραιότητα των αρχείων γραμματοσειρών και βεβαιωθείτε ότι η διαδικασία Java έχει πρόσβαση ανάγνωσης. |
