# Περιστροφή Εικόνας 270 Μοιρών με Aspose.PSD για Java

## Εισαγωγή

Σε αυτό το **java image processing tutorial**, θα ανακαλύψετε **πώς να περιστρέψετε μια εικόνα** 270 μοίρες γρήγορα και αξιόπιστα χρησιμοποιώντας το Aspose.PSD για Java. Είτε δημιουργείτε ένα εργαλείο επεξεργασίας φωτογραφιών, αυτοματοποιείτε μαζικές μετατροπές, είτε απλώς χρειάζεστε να επαναπροσανατολίσετε ένα στρώμα PSD, η βιβλιοθήκη κάνει την εργασία χωρίς κόπο. Θα αγγίξουμε επίσης τεχνικές **flip image java** και τη μετατροπή του περιστραμμένου PSD σε JPEG, ώστε να έχετε μια πλήρη ροή εργασίας από άκρη σε άκρη χωρίς Photoshop.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται την περιστροφή;** Aspose.PSD για Java  
- **Ποια γωνία περιστροφής χρησιμοποιεί το παράδειγμα;** 270 μοίρες  
- **Μπορώ επίσης να αντιστρέψω (flip) την εικόνα;** Ναι – χρησιμοποιήστε επιλογές `RotateFlipType` όπως `Rotate90FlipX`  
- **Πώς αποθηκεύεται το αποτέλεσμα;** Στο παράδειγμα αποθηκεύουμε ως JPEG χρησιμοποιώντας `JpegOptions`  
- **Χρειάζεται άδεια για παραγωγική χρήση;** Απαιτείται έγκυρη άδεια Aspose.PSD για εμπορική χρήση  

## Τι σημαίνει «rotate image 270 degrees»;
Η περιστροφή μιας εικόνας 270 μοίρες σημαίνει την στροφή της φωτογραφίας τρία τέταρτα του πλήρους κύκλου δεξιόστροφα (ή 90 μοίρες αριστερόστροφα). Σε πολλές περιπτώσεις επεξεργασίας γραφικών αυτή η προσανατολισμός ταιριάζει με την αρχική διάταξη πορτραίτου μετά από μια σειρά μετασχηματισμών.

## Γιατί να Χρησιμοποιήσετε το Aspose.PSD για Αυτό το Καθήκον;
- **Πλήρης υποστήριξη PSD** – λειτουργεί με στρώματα, μάσκες και αντικείμενα ρυθμίσεων.  
- **Δεν απαιτείται τοπικό Photoshop** – τρέχει σε οποιοδήποτε περιβάλλον Java.  
- **Απλό API** – μια κλήση μεθόδου (`rotateFlip`) διαχειρίζεται την περιστροφή και την αντιστροφή.  
- **Εύκολη μετατροπή μορφής** – εξαγωγή απευθείας σε JPEG, PNG ή άλλες κοινές μορφές.

## Πώς να Περιστρέψετε Εικόνα με Aspose.PSD για Java
Παρακάτω θα βρείτε έναν οδηγό βήμα‑βήμα που σας οδηγεί στη φόρτωση ενός PSD, στην εφαρμογή περιστροφής 270°, στην προαιρετική αντιστροφή της εικόνας και, τέλος, στην αποθήκευση του αποτελέσματος ως JPEG.

### Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Βιβλιοθήκη Aspose.PSD για Java** εγκατεστημένη. Μπορείτε να τη κατεβάσετε και να δείτε την πλήρη τεκμηρίωση API [εδώ](https://reference.aspose.com/psd/java/).  
- Περιβάλλον ανάπτυξης Java (JDK 8 ή νεότερο).  
- Ένα δείγμα αρχείου PSD που θέλετε να περιστρέψετε. Ενημερώστε τη μεταβλητή `sourceFile` στον κώδικα με τη σωστή διαδρομή προς το αρχείο σας.

### Εισαγωγή Πακέτων

Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις από το πακέτο Aspose.PSD:

```java
import com.aspose.psd.Image;
import com.aspose.psd.RotateFlipType;

import com.aspose.psd.imageoptions.JpegOptions;
```

### Βήμα 1: Φόρτωση της Εικόνας

Δημιουργήστε ένα αντικείμενο `Image` που δείχνει στο αρχείο PSD προέλευσης:

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";
Image image = Image.load(sourceFile);
```

### Βήμα 2: Περιστροφή της Εικόνας 270 Μοιρών

Χρησιμοποιήστε τη μέθοδο `rotateFlip` με `RotateFlipType.Rotate270FlipNone` για να πετύχετε περιστροφή 270° χωρίς καμία αντιστροφή:

```java
image.rotateFlip(RotateFlipType.Rotate270FlipNone);
```

> **Συμβουλή επαγγελματία:** Εάν χρειάζεστε επίσης οριζόντια ή κάθετη αντιστροφή, επιλέξτε διαφορετικό `RotateFlipType` όπως `Rotate90FlipX` ή `Rotate180FlipY`. Αυτός είναι ο πιο συνηθισμένος τρόπος για λειτουργίες **flip image java**.

### Βήμα 3: Μετατροπή PSD σε JPEG και Αποθήκευση

Μετά τη περιστροφή, μπορείτε να **μετατρέψετε PSD σε JPEG** (ή οποιαδήποτε άλλη υποστηριζόμενη μορφή) χρησιμοποιώντας την κατάλληλη κλάση επιλογών:

```java
String destName = dataDir + "RotatedImage_out.jpg";
image.save(destName, new JpegOptions());
```

Το αρχείο `RotatedImage_out.jpg` περιέχει τώρα το αρχικό περιεχόμενο PSD περιστραμμένο 270 μοίρες και αποθηκευμένο ως JPEG.
