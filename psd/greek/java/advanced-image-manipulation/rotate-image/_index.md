# Περιστροφή Εικόνας 270 Μοιρών με το Aspose.PSD για Java

## Εισαγωγή

Σε αυτό το **java tutorial επεξεργασίας εικόνας**, θα ανακαλύψετε πώς να **περιστρέψετε την εικόνα 270 μοίρες** γρήγορα και αξιόπιστα χρησιμοποιώντας το Aspose.PSD για Java. Είτε δημιουργείτε ένα εργαλείο επεξεργασίας φωτογραφιών, αυτοματοποιείτε μαζικές μετατροπές, είτε απλώς χρειάζεστε να επαναπροσανατολίσετε ένα στρώμα PSD, η βιβλιοθήκη κάνει την εργασία χωρίς κόπο. Θα αγγίξουμε επίσης την αντιστροφή εικόνων και τη μετατροπή του περιστραμμένου PSD σε JPEG, ώστε να έχετε μια πλήρη ροή εργασίας από την αρχή μέχρι το τέλος.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται την περιστροφή;** Aspose.PSD for Java  
- **Ποια γωνία περιστροφής χρησιμοποιεί το παράδειγμα;** 270 degrees  
- **Μπορώ επίσης να αντιστρέψω την εικόνα;** Yes – use `RotateFlipType` options like `Rotate90FlipX`  
- **Πώς αποθηκεύεται το αποτέλεσμα;** In the example we save as JPEG using `JpegOptions`  
- **Χρειάζομαι άδεια για παραγωγή;** A valid Aspose.PSD license is required for commercial use  

## Τι είναι η «περιστροφή εικόνας 270 μοιρών»;
Η περιστροφή μιας εικόνας κατά 270 μοίρες σημαίνει την στροφή της εικόνας τρία τέταρτα ενός πλήρους κύκλου δεξιόστροφα (ή 90 μοίρες αριστερόστροφα). Σε πολλές περιπτώσεις επεξεργασίας γραφικών αυτή η προσανατολισμός ταιριάζει με την αρχική διάταξη πορτραίτου μετά από μια σειρά μετασχηματισμών.

## Γιατί να χρησιμοποιήσετε το Aspose.PSD για αυτήν την εργασία;
- **Πλήρης υποστήριξη PSD** – λειτουργεί με στρώματα, μάσκες και αντικείμενα προσαρμογής.  
- **Δεν απαιτείται το εγγενές Photoshop** – εκτελείται σε οποιοδήποτε περιβάλλον Java.  
- **Απλό API** – μια κλήση μεθόδου (`rotateFlip`) διαχειρίζεται την περιστροφή και την αντιστροφή.  
- **Εύκολη μετατροπή μορφής** – εξαγωγή απευθείας σε JPEG, PNG ή άλλες κοινές μορφές.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Aspose.PSD for Java** βιβλιοθήκη εγκατεστημένη. Μπορείτε να την κατεβάσετε και να δείτε την πλήρη αναφορά API [εδώ](https://reference.aspose.com/psd/java/).  
- Ένα περιβάλλον ανάπτυξης Java (JDK 8 ή νεότερο).  
- Ένα δείγμα αρχείου PSD που θέλετε να περιστρέψετε. Ενημερώστε τη μεταβλητή `sourceFile` στον κώδικα με τη σωστή διαδρομή προς το αρχείο σας.

## Εισαγωγή Πακέτων

Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις από το πακέτο Aspose.PSD:

```java
import com.aspose.psd.Image;
import com.aspose.psd.RotateFlipType;

import com.aspose.psd.imageoptions.JpegOptions;
```

## Πώς να Περιστρέψετε PSD – Βήμα 1: Φόρτωση της Εικόνας

Δημιουργήστε μια παρουσία `Image` που δείχνει στο πηγαίο αρχείο PSD:

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";
Image image = Image.load(sourceFile);
```

## Πώς να Περιστρέψετε PSD – Βήμα 2: Περιστροφή της Εικόνας 270 Μοιρών

Χρησιμοποιήστε τη μέθοδο `rotateFlip` με `RotateFlipType.Rotate270FlipNone` για να επιτύχετε μια περιστροφή 270 μοίρες χωρίς καμία αντιστροφή:

```java
image.rotateFlip(RotateFlipType.Rotate270FlipNone);
```

> **Pro tip:** Εάν χρειάζεστε επίσης αντιστροφή της εικόνας οριζόντια ή κάθετα, επιλέξτε διαφορετικό `RotateFlipType` όπως `Rotate90FlipX` ή `Rotate180FlipY`.

## Πώς να Περιστρέψετε PSD – Βήμα 3: Μετατροπή PSD σε JPEG και Αποθήκευση

Μετά την περιστροφή, μπορείτε **να μετατρέψετε το PSD σε JPEG** (ή οποιαδήποτε άλλη υποστηριζόμενη μορφή) χρησιμοποιώντας την κατάλληλη κλάση επιλογών:

```java
String destName = dataDir + "RotatedImage_out.jpg";
image.save(destName, new JpegOptions());
```

Το αρχείο `RotatedImage_out.jpg` τώρα περιέχει το αρχικό περιεχόμενο PSD περιστραμμένο 270 μοίρες και αποθηκευμένο ως JPEG.

**Q: Είναι το Aspose.PSD συμβατό με διαφορετικές μορφές εικόνας;**  
A: Ναι, το Aspose.PSD υποστηρίζει PSD, JPEG, PNG, BMP, GIF και πολλές άλλες μορφές raster.

**Q: Μπορώ να εφαρμόσω προσαρμοσμένες περιστροφές, όχι μόνο προεπιλεγμένες αντιστροφές;**  
A: Απόλυτα! Ενώ το `RotateFlipType` παρέχει κοινές γωνίες, μπορείτε να συνδυάσετε πολλαπλές κλήσεις ή να χρησιμοποιήσετε πίνακες μετασχηματισμού για αυθαίρετες γωνίες.

**Q: Πώς μετατρέπω το περιστραμμένο PSD σε άλλη μορφή, όπως PNG;**  
A: Αντικαταστήστε το `JpegOptions` με `PngOptions` (ή την κατάλληλη κλάση επιλογών) στη μέθοδο `save`.
