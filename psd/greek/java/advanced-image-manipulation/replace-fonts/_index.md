# αποθήκευση psd ως png – αντικατάσταση ελλειπουσών γραμματοσειρών σε Java

## Εισαγωγή

Αν χρειάζεστε να **αποθηκεύσετε PSD ως PNG** ενώ αντικαθιστάτε ελλειπουσές ή ανεπιθύμητες γραμματοσειρές μέσα σε ένα αρχείο Photoshop (PSD), η αντικατάσταση γραμματοσειρών Aspose PSD το κάνει εύκολο. Σε εφαρμογές Java μπορείτε να φορτώσετε ένα PSD, να υποδείξετε στο Aspose ποια εναλλακτική γραμματοσειρά να χρησιμοποιήσει, και στη συνέχεια να εξάγετε τη διορθωμένη εικόνα σε οποιαδήποτε μορφή θέλετε. Αυτό το μάθημα σας καθοδηγεί μέσα από τη πλήρη ροή εργασίας — από τη ρύθμιση του έργου μέχρι την εξαγωγή του ενημερωμένου PNG — ώστε να μπορείτε αξιόπιστα να **χειρίζεστε σενάρια ελλειπουσών γραμματοσειρών PSD** χωρίς ποτέ να ανοίξετε το Photoshop.

- **Μπορώ να αποθηκεύσω σε μορφές εκτός του PNG;** Ναι – υποστηρίζονται PSD, JPEG, BMP και άλλα

## Εισαγωγή Πακέτων

Η κλάση `PsdImage` αντιπροσωπεύει ένα έγγραφο Photoshop στη μνήμη, παρέχοντας πρόσβαση σε επίπεδα, κείμενο και δυνατότητες απόδοσης. Η `PsdLoadOptions` ελέγχει πώς διαβάζεται το αρχείο, συμπεριλαμβανομένης της εναλλακτικής γραμματοσειράς, ενώ οι `SaveOptions` (ή υποκλάσεις συγκεκριμένων μορφών) ορίζουν πώς γράφεται η εικόνα.

```java
import com.aspose.psd.Image;

import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageloadoptions.PsdLoadOptions;
import com.aspose.psd.imageoptions.PngOptions;
```

## Βήμα 1: Ορίστε τον Κατάλογο του Εγγράφου σας

Καθορίστε το φάκελο που περιέχει το αρχείο PSD προέλευσης. Αντικαταστήστε το σύμβολο κράτησης θέσης με την πραγματική διαδρομή στο σύστημά σας.

Το αντικείμενο `File` απλώς δείχνει στο PSD που θέλετε να επεξεργαστείτε· δεν απαιτείται πρόσθετη διαμόρφωση εδώ.  

```java
String dataDir = "Your Document Directory";
```

## Βήμα 2: Φορτώστε την Εικόνα με Γραμματοσειρά Αντικατάστασης

Δημιουργήστε μια παρουσία της `PsdLoadOptions`, ορίστε τη προεπιλεγμένη γραμματοσειρά αντικατάστασης (π.χ., **Arial**) και φορτώστε το PSD. Το Aspose θα εφαρμόσει αυτόματα την εναλλακτική γραμματοσειρά όποτε συναντήσει μια ελλειπούσα γραμματοσειρά.

Η `PsdLoadOptions` σας επιτρέπει να ορίσετε τη συμπεριφορά φόρτωσης, συμπεριλαμβανομένης της εναλλακτικής γραμματοσειράς που αντικαθιστά οποιαδήποτε ελλειπούσα γραμματοσειρά κατά τη φάση εισαγωγής.  

```java
PsdLoadOptions psdLoadOptions = new PsdLoadOptions(); 
psdLoadOptions.setDefaultReplacementFont("Arial");

PsdImage psdImage = (PsdImage)Image.load(dataDir + "Cloud_AzPlat_Banner3A_SB_EN_US_160x600_chinese_font.psd", psdLoadOptions);
```

## Βήμα 3: Αποθηκεύστε την Αντικατεστημένη Εικόνα ως PNG

Μετά την αντικατάσταση της γραμματοσειράς, μπορείτε να εξάγετε την εικόνα σε οποιαδήποτε υποστηριζόμενη μορφή. Εδώ την αποθηκεύουμε ως PNG, αλλά μπορείτε επίσης να επιλέξετε JPEG, BMP ή ακόμη και να την γράψετε ξανά ως PSD.

Η μέθοδος `save` της `PsdImage` δέχεται ένα αντικείμενο `SaveOptions`; η χρήση του `PngOptions` εξασφαλίζει ότι η έξοδος είναι ένα loss‑less PNG κατάλληλο για web ή περαιτέρω επεξεργασία.  

```java
PngOptions pngOptions = new PngOptions();
psdImage.save(dataDir + "replaced_font.png", pngOptions);
```

## Πώς να αποθηκεύσετε PSD ως PNG μετά την αντικατάσταση των γραμματοσειρών;

Φορτώστε το PSD με μια εναλλακτική γραμματοσειρά, στη συνέχεια καλέστε `psdImage.save("output.png", new PngOptions())`. Αυτή η εντολή αποθήκευσης μιας γραμμής γράφει ένα πλήρως αποδομένο PNG που αντικατοπτρίζει τη γραμματοσειρά που αντικαταστάθηκε, επιτρέποντάς σας να ενσωματώσετε την εικόνα οπουδήποτε χωρίς ανησυχία για ελλειπούσες γραμματοσειρές. Βεβαιωθείτε ότι έχετε εφαρμόσει τη γραμματοσειρά αντικατάστασης πριν από την αποθήκευση, και προαιρετικά προσαρμόστε τις ρυθμίσεις συμπίεσης PNG μέσω του αντικειμένου `PngOptions` για βέλτιστο μέγεθος αρχείου.
