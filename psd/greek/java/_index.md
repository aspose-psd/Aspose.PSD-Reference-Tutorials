## Quick Overview  

- **What is a grayscale image in Java?** Μία εικόνα μονοκαναλική όπου κάθε pixel αποθηκεύει μόνο πληροφορίες έντασης (χωρίς χρώμα).  
- **Why use Aspose.PSD?** Υποστηρίζει την πλήρη προδιαγραφή PSD, προσφέρει φίλτρα υψηλού επιπέδου και λειτουργεί δια‑πλατφόρμα χωρίς εγγενείς εξαρτήσεις.  
- **When to optimize PNG in Java?** Πριν τη δημοσίευση στο web ή την ενσωμάτωση σε κινητές εφαρμογές για μείωση του εύρους ζώνης και βελτίωση του χρόνου φόρτωσης.  

## What is a Grayscale Image in Java?  

Μια εικόνα γκρι κλίμακας αποθηκεύει ένα byte ανά pixel που αντιπροσωπεύει τη φωτεινότητα, κυμαινόμενη από 0 (μαύρο) έως 255 (άσπρο). Η μετατροπή ενός πλήρους χρωματικού PSD σε γκρι κλίμακα σας επιτρέπει να εστιάσετε στη λαμπρότητα, κάτι που είναι χρήσιμο για καλλιτεχνικά εφέ, προεπεξεργασία αλγορίθμων υπολογιστικής όρασης ή δημιουργία ελαφριών πόρων.

## Why Optimize PNG Java with Aspose.PSD?  

Το PNG είναι χωρίς απώλειες, αλλά τα μη βελτιστοποιημένα αρχεία μπορεί να είναι περιττά μεγάλα. Χρησιμοποιώντας το Aspose.PSD μπορείτε:

1. Να μειώσετε το βάθος χρώματος όπου είναι δυνατόν.  
2. Να εφαρμόσετε φίλτρα συμπίεσης που ταιριάζουν στο περιεχόμενο της εικόνας.  
3. Να αφαιρέσετε περιττά μεταδεδομένα.  

Αυτά τα βήματα διατηρούν την οπτική ποιότητα ενώ μειώνουν το μέγεθος του αρχείου — ιδανικό για web και κινητές εφαρμογές.
