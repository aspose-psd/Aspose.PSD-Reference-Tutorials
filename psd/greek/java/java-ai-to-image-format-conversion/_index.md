## Συχνές Ερωτήσεις

**Q: Μπορώ να χρησιμοποιήσω το Aspose.PSD για εμπορικές εφαρμογές Java;**  
A: Ναι, με έγκυρη άδεια Aspose. Διατίθεται δωρεάν δοκιμή για αξιολόγηση.

**Q: Υποστηρίζει η βιβλιοθήκη αρχεία AI με ενματωμένες γραμματοσειρές;**  
A: Απολύτως. Το Aspose.PSD διαβάζει ενσωματωμένες γραμματοσειρές και διατηρεί την απόδοση του κειμένου όταν είναι δυνατόν.

**Q: Τι κάνω αν χρειαστεί να μετατρέψω μια μεγάλη παρτίδα αρχείων AI;**  
A: Χρησιμοποιήστε έναν βρόχο για να φορτώσετε κάθε αρχείο και να καλέσετε τη μέθοδο `save`. Η βιβλιοθήκη είναι βελτιστοποιημένη για επεξεργασία παρτίδας.

**Q: Υπάρχουν περιορισμοί μεγέθους για αρχεία AI;**  
A: Η βιβλιοθήκη διαχειρίζεται αρχεία έως μερικές εκατοντάδες megabytes, περιοριζόμενη μόνο από το μέγεθος της μνήμης heap της JVM.

**Q: Πώς διατηρώ τη διαφάνεια κατά τη μετατροπή σε PNG;**  
A: Βεβαιωθείτε ότι χρησιμοποιείτε το `PngOptions` με `ColorType = PngColorType.TrueColorWithAlpha`.
