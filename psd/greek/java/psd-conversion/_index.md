## Συχνές Ερωτήσεις

**Q: Πώς μετατρέπω PSD σε TIFF χρησιμοποιώντας Aspose.PSD για Java;**  
A: Χρησιμοποιήστε την κλάση `PsdImage` για να φορτώσετε το αρχείο PSD και καλέστε τη μέθοδο `save` με `TiffOptions`. Το tutorial “Convert CMYK PSD to CMYK TIFF” δείχνει τον ακριβή κώδικα.

**Q: Μπορώ να εξάγω εικόνες με πολυ‑νηματικό τρόπο;**  
A: Ναι. Ο οδηγός “Export Images in Multi‑Threaded Environment” δείχνει πώς να εκτελείτε με ασφάλεια μετατροπές σε ξεχωριστά νήματα.

**Q: Πώς μετατρέπω στρώματα εικόνας GIF σε TIFF;**  
A: Φορτώστε το GIF με `Image.load`, επαναλάβετε τα frames του και αποθηκεύστε το καθένα ως σελίδα TIFF χρησιμοποιώντας `TiffOptions`. Δείτε το tutorial “Convert GIF Image Layers to TIFF”.

**Q: Υποστηρίζει το Aspose.PSD την επεξεργασία εικόνας Java για διαχείριση χρώματος;**  
A: Απόλυτα. Μπορείτε να χρησιμοποιήσετε προεπιλεγμένα προφίλ ή να παρέχετε προσαρμοσμένα ICC προφίλ, όπως εξηγείται στις ενότητες “Color Conversion using Default Profiles” και “Color Conversion using ICC Profiles”.

**Q: Πού μπορώ να βρω περισσότερα παραδείγματα για την εξαγωγή εικόνων;**  
A: Η σελίδα “Export Images in Multi‑Threaded Environment” περιέχει λεπτομερή παραδείγματα για αποδοτική εξαγωγή εικόνων.
