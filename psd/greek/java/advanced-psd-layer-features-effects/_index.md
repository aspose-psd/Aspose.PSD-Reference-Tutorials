## Συχνές Ερωτήσεις

**Q: Χρειάζομαι επί πληρωμή άδεια για να χρησιμοποιήσω το Aspose.PSD for Java;**  
A: Διατίθεται δωρεάν δοκιμή για ανάπτυξη και δοκιμές, αλλά απαιτείται εμπορική άδεια για παραγωγικές εγκαταστάσεις.

**Q: Μπορώ να εφαρμόσω πολλαπλά εφέ, όπως εσωτερική σκιά και εξωτερική λάμψη, στο ίδιο στρώμα;**  
A: Ναι. Τα εφέ μπορούν να στοιβάζονται προσθέτοντάς τα διαδοχικά στη συλλογή εφέ του στρώματος.

**Q: Είναι δυνατόν η επεξεργασία ενός αρχείου PSD χωρίς να χαθούν οι πληροφορίες των στρωμάτων;**  
A: Απόλυτα. Το Aspose.PSD διατηρεί όλα τα μεταδεδομένα στρωμάτων, μάσκες και στρώματα προσαρμογής όταν αποθηκεύετε τις αλλαγές.

**Q: Ποιες εκδόσεις Java υποστηρίζονται από την τελευταία βιβλιοθήκη Aspose.PSD;**  
A: Java 8, 11 και νεότερες εκδόσεις LTS υποστηρίζονται πλήρως.

**Q: Πώς να διαχειριστώ μεγάλα αρχεία PSD ώστε να αποφύγω προβλήματα μνήμης;**  
A: Χρησιμοποιήστε την υπερφόρτωση `Load` της κλάσης `PsdImage` με ένα `FileStream` και ενεργοποιήστε τη lazy loading όπου είναι κατάλληλο.
