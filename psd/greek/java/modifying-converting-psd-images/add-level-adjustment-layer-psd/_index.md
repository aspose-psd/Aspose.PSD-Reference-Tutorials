# Προσθήκη Στρώματος Ρύθμισης Επιπέδων σε PSD

## Εισαγωγή
Αν ψάχνετε για **πώς να ρυθμίσετε τα επίπεδα** στα έγγραφα Photoshop, το Στρώμα Ρύθμισης Επιπέδων είναι το τέλειο εργαλείο. Σας επιτρέπει να ρυθμίσετε ακριβώς τις σκιές, τις μεσαίες αποχρώσεις και τις φωτεινές περιοχές χωρίς να αλλάξετε μόνιμα τα αρχικά pixel. Σε αυτό το tutorial θα δούμε πώς να προσθέσουμε ένα Στρώμα Ρύθμισης Επιπέδων σε αρχείο PSD χρησιμοποιώντας το Aspose.PSD for Java, ώστε να επιτύχετε επαγγελματικό έλεγχο τόνου σε λίγα μόνο βήματα.

## Γρήγορες Απαντήσεις
- **Τι κάνει ένα Στρώμα Ρύθμισης Επιπέδων;** Τροποποιεί το φάσμα τόνων μιας εικόνας μη καταστροφικά.  
- **Ποια βιβλιοθήκη χρησιμοποιείται;** Aspose.PSD for Java.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται άδεια για παραγωγή.  
- **Πόσο διαρκεί η υλοποίηση;** Περίπου 10‑15 λεπτά για μια βασική ρύθμιση.  
- **Μπορώ να ρυθμίσω πολλαπλά κανάλια;** Ναι, μπορείτε να ορίσετε τα επίπεδα εισόδου/εξόδου για κάθε κανάλι χρώματος ξεχωριστά.

## Τι είναι ένα Στρώμα Ρύθμισης Επιπέδων;
Ένα Στρώμα Ρύθμισης Επιπέδων σας επιτρέπει να διορθώσετε την ισορροπία τόνων μιας εικόνας ρυθμίζοντας τις σκιές εισόδου, τις μεσαίες αποχρώσεις και τις φωτεινές περιοχές, καθώς και τα επίπεδα εξόδου. Επειδή βρίσκεται σε ξεχωριστό στρώμα, μπορείτε να εναλλάσσετε την ορατότητά του ή να το διαγράψετε χωρίς να επηρεάσετε το υποκείμενο έργο.

## Γιατί να προσθέσετε ένα Στρώμα Ρύθμισης Επιπέδων με το Aspose.PSD;
- **Αυτοματοποίηση:** Ενσωματώστε τις ρυθμίσεις επιπέδων σε αγωγούς επεξεργασίας παρτίδας.  
- **Διαπλατφορμική:** Λειτουργεί σε οποιοδήποτε λειτουργικό σύστημα που υποστηρίζει Java.  
- **Ακρίβεια:** Πρόσβαση στις ρυθμίσεις κάθε καναλιού προγραμματιστικά για ακριβή αποτελέσματα.  

## Προαπαιτούμενα
1. Java Development Kit (JDK). Εάν δεν το έχετε, κατεβάστε το από την [ιστοσελίδα Oracle](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) ή χρησιμοποιήστε το OpenJDK.  
2. Βιβλιοθήκη Aspose.PSD for Java – λάβετε το πιο πρόσφατο JAR από αυτόν τον [σύνδεσμο λήψης](https://releases.aspose.com/psd/java/).  
3. Βασικές γνώσεις προγραμματισμού Java.  
4. Ένα IDE όπως IntelliJ IDEA, Eclipse ή NetBeans με το Aspose.PSD JAR προστιθέμενο στο classpath του έργου.

## Εισαγωγή Πακέτων
Πριν αρχίσουμε να γράφουμε τον κώδικά μας, πρέπει να εισάγουμε τα απαραίτητα πακέτα από τη βιβλιοθήκη Aspose.PSD. Δείτε πώς μπορείτε να το κάνετε:
```java
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.adjustmentlayers.LevelsLayer;
import com.aspose.psd.fileformats.psd.layers.layerresources.LevelChannel;
```
Αυτές οι εισαγωγές μας δίνουν πρόσβαση σε κλάσεις για φόρτωση αρχείων PSD, εργασία με στρώματα ρύθμισης επιπέδων και διαχείριση ρυθμίσεων μεμονωμένων καναλιών.

## Πώς να Ρυθμίσετε τα Επίπεδα σε Αρχείο PSD
Παρακάτω είναι ένας οδηγός βήμα‑βήμα που σας δείχνει ακριβώς **πώς να ρυθμίσετε τα επίπεδα** προγραμματιστικά.

### Βήμα 1: Ρύθμιση Διαδρομών Αρχείων
Ορίστε πού βρίσκεται το αρχικό PSD και πού θα αποθηκευτεί το επεξεργασμένο αρχείο.
```java
String dataDir = "Your Document Directory";
String sourceFileName = dataDir + "LevelsAdjustmentLayer.psd";
String psdPathAfterChange = dataDir + "LevelsAdjustmentLayerChanged.psd";
```
Αντικαταστήστε το `"Your Document Directory"` με το πραγματικό φάκελο στον υπολογιστή σας.

### Βήμα 2: Φόρτωση του Αρχείου PSD
Δημιουργήστε ένα αντικείμενο `PsdImage` από το αρχικό αρχείο.
```java
PsdImage im = (PsdImage) Image.load(sourceFileName);
```
Τώρα έχετε πλήρη πρόσβαση σε όλα τα στρώματα μέσα στο PSD.

### Βήμα 3: Επανάληψη μέσω των Στρωμάτων
Βρείτε το Στρώμα Ρύθμισης Επιπέδων που θέλετε να τροποποιήσετε.
```java
for (int i = 0; i < im.getLayers().length; i++) {
    if (im.getLayers()[i] instanceof LevelsLayer) {
        LevelsLayer levelsLayer = (LevelsLayer) im.getLayers()[i];
        // Further manipulation will go here...
    }
}
```
Ο έλεγχος `instanceof LevelsLayer` διασφαλίζει ότι δουλεύουμε μόνο με στρώματα ρύθμισης επιπέδων.

### Βήμα 4: Ρύθμιση των Ρυθμίσεων Καναλιού Επιπέδου
Προσαρμόστε τις τιμές εισόδου και εξόδου για το επιλεγμένο κανάλι.
```java
LevelChannel channel = levelsLayer.getChannel(0);
channel.setInputMidtoneLevel(2.0f);
channel.setInputShadowLevel((short) 10);
channel.setInputHighlightLevel((short) 230);
channel.setOutputShadowLevel((short) 20);
channel.setOutputHighlightLevel((short) 200);
```
- **Επίπεδο Εισόδου Μεσαίας Απόχρωσης:** Μετατοπίζει το εύρος των μεσαίων αποχρώσεων.  
- **Επίπεδο Εισόδου Σκιάς:** Σκοτεινώνει ή φωτίζει τις σκιές.  
- **Επίπεδο Εισόδου Φωτεινότητας:** Ελέγχει τα πιο φωτεινά τμήματα.  
- **Επίπεδα Εξόδου Σκιάς/Φωτεινότητας:** Ορίζουν το τελικό εύρος εξόδου.

Μη διστάσετε να πειραματιστείτε με διαφορετικές τιμές για να δείτε πώς επηρεάζουν την εικόνα.

### Βήμα 5: Αποθήκευση του Τροποποιημένου Αρχείου PSD
Αποθηκεύστε τις αλλαγές σας σε νέο αρχείο.
```java
im.save(psdPathAfterChange);
```
Θα βρείτε το ενημερωμένο PSD στην τοποθεσία που καθορίσατε στο `psdPathAfterChange`.

## Συχνά Προβλήματα και Λύσεις
- **Αρχείο δεν βρέθηκε:** Επαληθεύστε ότι το `dataDir` δείχνει στο σωστό φάκελο και ότι το αρχικό PSD υπάρχει.  
- **ClassCastException:** Βεβαιωθείτε ότι το αρχείο που φορτώνετε είναι πράγματι PSD· άλλες μορφές απαιτούν διαφορετικές κλάσεις.  
- **Σφάλματα άδειας:** Χρησιμοποιήστε μια έγκυρη άδεια Aspose.PSD για παραγωγικές εκδόσεις· η δοκιμαστική έκδοση λειτουργεί για ανάπτυξη.
