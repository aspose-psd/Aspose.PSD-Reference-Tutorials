# Προσθήκη Υπογραφής σε Εικόνα – Σχεδίαση Εικόνας σε Καμβά με Aspose.PSD for Java

## Εισαγωγή

Η προσθήκη χειρόγραφης ή ψηφιακής **προσθήκης υπογραφής σε εικόνα** είναι μια συχνή απαίτηση για συμβόλαια, τιμολόγια ή οποιοδήποτε έγγραφο που χρειάζεται απόδειξη αυθεντικότητας. Σε αυτό το tutorial θα δείτε πώς το Aspose.PSD for Java σας επιτρέπει να **σχεδιάζετε εικόνα σε καμβά** και να αντιμετωπίζετε την υπογραφή ως απλώς ένα ακόμη επίπεδο επικάλυψης. Θα περάσουμε από τη φόρτωση της βασικής εικόνας, τη φόρτωση του αρχείου υπογραφής, την αρχικοποίηση ενός graphics context, τη σχεδίαση της επικάλυψης και τέλος **αποθήκευση εικόνας ως PNG**. Στο τέλος θα έχετε ένα επαναχρησιμοποιήσιμο μοτίβο για οποιοδήποτε σενάριο **java image processing**, είτε πρόκειται για υπογραφή, υδατογράφημα ή λογότυπο.

- **Πώς να προσθέσω υπογραφή σε Java;** Φορτώστε το αρχείο υπογραφής ως `Image` και χρησιμοποιήστε `Graphics.drawImage`.

- **Μπορώ να επικάλυψω πολλαπλές εικόνες;** Ναι—επαναλάβετε την κλήση `drawImage` με διαφορετικές πηγές.

## Τι Είναι “Σχεδίαση Εικόνας σε Καμβά”;
Στην ορολογία του Aspose.PSD, η σχεδίαση μιας εικόνας σε καμβά σημαίνει το βάψιμο ενός αντικειμένου `Image` πάνω σε άλλο χρησιμοποιώντας ένα `Graphics` context. Αυτή η λειτουργία αποτελεί τη ραχοκοκαλιά των τεχνικών **overlay images java** όπως η προσθήκη υδατογραφημάτων, λογοτύπων ή υπογραφών.

## Προαπαιτούμενα
- Java Development Kit (JDK) 8 ή νεότερο.  
- Aspose.PSD for Java JAR προστιθέμενο στο classpath του έργου σας.  
- Δύο αρχεία εικόνας: μια βασική εικόνα (π.χ., `layers.psd`) και ένα γραφικό υπογραφής (`sample.psd`).  

## Εισαγωγή Πακέτων

```java
import com.aspose.psd.Graphics;
import static com.aspose.psd.GraphicsUnit.Point;
import com.aspose.psd.Image;
import com.aspose.psd.Point;

import com.aspose.psd.imageoptions.PngOptions;
```

## Βήμα 1: Φόρτωση Πρωτεύουσας και Δευτερεύουσας Εικόνας

```java
//ExStart:LoadImages
String dataDir = "Your Document Directory";

// Load the primary image (the canvas)
Image canvas = Image.load(dataDir + "layers.psd");

// Load the secondary image containing the signature graphics
Image signature = Image.load(dataDir + "sample.psd");
//ExEnd:LoadImages
```

> **Συμβουλή:** Κρατήστε και τις δύο εικόνες στην ίδια χρωματική λειτουργία (RGB) για να αποφύγετε απρόσμενες μεταβολές χρώματος κατά τη σχεδίαση.

## Βήμα 2: Αρχικοποίηση Graphics (initialize graphics java)

```java
//ExStart:InitializeGraphics
Graphics graphics = new Graphics(canvas);
//ExEnd:InitializeGraphics
```

Το αντικείμενο `Graphics` λειτουργεί σαν πινέλο που σας επιτρέπει να **σχεδιάζετε εικόνα σε καμβά**. Η αρχικοποίησή του με την πρωτεύουσα `Image` συνδέει όλες τις επόμενες εντολές σχεδίασης με αυτόν τον καμβά.

## Βήμα 3: Προσθήκη Υπογραφής στην Εικόνα (how to add signature)

```java
//ExStart:AddSignatureToImage
graphics.drawImage(
    signature,
    new Point(
        canvas.getHeight() - signature.getHeight(),   // X‑coordinate (bottom)
        canvas.getWidth() - signature.getWidth()      // Y‑coordinate (right)
    )
);
canvas.save(dataDir + "AddSignatureToImage_out.png", new PngOptions());
//ExEnd:AddSignatureToImage
```

Σε αυτό το απόσπασμα **επικάλυπουμε εικόνες java** τοποθετώντας την υπογραφή στην κάτω‑δεξιά γωνία. Προσαρμόστε τις τιμές του `Point` αν χρειάζεστε διαφορετική θέση.

## Συχνά Προβλήματα & Λύσεις
| Συμπτωμα | Αιτία | Διόρθωση |
|---------|-------|----------|
| Η υπογραφή εμφανίζεται παραμορφωμένη | Μη αντιστοιχία DPI μεταξύ καμβά και υπογραφής | Χρησιμοποιήστε `signature.resize` πριν τη σχεδίαση ή βεβαιωθείτε ότι και τα δύο αρχεία έχουν το ίδιο DPI. |
| Το αρχείο εξόδου είναι τεράστιο | Αποθήκευση χωρίς συμπίεση | Περάστε ένα ρυθμισμένο `PngOptions` με `CompressionLevel` ορισμένο σε υψηλότερη τιμή. |
| Δεν σχεδιάζεται τίποτα | Το Graphics δεν έχει διαγραφεί | Καλέστε `graphics.dispose()` μετά τη σχεδίαση (προαιρετικό, αλλά καλή πρακτική). |

## Πρόσθετες Συμβουλές για Πραγματική Χρήση

- **Πολλαπλές υπογραφές:** Καλέστε `graphics.drawImage` επανειλημμένα με διαφορετικά αντικείμενα `Image` και συντεταγμένες.  
- **Έλεγχος διαφάνειας:** Χρησιμοποιήστε `graphics.setOpacity(float opacity)` πριν τη σχεδίαση για να κάνετε την υπογραφή ημιδιαφανή.  
- **Περιστροφή της υπογραφής:** Εφαρμόστε `graphics.rotateTransform(angle)` αν χρειάζεστε κεκλιμένη υπογραφή.  
- **Αποθήκευση σε άλλες μορφές:** Αντικαταστήστε το `PngOptions` με `JpegOptions` ή `BmpOptions` για έξοδο JPEG, BMP κ.λπ.
