Θα αποκτήσετε ένα επαναχρησιμοποιήσιμο απόσπασμα κώδικα που μπορεί να ενσωματωθεί σε οποιοδήποτε αυτοματοποιημένο pipeline επεξεργασίας εικόνας. Ας μπει το μανίκι και ας ξεκινήσουμε!

## Εισαγωγή Πακέτων

Το πρώτο βήμα στην πορεία του κώδικα είναι η εισαγωγή των απαραίτητων πακέτων. Πριν χρησιμοποιήσετε τις λειτουργίες που παρέχει η Aspose.PSD, πρέπει να βεβαιωθείτε ότι η βιβλιοθήκη βρίσκεται στο classpath σας. Δείτε πώς:

```java
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.adjustmentlayers.BrightnessContrastLayer;
```

Ολοκληρώνοντας αυτά τα βήματα, θέτετε τη βάση για αποτελεσματική εργασία με αρχεία PSD!

Τώρα που έχουμε όλα έτοιμα, ήρθε η ώρα να περάσουμε στο κυρίως μέρος του tutorial: ρύθμιση φωτεινότητας και αντίθεσης σε στρώματα PSD. Θα διασπάσουμε αυτή τη διαδικασία σε σαφή βήματα ώστε να μπορείτε να ακολουθήσετε εύκολα.

## Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ξεκινήστε ορίζοντας τον κατάλογο όπου βρίσκονται τα αρχεία PSD. Αυτό το βήμα βοηθά στην οργάνωση των αρχείων σας αποδοτικά.

```java
String dataDir = "Your Document Directory";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή προς τον φάκελο των αρχείων PSD.

## Βήμα 2: Καθορίστε τα Ονόματα Αρχείων Πηγής και Προορισμού

Στη συνέχεια, πρέπει να καθορίσετε το όνομα του αρχείου πηγής του PSD και το αρχείο προορισμού όπου θα αποθηκευτεί το επεξεργασμένο PSD.

```java
String sourceFileName = dataDir + "BrightnessContrastModern.psd";
String psdPathAfterChange = dataDir + "BrightnessContrastModernChanged.psd";
```

Σε αυτό το παράδειγμα, υποθέτουμε ότι έχετε ένα αρχείο PSD με όνομα `BrightnessContrastModern.psd` στον κατάλογό σας.

## Βήμα 3: Φορτώστε το Αρχείο PSD

Τώρα ήρθε η ώρα να φορτώσετε το αρχείο PSD στην εφαρμογή σας ώστε να μπορείτε να το επεξεργαστείτε.

```java
PsdImage im = (PsdImage) Image.load(sourceFileName);
```

Αυτή η γραμμή κώδικα δημιουργεί μια παρουσία του `PsdImage` που αντιπροσωπεύει το αρχείο PSD σας. Με αυτόν τον τρόπο μπορείτε πλέον να έχετε πρόσβαση σε όλα τα στρώματα του PSD.

## Βήμα 4: Επανάληψη μέσω Στρωμάτων

Το επόμενο βήμα περιλαμβάνει την επανάληψη μέσω κάθε στρώματος του αρχείου PSD για να βρείτε και να τροποποιήσετε τις ρυθμίσεις φωτεινότητας και αντίθεσης.

```java
for(int i = 0; i < im.getLayers().length; i++) {
    if (im.getLayers()[i] instanceof BrightnessContrastLayer) {
        BrightnessContrastLayer brightnessContrastLayer = (BrightnessContrastLayer)im.getLayers()[i];
```

Ο βρόχος `for` διασχίζει κάθε στρώμα του PSD. Ελέγχουμε αν ένα στρώμα είναι μια παρουσία του `BrightnessContrastLayer`. Αυτό είναι απαραίτητο για να διασφαλίσουμε ότι προσπαθείτε να αλλάξετε τη φωτεινότητα μόνο στα σωστά στρώματα.

## Βήμα 5: Ρύθμιση Φωτεινότητας και Αντίθεσης

Μέσα στον βρόχο, μπορείτε τώρα να ορίσετε τη φωτεινότητα και την αντίθεση για κάθε `BrightnessContrastLayer`.

```java
        brightnessContrastLayer.setBrightness(50);
        brightnessContrastLayer.setContrast(50);
    }
}
```

Σε αυτό το παράδειγμα, ορίζουμε τη φωτεινότητα και την αντίθεση στο `50`. Μπορείτε να προσαρμόσετε αυτές τις τιμές ανάλογα με τις απαιτήσεις σας. Ένας υψηλότερος αριθμός αυξάνει τη φωτεινότητα/αντίθεση, ενώ ένας χαμηλότερος αριθμός τη μειώνει.

## Βήμα 6: Αποθήκευση Αλλαγών

Το τελευταίο βήμα είναι η αποθήκευση των αλλαγών στο αρχείο PSD. Θα πρέπει να γράψετε την τροποποιημένη εικόνα πίσω στον καθορισμένο προορισμό.

```java
im.save(psdPathAfterChange);
```

Αυτή η γραμμή κώδικα αποθηκεύει το επεξεργασμένο αρχείο PSD με τις νέες ρυθμίσεις φωτεινότητας και αντίθεσης.
