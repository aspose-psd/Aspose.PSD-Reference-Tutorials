## Πώς να Εισάγετε Εικόνες PSD σε Στρώματα
Παρακάτω υπάρχει ένας σαφής, αριθμημένος οδηγός που εξηγεί **πώς να προσθέσετε εικόνα** σε ένα στρώμα PSD, **να ορίσετε τις συντεταγμένες του στρώματος**, και **να γεμίσετε το χρώμα του στρώματος PSD**.

### Βήμα 1: Εισαγωγή Απαιτούμενων Πακέτων
Αρχικά, φέρνουμε τις κλάσεις Aspose.PSD που θα χρειαστούμε. Αυτό θέτει το σκηνικό για τον υπόλοιπο κώδικα.

```java
import com.aspose.psd.Color;
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.Point;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
```

### Βήμα 2: Ορισμός Καταλόγου Εγγράφου
Ορίστε πού βρίσκεται το πηγαίο PSD σας και πού θα αποθηκευτεί το αποτέλεσμα.

```java
String dataDir = "Your Document Directory";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή στο σύστημά σας όπου βρίσκονται τα αρχεία PSD.

### Βήμα 3: Φόρτωση του Αρχείου PSD
Ανοίξτε το PSD ώστε να δουλέψουμε με τα στρώματά του.

```java
PsdImage image = (PsdImage) Image.load(dataDir + "sample.psd");
```

Βεβαιωθείτε ότι το `"sample.psd"` ταιριάζει με το όνομα αρχείου που θέλετε να επεξεργαστείτε.

### Βήμα 4: Εξαγωγή του Στόχου Στρώματος
Επιλέξτε το στρώμα που θα λάβει τη νέα εικόνα. Εδώ χρησιμοποιούμε το δεύτερο στρώμα (δείκτης 1).

```java
Layer layer = image.getLayers()[1];
```

Αν χρειάζεστε διαφορετικό στρώμα, απλώς αλλάξτε τον δείκτη.

### Βήμα 5: Δημιουργία Νέας Εικόνας για Εισαγωγή
Τώρα θα **προσθέσουμε εικόνα σε στρώμα PSD** δημιουργώντας ένα νέο `PsdImage` πάνω στο οποίο θα σχεδιάσουμε.

```java
PsdImage drawImage = new PsdImage(200, 200);
```

Μπορείτε να προσαρμόσετε το πλάτος και το ύψος ώστε να ταιριάζουν με την πηγαία εικόνα σας.

### Βήμα 6: Γέμισμα της Επιφάνειας της Εικόνας (Ορισμός Χρώματος Στρώματος)
Ας **γεμίσουμε το χρώμα του στρώματος PSD** με ένα φωτεινό κίτρινο φόντο. Αυτό δείχνει πώς να ορίσετε ένα στερεό χρώμα πριν το σχεδιάσετε.

```java
Graphics g = new Graphics(drawImage);
g.clear(Color.getYellow());
```

Μη διστάσετε να αντικαταστήσετε το `Color.getYellow()` με οποιοδήποτε άλλο `Color` προτιμάτε.

### Βήμα 7: Σχεδίαση της Εικόνας στο Στρώμα (Ορισμός Συντεταγμένων Στρώματος)
Αυτή είναι η ουσία του **πώς να προσθέσετε εικόνα** – τοποθετούμε τη νεοδημιουργημένη εικόνα στο επιλεγμένο στρώμα σε συγκεκριμένες συντεταγμένες.

```java
layer.drawImage(new Point(10, 10), drawImage);
```

Η κλήση `Point(10, 10)` **ορίζει τις συντεταγμένες του στρώματος** (X = 10, Y = 10). Αλλάξτε αυτές τις τιμές για να τοποθετήσετε την εικόνα ακριβώς όπου τη χρειάζεστε.

### Βήμα 8: Αποθήκευση του Ενημερωμένου Αρχείου PSD
Τέλος, γράψτε τις αλλαγές πίσω στο δίσκο.

```java
image.save(dataDir + "ImportImageToPSDLayer_out.psd");
```

Δώστε στο αρχείο εξόδου ένα περιγραφικό όνομα· το παράδειγμα προσθέτει `_out` για να διατηρηθεί το αρχικό ανέπαφο.

## Συχνά Προβλήματα και Λύσεις
- **Η εικόνα εμφανίζεται κενή** – Βεβαιωθείτε ότι κάλεσατε `Graphics.clear()` πριν το σχεδιάσετε· διαφορετικά ο καμβάς μπορεί να είναι διαφανής.  
- **Τροποποιείται το λάθος στρώμα** – Θυμηθείτε ότι οι δείκτες των στρωμάτων ξεκινούν από 0. Ελέγξτε ξανά τον δείκτη που χρησιμοποιείτε στο `getLayers()`.  
- **Μη υποστηριζόμενο προφίλ χρώματος** – Το Aspose.PSD διαχειρίζεται τα περισσότερα προφίλ, αλλά αν παρατηρήσετε μεταβολές χρώματος, δοκιμάστε να μετατρέψετε την πηγαία εικόνα σε sRGB πριν την εισαγωγή.
