# Δημιουργία Νέου Στρώματος PSD και Διαχείριση Ημερομηνίας/Ώρας Δημιουργίας σε Java

## Εισαγωγή
Όταν εργάζεστε προγραμματιστικά με αρχεία Photoshop (PSD), η δυνατότητα **δημιουργίας νέων αντικειμένων στρώματος PSD** και η παρακολούθηση των χρονικών σημείων δημιουργίας τους αποτελεί πραγματικό «game‑changer». Είτε δημιουργείτε σύστημα ελέγχου εκδόσεων για σχέδια, αυτοματοποιείτε μαζικές επεξεργασίες, είτε χρειάζεστε ένα αρχείο ελέγχου για συνεργατικά έργα, η γνώση του πώς να διαβάζετε και να ορίζετε την ημερομηνία δημιουργίας του στρώματος σας επιτρέπει να διατηρείτε πλήρη προέλευση κάθε αλλαγής. Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία χρησιμοποιώντας το Aspose.PSD for Java — από τη φόρτωση ενός PSD, την ανάκτηση της ημερομηνίας δημιουργίας ενός στρώματος, την επικύρωσή της, μέχρι την προσθήκη ενός ολοκαίνουργιου στρώματος προσαρμογής.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται αρχεία PSD σε Java;** Aspose.PSD for Java  
- **Μπορώ να διαβάσω την ημερομηνία δημιουργίας ενός στρώματος;** Ναι, χρησιμοποιώντας `layer.getLayerCreationDateTime()`  
- **Είναι δυνατόν να προσθέσω νέο στρώμα προσαρμογής;** Απόλυτα – `im.addLevelsAdjustmentLayer()` δημιουργεί ένα  
- **Χρειάζομαι άδεια για παραγωγική χρήση;** Απαιτείται εμπορική άδεια για μη‑δοκιμαστικές εγκαταστάσεις  
- **Ποια έκδοση Java υποστηρίζεται;** JDK 8 ή νεότερη  

## Τι σημαίνει «create new PSD layer»;
Η δημιουργία νέου στρώματος PSD σημαίνει την προγραμματιστική εισαγωγή ενός φρέσκου αντικειμένου στρώματος — όπως ένα στρώμα προσαρμογής, κειμένου ή pixel — σε ένα υπάρχον έγγραφο PSD. Αυτή η λειτουργία σας επιτρέπει να επεκτείνετε ή να τροποποιήσετε την εικόνα χωρίς να ανοίξετε χειροκίνητα το Photoshop.

## Γιατί να διαχειρίζεστε την Ημερομηνία/Ώρα Δημιουργίας του στρώματος;
Η παρακολούθηση της Ημερομηνίας/Ώρας δημιουργίας κάθε στρώματος σας βοηθά:
- **Έλεγχος εκδόσεων** – γνωρίζετε ακριβώς πότε προστέθηκε ένα στρώμα.  
- **Συγχρονισμός πόρων** μεταξύ ομάδων συγκρίνοντας χρονικές σφραγίδες.  
- **Αυτοματοποίηση ροών εργασίας** που εξαρτώνται από κανόνες βάσει χρόνου (π.χ., απόκρυψη στρωμάτων παλαιότερων από ένα μήνα).

## Εισαγωγή Πακέτων
Πρώτα, εισάγετε τις κλάσεις Aspose.PSD και τις βοηθητικές βιβλιοθήκες Java που θα χρειαστείτε. Αυτές οι εισαγωγές σας δίνουν πρόσβαση στη διαχείριση εικόνας, τη διαχείριση στρωμάτων και τη μορφοποίηση ημερομηνιών.

```java
import com.aspose.psd.Image;
import com.aspose.psd.examples.Utils.Assert;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
import java.text.DateFormat;
import java.text.SimpleDateFormat;
import java.util.Date;
```

## Βήμα 1: Ρύθμιση Καταλόγου Εγγράφου
Καθορίστε το φάκελο που περιέχει το PSD με το οποίο θέλετε να εργαστείτε. Αντικαταστήστε το placeholder με την απόλυτη διαδρομή στο σύστημά σας.

```java
String dataDir = "Your Document Directory";
```

## Βήμα 2: Φόρτωση του Αρχείου PSD
Δημιουργήστε μια παρουσία `PsdImage` φορτώνοντας το αρχείο-στόχο. Αυτό το αντικείμενο είναι το σημείο εισόδου για όλες τις λειτουργίες στρώματος.

```java
String sourceName = dataDir + "OneLayer.psd";
PsdImage im = (PsdImage) Image.load(sourceName);
```

## Βήμα 3: Πρόσβαση στο Στρώμα και στην Ημερομηνία Δημιουργίας του
Αποκτήστε το πρώτο στρώμα (δείκτης 0) και ανακτήστε τη χρονική σφραγίδα δημιουργίας του. Αυτή είναι η ημερομηνία που θα συγκρίνετε ή θα καταγράψετε αργότερα.

```java
Layer layer = im.getLayers()[0];
Date creationDateTime = layer.getLayerCreationDateTime();
```

## Βήμα 4: Μορφοποίηση της Ημερομηνίας Δημιουργίας
Μετατρέψτε το ακατέργαστο αντικείμενο `Date` σε αναγνώσιμη αλφαριθμητική συμβολοσειρά. Προσαρμόστε το μοτίβο αν προτιμάτε διαφορετική μορφή.

```java
DateFormat dateFormat = new SimpleDateFormat("yyyy/MM/dd HH:mm:ss");
```

## Βήμα 5: Επικύρωση της Ημερομηνίας Δημιουργίας
Για επίδειξη, συγκρίνουμε την ανακτημένη ημερομηνία με μια αναμενόμενη τιμή. Σε πραγματικά έργα μπορεί να συγκρίνετε με εγγραφή βάσης δεδομένων ή αρχείο ρυθμίσεων.

```java
Date expectedDateTime = new Date("2018/7/17 8:57:24");
Assert.areEqual(expectedDateTime, creationDateTime);
```

## Βήμα 6: Δημιουργία Νέου Στρώματος
Τώρα δημιουργούμε πραγματικά **νέα στρώματα PSD**. Εδώ προσθέτουμε ένα στρώμα προσαρμογής Levels, το οποίο σας επιτρέπει να ρυθμίσετε τις τόνους χωρίς να αλλάξετε τα αρχικά pixel.

```java
Date now = new Date();
Layer createdLayer = im.addLevelsAdjustmentLayer();
```

> **Pro tip:** Η μεταβλητή `now` καταγράφει τη στιγμή που προσθέτετε το στρώμα, την οποία μπορείτε αργότερα να αποθηκεύσετε ως μεταδεδομένα αν χρειάζεστε προσαρμοσμένη χρονική σφραγίδα.

## Συνηθισμένα Προβλήματα και Λύσεις
| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| `NullPointerException` στο `layer.getLayerCreationDateTime()` | Το PSD δεν έχει στρώματα ή ο δείκτης στρώματος είναι εκτός εύρους. | Επαληθεύστε ότι `im.getLayers().length > 0` πριν την πρόσβαση. |
| Ασυμφωνία ημερομηνίας στην επικύρωση | Ο κατασκευαστής `Date` αναλύει συμβολοσειρές με τρόπο εξαρτώμενο από την τοπική ρύθμιση. | Χρησιμοποιήστε `SimpleDateFormat.parse("2018/07/17 08:57:24")` για αξιόπιστη ανάλυση. |
| Το νέο στρώμα δεν εμφανίζεται στο Photoshop | Το στρώμα προσαρμογής μπορεί να είναι κρυφό από προεπιλογή. | Καλέστε `createdLayer.setVisible(true);` μετά τη δημιουργία. |
