Οι παρακάτω εισαγωγές φέρνουν τις βασικές κλάσεις του Aspose.PSD που χρειάζονται για τη φόρτωση αρχείων PSD και τη διαμόρφωση επιλογών εξαγωγής PNG.

import com.aspose.psd.Image;

## Πώς να αποθηκεύσετε PSD ως PNG χρησιμοποιώντας το Aspose.PSD for Java;

Η κλάση `PsdImage` αντιπροσωπεύει ένα έγγραφο Photoshop στη μνήμη, ενώ το `ImageSaveOptions` μαζί με το `SaveFormat` καθορίζουν τη ζητούμενη μορφή εξόδου και τις ρυθμίσεις συμπίεσης. Φορτώνοντας ένα PSD και καλώντας τη μέθοδο αποθήκευσης με επιλογές PNG, μπορείτε να μετατρέψετε το αρχείο με μία μόνο, αποδοτική κλήση.

Φορτώστε το αρχείο PSD με `new PsdImage("source.psd")` και καλέστε `psd.save("output.png", ImageSaveOptions.createSaveOptions(SaveFormat.Png))`. Αυτή η κλήση μίας γραμμής διαχειρίζεται την εξομάλυνση των στρώσεων, τη διατήρηση του προφίλ χρώματος και τη συμπίεση PNG αυτόματα. Για λειτουργίες δέσμης, τοποθετήστε την κλήση μέσα σε βρόχο πάνω στα αρχεία προέλευσης.

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ορίστε τη διαδρομή για τον κατάλογο εγγράφων σας, όπου βρίσκεται το αρχείο PSD:

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Καθορίστε τις Διαδρομές Πηγής και Προορισμού

Ορίστε τις διαδρομές για το αρχείο PSD πηγής και το αρχείο προορισμού όπου θα αποθηκευτεί η εικόνα:

```java
String sourceFile = dataDir + "sample.psd";
String destName = dataDir + "result.png";
```

### Βήμα 3: Φόρτωση Εικόνας PSD

Φορτώστε την εικόνα PSD χρησιμοποιώντας το Aspose.PSD:

```java
Image image = Image.load(sourceFile);
```

### Βήμα 4: Αποθήκευση Εικόνας με Επιλογές

`PsdImage` είναι η βασική κλάση του Aspose.PSD που αντιπροσωπεύει ένα έγγραφο Photoshop στη μνήμη. Μετατρέψτε την φορτωμένη εικόνα σε `PsdImage` και αποθηκεύστε την ως αρχείο PNG:

```java
PsdImage psdImage = (PsdImage)image;
psdImage.save(destName, new PngOptions());
```

Επαναλάβετε αυτά τα βήματα για κάθε εικόνα που θέλετε να αποθηκεύσετε, εξασφαλίζοντας μια αδιάκοπη διαδικασία με το Aspose.PSD.

## Συχνά Προβλήματα και Λύσεις

- **OutOfMemoryError σε μεγάλα αρχεία** – Ενεργοποιήστε τη ροή δεδομένων χρησιμοποιώντας `PsdImage.load(inputStream, true)` για να αποφύγετε τη φόρτωση ολόκληρου του αρχείου στη μνήμη RAM.
- **Απουσία διαφάνειας** – Βεβαιωθείτε ότι χρησιμοποιείτε `PngOptions` με `ColorType = PngColorType.Rgba` για να διατηρήσετε το κανάλι άλφα.
- **Λανθασμένα χρώματα** – Επαληθεύστε ότι το προφίλ χρώματος του PSD πηγής είναι ενσωματωμένο· το Aspose.PSD το εφαρμόζει αυτόματα κατά την εξαγωγή.
