## Pertanyaan yang Sering Diajukan

**Q: Apakah saya memerlukan lisensi berbayar untuk menggunakan Aspose.PSD for Java?**  
A: Versi percobaan gratis tersedia untuk pengembangan dan pengujian, namun lisensi komersial diperlukan untuk penerapan produksi.

**Q: Bisakah saya menerapkan beberapa efek, seperti inner shadow dan outer glow, pada lapisan yang sama?**  
A: Ya. Efek dapat ditumpuk dengan menambahkannya secara berurutan ke koleksi efek lapisan.

**Q: Apakah memungkinkan mengedit file PSD tanpa kehilangan informasi lapisan?**  
A: Tentu saja. Aspose.PSD mempertahankan semua metadata lapisan, mask, dan lapisan penyesuaian saat Anda menyimpan perubahan.

**Q: Versi Java apa yang didukung oleh pustaka Aspose.PSD terbaru?**  
A: Java 8, 11, dan rilis LTS yang lebih baru sepenuhnya didukung.

**Q: Bagaimana cara menangani file PSD besar agar tidak terjadi masalah memori?**  
A: Gunakan overload `Load` kelas `PsdImage` dengan `FileStream` dan aktifkan lazy loading bila diperlukan.
