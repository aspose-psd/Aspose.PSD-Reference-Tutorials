## Co je PSB a proč jej převádět?
PSB (Photoshop Big) je formát Adobe pro soubory, které překračují limit 2 GB běžného PSD. Velké návrhy, vysoce rozlišené kompozice nebo vícevrstvé umělecké dílo často používají PSB. Převod PSB do lehčích formátů, jako je JPG nebo PDF, usnadňuje sdílení, vkládání na webové stránky nebo archivaci bez ztráty podstatných vizuálních informací.

## Proč použít Aspose.PSD pro Java?
- **Plná věrnost vrstev** – všechny vrstvy, masky a efekty jsou zachovány při převodu do PSD.  
- **Vysoce výkonný rendering** – optimalizováno pro velké soubory, ideální pro PSB.  
- **Podpora více formátů** – jedním voláním API můžete exportovat do JPG, PDF, PNG, TIFF a dalších.  
- **Žádná závislost na Photoshopu** – funguje v jakémkoli serverovém prostředí Java.

## Často kladené otázky

**Q: Mohu převést PSB soubor chráněný heslem?**  
A: Ano. Aspose.PSD vám umožní otevřít šifrované PSB soubory zadáním hesla během operace načtení.

**Q: Ztratí se při převodu PSB na JPG informace o vrstvách?**  
A: Převod na JPG zploští obrázek, takže data vrstev jsou sloučena. Použijte PDF nebo PSD, pokud potřebujete vrstvy zachovat.

**Q: Existuje limit velikosti PSB souborů při použití Aspose.PSD?**  
A: Knihovna je navržena tak, aby zvládla velmi velké PSB soubory (několik gigabajtů), pokud má vaše JVM dostatek heap paměti.

**Q: Potřebuji samostatnou licenci pro každý výstupní formát?**  
A: Ne. Jedna licence Aspose.PSD for Java pokrývá všechny podporované exportní formáty, včetně JPG, PDF a PSD.

**Q: Jak zlepšit výkon při hromadném převodu mnoha PSB souborů?**  
A: Opakovaně používejte stejnou instanci `PsdImage`, pokud je to možné, a zvažte zpracování souborů paralelně pomocí streamů s vhodným nastavením paměti.
