## Perguntas Frequentes

**Q: Como converto PSD para TIFF usando Aspose.PSD para Java?**  
A: Use a classe `PsdImage` para carregar o arquivo PSD e chame o método `save` com `TiffOptions`. O tutorial “Convert CMYK PSD to CMYK TIFF” mostra o código exato.

**Q: Posso exportar imagens de forma multi‑thread?**  
A: Sim. O guia “Export Images in Multi‑Threaded Environment” demonstra como executar conversões com segurança em threads separadas.

**Q: Como converter camadas de imagem GIF para TIFF?**  
A: Carregue o GIF com `Image.load`, itere pelos seus quadros e salve cada um como uma página TIFF usando `TiffOptions`. Veja o tutorial “Convert GIF Image Layers to TIFF”.

**Q: O Aspose.PSD suporta processamento de imagens java para gerenciamento de cores?**  
A: Absolutamente. Você pode usar perfis padrão ou fornecer perfis ICC personalizados, conforme explicado nas seções “Color Conversion using Default Profiles” e “Color Conversion using ICC Profiles”.

**Q: Onde posso encontrar mais exemplos de como exportar imagens?**  
A: A página “Export Images in Multi‑Threaded Environment” contém exemplos detalhados sobre como exportar imagens de forma eficiente.
