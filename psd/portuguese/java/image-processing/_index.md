## Respostas Rápidas
- **O que é um filtro mediana?** Um filtro não‑linear que substitui cada pixel pelo valor mediano de seus pixels vizinhos, reduzindo efetivamente o ruído enquanto preserva as bordas.  
- **Por escolher Aspose.PSD para Java?** Ele fornece uma API abrangente para PSD, PNG, JPEG e muitos outros formatos sem exigir bibliotecas nativas.  
- **Posso combinar mediana com outros filtros?** Sim—misture a mediana com filtros Gaussianos ou Wiener para ajustar finamente a redução de ruído e a preservação de detalhes.  
- **Preciso de uma licença?** Um teste gratuito está disponível; uma licença comercial é necessária para uso em produção.  
- **Quais versões do Java são suportadas?** Java 8 e superiores, incluindo Java 17 LTS.

## Perguntas Frequentes

**Q: Posso usar o filtro mediana em arquivos PSD grandes sem ficar sem memória?**  
A: Sim. Aspose.PSD processa camadas sequencialmente e fornece opções de streaming para manter o uso de memória baixo.

**Q: Como um filtro mediana difere de um filtro Gaussiano?**  
A: Um filtro mediana é não‑linear e se destaca na remoção de ruído sal‑e‑pimenta enquanto preserva as bordas, enquanto um filtro Gaussiano é linear e suaviza uniformemente.

**Q: É possível combinar **apply gaussian filter** e filtragem mediana em um único pipeline?**  
A: Absolutamente. Você pode encadear os filtros—aplicar um desfoque Gaussiano primeiro para reduzir ruído de alta frequência, então um filtro mediana para limpeza que preserva bordas.

**Q: O Aspose.PSD suporta operações de **crop psd file** em objetos inteligentes?**  
A: Sim. O recorte funciona em qualquer camada raster, incluindo as incorporadas em objetos inteligentes.

**Q: Qual versão do Java é necessária para estes tutoriais?**  
A: Java 8 ou superior é necessário; a biblioteca é totalmente compatível com Java 11, 17 e versões LTS posteriores.
