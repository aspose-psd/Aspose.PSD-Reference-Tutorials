## FAQ

**Q : Puis‑je ajouter une miniature EXIF à un JPEG sans affecter la qualité de l’image originale ?**  
R : Oui. Aspose.PSD écrit la miniature uniquement dans la section des métadonnées, laissant les données principales de l’image intactes.

**Q : Comment écrire des données EXIF en Java avec Aspose.PSD ?**  
R : Utilisez la classe `PsdImage` pour accéder à la propriété `ExifData`, créez ou modifiez les balises, puis appelez `Save()`.

**Q : Quelle est la meilleure façon de lire les balises EXIF en Java pour un traitement en masse ?**  
R : Parcourez les fichiers avec `PsdImage`, lisez `ExifData.Tags` et stockez les valeurs nécessaires dans une collection pour une analyse ultérieure.

**Q : Est‑il possible de modifier les balises EXIF JPEG en Java sans corrompre le fichier ?**  
R : Absolument. L’API met à jour uniquement le segment EXIF, préservant le reste de la structure JPEG.

**Q : Aspose.PSD prend‑il en charge l’écriture de balises EXIF personnalisées ?**  
R : Oui. Vous pouvez ajouter des balises non standard en spécifiant l’ID de la balise et la valeur via la classe `ExifTag`.
