## Qu'est‑ce que **comment fusionner des images** avec Aspose.PSD?
Fusionner des images signifie combiner deux ou plusieurs ressources visuelles dans un seul fichier PSD ou une sortie raster. Aspose.PSD pour Java offre des méthodes de haut niveau pour superposer des images, les fusionner et préserver toutes les fonctionnalités compatibles Photoshop sans perte de qualité.

## Pourquoi utiliser Aspose.PSD pour la fusion d'images en Java ?
- **Compatibilité totale avec Photoshop** – conserve les calques, masques et effets.
- **Pas besoin de Photoshop natif** – s'exécute entièrement sur le serveur ou le poste de travail.
- **API riche** – prend en charge les superpositions, métadonnées, recadrage et dithering dans un même flux de travail.
- **Optimisé pour la performance** – gérer efficacement les gros fichiers.

## Questions fréquemment posées

**Q : Puis‑je fusionner des images de dimensions différentes ?**  
R : Oui. Aspose.PSD redimensionne ou ajoute automatiquement des bordures aux calques en fonction de la taille du canevas que vous définissez.

**Q : Est‑il possible d’ajouter une superposition de couleur lors de la fusion ?**  
R : Absolument. Appliquez l’effet de superposition de couleur à chaque calque avant la fusion, ou à l’image composite finale.

**Q : Comment préserver les métadonnées EXIF ou XMP après la fusion ?**  
R : Créez ou copiez les métadonnées XMP à l’aide de l’API avant d’enregistrer le fichier fusionné.

**Q : Aspose.PSD prend‑il en charge la fusion d’images dans des formats autres que PSD ?**  
R : Oui. Vous pouvez charger des JPEG, PNG, BMP, etc., et les fusionner dans un PSD ou les exporter vers d’autres formats raster.

**Q : Quelles sont les considérations de performance pour de grands ensembles d’images ?**  
R : Utilisez les API de streaming et libérez rapidement les objets intermédiaires afin de maintenir une faible consommation de mémoire.
