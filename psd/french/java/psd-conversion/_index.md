## Foire aux questions

**Q : Comment convertir PSD en TIFF avec Aspose.PSD pour Java ?**  
R : Utilisez la classe `PsdImage` pour charger le fichier PSD et appelez la méthode `save` avec `TiffOptions`. Le tutoriel « Convertir PSD CMYK en TIFF CMYK » montre le code exact.

**Q : Puis‑je exporter des images de manière multithread ?**  
R : Oui. Le guide « Exporter des images dans un environnement multithread » montre comment exécuter les conversions en toute sécurité sur des threads séparés.

**Q : Comment convertir les calques d'image GIF en TIFF ?**  
R : Chargez le GIF avec `Image.load`, parcourez ses cadres et enregistrez chacun comme une page TIFF en utilisant `TiffOptions`. Consultez le tutoriel « Convertir les calques d'image GIF en TIFF ».

**Q : Aspose.PSD prend‑il en charge le traitement d'images Java pour la gestion des couleurs ?**  
R : Absolument. Vous pouvez utiliser les profils par défaut ou fournir des profils ICC personnalisés, comme expliqué dans les sections « Conversion de couleur avec les profils par défaut » et « Conversion de couleur avec des profils ICC ».

**Q : Où puis‑je trouver plus d'exemples sur l'exportation d'images ?**  
R : La page « Exporter des images dans un environnement multithread » contient des exemples détaillés sur la façon d'exporter les images efficacement.
