- **Is color management supported?** Absolut; Sie können während der Konvertierung Standard‑Profile oder benutzerdefinierte ICC‑Profile verwenden.

## Frequently Asked Questions

**Q: How do I convert PSD to TIFF using Aspose.PSD for Java?**  
A: Verwenden Sie die `PsdImage`‑Klasse, um die PSD‑Datei zu laden, und rufen Sie die `save`‑Methode mit `TiffOptions` auf. Das Tutorial „Convert CMYK PSD to CMYK TIFF“ zeigt den genauen Code.

**Q: Can I export images in a multi‑threaded way?**  
A: Ja. Der Leitfaden „Export Images in Multi‑Threaded Environment“ demonstriert, wie Sie Konvertierungen sicher in separaten Threads ausführen.

**Q: How to convert GIF image layers to TIFF?**  
A: Laden Sie das GIF mit `Image.load`, iterieren Sie durch die Frames und speichern Sie jedes als TIFF‑Seite mit `TiffOptions`. Siehe das Tutorial „Convert GIF Image Layers to TIFF“.

**Q: Does Aspose.PSD support java image processing for color management?**  
A: Absolut. Sie können Standard‑Profile verwenden oder benutzerdefinierte ICC‑Profile bereitstellen, wie in den Abschnitten „Color Conversion using Default Profiles“ und „Color Conversion using ICC Profiles“ erläutert.

**Q: Where can I find more examples on how to export images?**  
A: Die Seite „Export Images in Multi‑Threaded Environment“ enthält detaillierte Beispiele, wie Sie Bilder effizient exportieren können.
