## Was ist PSB und warum konvertieren?
PSB (Photoshop Big) ist Adobes Format für Dateien, die das 2 GB‑Limit einer regulären PSD überschreiten. Große Designs, hochauflösende Kompositionen oder mehrschichtige Kunstwerke nutzen häufig PSB. Das Konvertieren von PSB in leichtere Formate wie JPG oder PDF erleichtert das Teilen, Einbetten in Webseiten oder Archivieren, ohne wesentliche visuelle Informationen zu verlieren.

## Häufig gestellte Fragen

**Q: Kann ich eine passwortgeschützte PSB‑Datei konvertieren?**  
A: Ja. Aspose.PSD ermöglicht das Öffnen verschlüsselter PSB‑Dateien, indem das Passwort während des Ladevorgangs übergeben wird.

**Q: Verliert die Konvertierung von PSB zu JPG Ebeneninformationen?**  
A: Das Konvertieren zu JPG flacht das Bild ab, sodass Ebenendaten zusammengeführt werden. Verwenden Sie PDF oder PSD, wenn Sie Ebenen erhalten müssen.

**Q: Gibt es ein Größenlimit für PSB‑Dateien bei der Verwendung von Aspose.PSD?**  
A: Die Bibliothek ist darauf ausgelegt, sehr große PSB‑Dateien (mehrere Gigabyte) zu verarbeiten, solange Ihre JVM über ausreichend Heap‑Speicher verfügt.

**Q: Benötige ich für jedes Ausgabeformat eine separate Lizenz?**  
A: Nein. Eine einzelne Aspose.PSD for Java‑Lizenz deckt alle unterstützten Exportformate ab, einschließlich JPG, PDF und PSD.

**Q: Wie kann ich die Leistung verbessern, wenn ich viele PSB‑Dateien stapelweise konvertiere?**  
A: Wiederverwenden Sie nach Möglichkeit dieselbe `PsdImage`‑Instanz und erwägen Sie die Verarbeitung von Dateien in Parallel‑Streams mit geeigneten Speichereinstellungen.
