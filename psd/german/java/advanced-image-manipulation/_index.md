## Häufig gestellte Fragen

**Q: Kann ich ein PSD‑Bild drehen, ohne Ebeneninformationen zu verlieren?**  
A: Ja. Aspose.PSD dreht das Bild auf Ebenenebene und bewahrt alle Einstellungsebenen, Masken und Effekte.

**Q: Wie drehe ich ein Bild um einen Winkel, der kein Vielfaches von 90° ist?**  
A: Verwenden Sie die Methode `rotate(double angle)`, die jeden Gleitkomma‑Winkel (z. B. 45.0) für eine sanfte Drehung akzeptiert.

**Q: Gibt es einen Performance‑Einfluss beim Drehen großer PSD‑Dateien?**  
A: Minimal. Das Aktivieren des Font‑Cache (siehe das Tutorial „Font‑Cache erzwingen“) und die Verwendung der neuesten Bibliotheksversion reduzieren den Speicheraufwand.

**Q: Benötige ich eine separate Bibliothek für den Schriftartenaustausch?**  
A: Nein. Aspose.PSD enthält integrierte APIs zum Ersetzen von Schriftarten, sodass Sie **how to replace fonts java** ohne zusätzliche Abhängigkeiten implementieren können.

**Q: Welche Java‑Versionen werden offiziell unterstützt?**  
A: Aspose.PSD für Java unterstützt Java 8, 11, 17 und neuere LTS‑Versionen.
