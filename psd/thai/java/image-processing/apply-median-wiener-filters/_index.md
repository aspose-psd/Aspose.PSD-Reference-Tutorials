# ตัวกรองแบบขั้นตอนต่อขั้นตอน: ใช้ตัวกรอง Median & Wiener (Java)

## บทนำ

หากคุณกำลังมองหา **workflow ตัวกรองแบบขั้นตอนต่อขั้นตอน** เพื่อทำความสะอาดภาพที่มีสัญญาณรบกวนใน Java คุณมาถูกที่แล้ว Aspose.PSD for Java ทำให้การใช้ตัวกรอง Median และ Wiener เป็นเรื่องง่าย และยังสามารถ **แปลง PSD เป็น GIF** หลังการประมวลผลได้ ในบทแนะนำนี้เราจะพาคุณผ่านทุกขั้นตอนตั้งแต่การตั้งค่าไลบรารีจนถึงการบันทึกผลลัพธ์ที่ผ่านการกรอง เพื่อให้คุณสามารถรวมการลดสัญญาณรบกวนภาพคุณภาพสูงเข้าไปในแอปพลิเคชันของคุณได้อย่างมั่นใจ

## คำตอบสั้น
- **ตัวกรอง Median ทำอะไร?** ลดสัญญาณรบกวนแบบเกลือและพริกไทยพร้อมคงขอบภาพไว้  
- **เมื่อใดควรใช้ตัวกรอง Wiener?** สำหรับการลดสัญญาณรบกวนแบบปรับตัวที่พิจารณาความแปรปรวนของภาพในท้องถิ่น  
- **ฉันต้องมีใบอนุญาตเพื่อรันโค้ดหรือไม่?** เวอร์ชันทดลองฟรีใช้ได้สำหรับการพัฒนา; ต้องมีใบอนุญาตเชิงพาณิชย์สำหรับการใช้งานจริง  
- **ฉันสามารถบันทึกผลลัพธ์เป็น GIF ได้หรือไม่?** ได้ — Aspose.PSD ให้คุณ **แปลง PSD เป็น GIF** ในขั้นตอนเดียว  
- **การทำงานใช้เวลานานเท่าไหร่?** ปกติใช้เวลาน้อยกว่า 10 นาทีสำหรับการตั้งค่าเบื้องต้น

## ตัวกรองแบบขั้นตอนต่อขั้นตอนคืออะไร?

วิธี **ตัวกรองแบบขั้นตอนต่อขั้นตอน** จะแบ่งการประมวลผลภาพออกเป็นขั้นตอนที่ชัดเจนและจัดการได้ง่าย — โหลดภาพ, ตั้งค่าตัวกรอง, ใช้ตัวกรอง, และสุดท้ายบันทึกผลลัพธ์ วิธีการเช่นนี้ช่วยให้คุณดีบักแต่ละส่วน, ใช้โค้ดซ้ำได้, และปรับกระบวนการให้เข้ากับรูปแบบไฟล์ต่าง ๆ

- **แปลงด้วยคลิกเดียว** – ส่งออกตรงเป็น GIF, PNG หรือ JPEG หลังการประมวลผล

## นำเข้าชุดแพ็กเกจ

เริ่มต้นด้วยการนำเข้าคลาสที่จำเป็นเพื่อให้คุณเข้าถึงการจัดการภาพและตัวเลือกตัวกรองได้

```java
import com.aspose.psd.Image;
import com.aspose.psd.RasterImage;

import com.aspose.psd.imagefilters.filteroptions.MedianFilterOptions;
import com.aspose.psd.imageoptions.GifOptions;
```

## ตัวกรองแบบขั้นตอนต่อขั้นตอน: วิธีใช้ตัวกรอง Median

### ขั้นตอนที่ 1: โหลดภาพ

แรกสุดให้ชี้ API ไปที่ไฟล์ PSD ที่ต้องการทำความสะอาด

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";

// Load the source image
Image image = Image.load(sourceFile);
```

### ขั้นตอนที่ 2: แปลง Image เป็น RasterImage

ตัวกรองทำงานบนข้อมูลแรสเตอร์ ดังนั้นเราจึงแปลง `Image` ทั่วไปเป็น `RasterImage`

```java
// Cast the image into RasterImage
RasterImage rasterImage = (RasterImage)image;
if (rasterImage == null) {
    return;
}
```

### ขั้นตอนที่ 3: สร้างอินสแตนซ์ MedianFilterOptions

กำหนดขนาดของเคอร์เนล median ขนาด **4** ทำงานได้ดีสำหรับสัญญาณรบกวนระดับปานกลาง

```java
// Create an instance of MedianFilterOptions class and set the filter size
MedianFilterOptions options = new MedianFilterOptions(4);
```

### ขั้นตอนที่ 4: ใช้ตัวกรอง Median

จากนั้นให้ใช้ตัวกรองกับขอบเขตทั้งหมดของภาพ

```java
// Apply MedianFilterOptions filter to RasterImage object
rasterImage.filter(image.getBounds(), options);
```

### ขั้นตอนที่ 5: บันทึกภาพที่ได้ (แปลง PSD เป็น GIF)

หลังการกรอง เราจะบันทึกผลลัพธ์เป็น GIF — แสดงความสามารถ **แปลง PSD เป็น GIF** ของ Aspose.PSD

```java
String destName = dataDir + "median_test_denoise_out.gif";
// Save the resultant image as a GIF
image.save(destName, new GifOptions());
```

โดยทำตามขั้นตอนเหล่านี้คุณได้ใช้ตัวกรอง Median สำเร็จและส่งออกภาพที่ทำความสะอาดเป็น GIF แล้ว

## การใช้ตัวกรอง Wiener (ส่วนขยายเพิ่มเติม)

หากโครงการของคุณต้องการการลดสัญญาณรบกวนแบบปรับตัว คุณสามารถแทนที่ตัวกรอง Median ด้วยตัวกรอง Wiener โครงสร้างโค้ดเหมือนเดิม; เพียงแค่ต้องนำเข้า `WienerFilterOptions` แล้วสร้างอินสแตนซ์พร้อมรัศมีที่ต้องการ

> **เคล็ดลับ:** ทดลองเปลี่ยนขนาดเคอร์เนลของทั้งสองตัวกรองเพื่อหาจุดที่สมดุลระหว่างการกำจัดสัญญาณรบกวนและการคงรายละเอียดภาพ

## ปัญหาทั่วไปและการแก้ไข

| อาการ | สาเหตุที่เป็นไปได้ | วิธีแก้ |
|---------|---------------|-----|
| `ClassCastException` เมื่อแปลงเป็น `RasterImage` | ไฟล์ PSD ไม่รองรับการแปลงเป็นแรสเตอร์ | ตรวจสอบว่า PSD มีเลเยอร์แรสเตอร์หรือแปลงเลเยอร์เป็นแรสเตอร์ก่อน |
| GIF ที่บันทึกออกมาว่าง | เส้นทางปลายทางผิดหรือโฟลเดอร์ไม่มีสิทธิ์เขียน | ตรวจสอบให้ `dataDir` ชี้ไปยังไดเรกทอรีที่มีอยู่และสามารถเขียนได้ |
| ตัวกรองดูเหมือนไม่มีผล | ขนาดเคอร์เนลเล็กเกินกว่าระดับสัญญาณรบกวน | เพิ่มขนาดเคอร์เนล (เช่น `new MedianFilterOptions(6)`) |
