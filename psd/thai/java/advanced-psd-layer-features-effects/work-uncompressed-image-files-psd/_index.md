# แปลง PSD เป็น RAW ด้วย Java (ไฟล์ภาพที่ไม่ได้บีบอัด)

## บทนำ
เมื่อทำงานกับเอกสาร Photoshop (PSD) ใน Java การเข้าใจวิธี **convert PSD to RAW** และการส่งออก PSD โดยไม่บีบอัดเป็นสิ่งสำคัญเพื่อรักษาความคมชัดของภาพ ในบทแนะนำนี้เราจะสำรวจว่า Aspose.PSD ทำให้กระบวนการจัดการไฟล์ภาพที่ไม่ได้บีบอัดง่ายขึ้นอย่างไร ตั้งแต่การโหลด PSD จนถึงการบันทึกเป็นไฟล์ RAW‑style ที่ไม่ได้บีบอัด ไม่ว่าคุณจะสร้างแอปพลิเคชันที่ต้องการกราฟิกหนักหรือจำเป็นต้องส่งออกภาพแบบ loss‑less คุณจะพบทุกอย่างที่ต้องการที่นี่ พร้อมหรือยัง? มาเริ่มกันเลย!

## คำตอบสั้น
- **การแปลง PSD เป็น RAW หมายถึงอะไร?** มันบันทึกข้อมูล PSD โดยไม่มีการบีบอัดใด ๆ ทำให้ข้อมูลพิกเซลคงอยู่ในรูปแบบดั้งเดิม  
- **ไลบรารีใดจัดการเรื่องนี้?** Aspose.PSD for Java มี API ที่ตรงไปตรงมาสำหรับการบันทึกแบบไม่บีบอัด  
- **ต้องมีลิขสิทธิ์หรือไม่?** สามารถใช้รุ่นทดลองฟรีสำหรับการทดสอบ; ต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง  
- **ต้องใช้เวอร์ชัน Java ใด?** JDK 8 หรือสูงกว่า  
- **สามารถแก้ไขไฟล์หลังบันทึกได้หรือไม่?** ได้ – สามารถโหลด PSD ที่ไม่ได้บีบอัดใหม่และดำเนินการวาดหรือจัดชั้นต่อได้  

## การแปลง PSD เป็น RAW คืออะไร?
การแปลง PSD เป็น RAW หมายถึงการส่งออกเอกสาร Photoshop **โดยไม่มีการบีบอัดใด ๆ** ไฟล์ที่ได้จะคงข้อมูลพิกเซลทั้งหมดไว้ ซึ่งเหมาะอย่างยิ่งสำหรับสถานการณ์ที่คุณไม่สามารถยอมให้คุณภาพของภาพลดลงได้ เช่น การเก็บรักษาในคลังข้อมูล, การถ่ายภาพทางวิทยาศาสตร์, หรือกระบวนการพิมพ์ระดับสูง

## ทำไมต้องส่งออก PSD โดยไม่บีบอัด?
- **คุณภาพสูงสุด:** ไม่สูญเสียรายละเอียดจากศิลปะการบีบอัด  
- **ขนาดไฟล์คาดเดาได้:** ไฟล์ RAW มีขนาดที่สะท้อนมิติและบิตเดพธ์ของภาพโดยตรง  
- **การประมวลผลต่อเนื่องที่ง่ายขึ้น:** เครื่องมืออื่น ๆ สามารถอ่านข้อมูลพิกเซลโดยไม่ต้องทำการคลายบีบอัดก่อน  

## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะลงมือเขียนโค้ด มีข้อกำหนดบางอย่างที่ต้องตรวจสอบให้ครบถ้วน ไม่ต้องกังวลเลย มันง่ายมาก!

### ชุดพัฒนา Java (JDK)
- ตรวจสอบให้แน่ใจว่าคุณมี JDK 8 หรือสูงกว่าในระบบของคุณ หากยังไม่มี ให้ไปที่ [Oracle website](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) เพื่อดาวน์โหลดเวอร์ชันล่าสุด

### สภาพแวดล้อมการพัฒนาแบบบูรณาการ (IDE)
- IDE ที่ดี เช่น IntelliJ IDEA, Eclipse หรือ NetBeans จะทำให้การทำงานของคุณง่ายขึ้น ตั้งค่า IDE หากคุณยังไม่มี

### ไลบรารี Aspose.PSD สำหรับ Java
- ดาวน์โหลดไลบรารี Aspose.PSD for Java คุณสามารถรับเวอร์ชันล่าสุดได้จาก [here](https://releases.aspose.com/psd/java/)  

### ความรู้พื้นฐานของ Java
- คุณควรมีความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรม Java และแนวคิดเชิงวัตถุเพื่อให้สามารถทำตามได้อย่างราบรื่น

### ไฟล์ PSD
- เตรียมไฟล์ PSD ตัวอย่างสำหรับการทดสอบ คุณสามารถสร้างไฟล์ใน Photoshop หรือดาวน์โหลดตัวอย่างฟรีออนไลน์ได้

ตอนนี้ทุกอย่างพร้อมแล้ว มาเริ่มเขียนโค้ดกันเถอะ!

## นำเข้าแพ็กเกจ
เพื่อเริ่มต้น เราต้องนำเข้าแพ็กเกจที่จำเป็นสำหรับโค้ดของเรา ด้านล่างเป็นรายการ import ที่คุณต้องใช้:

```java
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.CompressionMethod;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PsdOptions;
```

การ import เหล่านี้จะนำคลาสและเมธอดที่จำเป็นเข้าสู่โปรเจกต์ของเรา ทำให้เราสามารถจัดการไฟล์ PSD ได้อย่างราบรื่น เราจะแบ่งกระบวนการออกเป็นขั้นตอนที่จัดการได้ง่าย

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีไฟล์ของคุณ
ก่อนอื่น คุณต้องระบุตำแหน่งที่ไฟล์ PSD ของคุณอยู่และตำแหน่งที่ต้องการบันทึกผลลัพธ์ ในโค้ดตัวอย่างของเรา เราจะสร้างตัวแปรเพื่อเก็บเส้นทางไดเรกทอรี

```java
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยเส้นทางจริงที่ไฟล์ PSD (`layers.psd`) ของคุณถูกเก็บไว้ การทำเช่นนี้จะทำให้โปรแกรมของคุณรู้ว่าจะไปหาไฟล์ที่ไหน

## ขั้นตอนที่ 2: โหลดไฟล์ PSD
ต่อไปให้โหลดไฟล์ PSD ด้วยเมธอด `Image.load()` แล้วแคสท์เป็นประเภท `PsdImage`

```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "layers.psd");
```

บรรทัดนี้เรียกเมธอด `load` จากคลาส `Image` เพื่อโหลดไฟล์ PSD ของคุณเข้าสู่หน่วยความจำ โดยการแคสท์เป็น `PsdImage` เรากำลังบอก Java ให้จัดการภาพนี้เป็นไฟล์ PSD ที่มีฟังก์ชันเฉพาะสำหรับการทำงานกับ PSD

## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก
ต่อไปเราต้องตั้งค่าตัวเลือกสำหรับการบันทึกไฟล์ของเรา โดยเฉพาะการระบุว่าต้องการผลลัพธ์เป็น **uncompressed** (คือแปลง PSD เป็น RAW)

```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw);
```

คลาส `PsdOptions` ให้เรากำหนดตัวเลือกต่าง ๆ สำหรับการบันทึกไฟล์ PSD การตั้งค่า `setCompressionMethod` เป็น `CompressionMethod.Raw` จะทำให้ไฟล์ที่บันทึกเป็นไฟล์ที่ไม่ได้บีบอัดและคงคุณภาพสูง

## ขั้นตอนที่ 4: บันทึกไฟล์ PSD ที่ไม่ได้บีบอัด
ตอนนี้ถึงเวลาบันทึกภาพ PSD ที่กำหนดค่าใหม่แล้ว

```java
psdImage.save(dataDir + "uncompressed_out.psd", saveOptions);
```

บรรทัดนี้เรียกฟังก์ชันบันทึกบนอินสแตนซ์ `PsdImage` (`psdImage`) ของเรา มันจะบันทึกไฟล์เป็น `uncompressed_out.psd` ในไดเรกทอรีที่ระบุและใช้ตัวเลือกที่กำหนดไว้ก่อนหน้า

## ขั้นตอนที่ 5: เปิดไฟล์ภาพที่สร้างใหม่อีกครั้ง
หลังจากบันทึกแล้ว ให้โหลดภาพผลลัพธ์ใหม่เพื่อยืนยันว่าทุกอย่างทำงานตามที่คาดหวัง

```java
PsdImage img = (PsdImage) Image.load(dataDir + "uncompressed_out.psd");
```

โดยการเรียก `load` อีกครั้ง เราสามารถสร้างอินสแตนซ์ `PsdImage` ใหม่ที่อ้างอิงไฟล์ที่บันทึกไว้ ขั้นตอนนี้สำคัญหากคุณต้องการจัดการหรือแสดงภาพหลังจากบันทึก

## ขั้นตอนที่ 6: วาดหรือจัดการภาพ
สุดท้าย คุณอาจต้องการวาดหรือจัดการภาพที่เปิดใหม่

```java
Graphics graphics = new Graphics(img);
```

ที่นี่เราสร้างอ็อบเจกต์ `Graphics` ซึ่งช่วยให้เราทำการดำเนินการกราฟิกต่าง ๆ บน `img` ได้ คุณสามารถวาดรูปทรง, เพิ่มข้อความ, หรือแม้แต่แก้ไขเลเยอร์ตามต้องการ!

## กรณีการใช้งานทั่วไป
- **การเก็บรักษาในคลังข้อมูล:** รักษาผลงานต้นฉบับโดยไม่มีการสูญเสียใด ๆ  
- **การถ่ายภาพทางวิทยาศาสตร์:** ส่งออกข้อมูลพิกเซลดิบสำหรับการวิเคราะห์  
- **การผลิตพิมพ์:** รับประกันความคมชัดสูงสุดก่อนส่งไฟล์ไปยังเครื่องพิมพ์
