# หมุนรูปภาพ 270 องศาด้วย Aspose.PSD for Java

## บทนำ

ใน **java image processing tutorial** นี้ คุณจะได้เรียนรู้วิธี **หมุนรูปภาพ 270 องศา** อย่างรวดเร็วและเชื่อถือได้ด้วย Aspose.PSD for Java ไม่ว่าคุณจะกำลังสร้างเครื่องมือแก้ไขรูปภาพ, ทำการแปลงแบบชุดอัตโนมัติ, หรือเพียงต้องการปรับทิศทางของเลเยอร์ PSD ไลบรารีนี้ทำให้การทำงานเป็นเรื่องง่าย เราจะพูดถึงการพลิกภาพและการแปลง PSD ที่หมุนแล้วเป็น JPEG ด้วย เพื่อให้คุณได้กระบวนการทำงานแบบครบวงจรจากต้นจนจบ

## คำตอบสั้น
- **ไลบรารีที่จัดการการหมุนคืออะไร?** Aspose.PSD for Java  
- **มุมการหมุนที่ตัวอย่างใช้คืออะไร?** 270 องศา  
- **ฉันสามารถพลิกภาพได้ด้วยหรือไม่?** ได้ – ใช้ตัวเลือก `RotateFlipType` เช่น `Rotate90FlipX`  
- **ฉันจะบันทึกผลลัพธ์อย่างไร?** ในตัวอย่างเราบันทึกเป็น JPEG ด้วย `JpegOptions`  
- **ต้องมีลิขสิทธิ์สำหรับการใช้งานในผลิตภัณฑ์หรือไม่?** จำเป็นต้องมีลิขสิทธิ์ Aspose.PSD ที่ถูกต้องสำหรับการใช้งานเชิงพาณิชย์  

## “rotate image 270 degrees” คืออะไร?
การหมุนรูปภาพ 270 องศาหมายถึงการหมุนภาพเป็นสามในสี่ของวงกลมเต็มตามเข็มนาฬิกา (หรือ 90 องศาตรงข้ามเข็มนาฬิกา) ในหลายสถานการณ์การแก้ไขกราฟิก การจัดแนวนี้จะตรงกับรูปแบบแนวตั้งเดิมหลังจากการแปลงหลายขั้นตอน

## ทำไมต้องใช้ Aspose.PSD สำหรับงานนี้?
- **รองรับ PSD อย่างเต็มรูปแบบ** – ทำงานกับเลเยอร์, มาสก์, และอ็อบเจกต์การปรับค่า  
- **ไม่ต้องใช้ Photoshop ติดตั้ง** – ทำงานบน Java runtime ใดก็ได้  
- **API ง่าย** – การเรียกเมธอดเดียว (`rotateFlip`) จัดการการหมุนและการพลิกได้ทั้งหมด  
- **แปลงฟอร์แมตง่าย** – ส่งออกโดยตรงเป็น JPEG, PNG หรือฟอร์แมตทั่วไปอื่น ๆ  

## ข้อกำหนดเบื้องต้น

ก่อนเริ่มทำงาน ให้ตรวจสอบว่าคุณมี:

- ไลบรารี **Aspose.PSD for Java** ติดตั้งแล้ว คุณสามารถดาวน์โหลดและดูเอกสารอ้างอิง API เต็มรูปแบบได้ [ที่นี่](https://reference.aspose.com/psd/java/)  
- สภาพแวดล้อมการพัฒนา Java (JDK 8 หรือสูงกว่า)  
- ไฟล์ PSD ตัวอย่างที่ต้องการหมุน ให้แก้ไขตัวแปร `sourceFile` ในโค้ดให้ชี้ไปยังพาธที่ถูกต้องของไฟล์ของคุณ  

## นำเข้าแพ็กเกจ

เริ่มต้นด้วยการนำเข้าคลาสที่จำเป็นจากแพ็กเกจ Aspose.PSD:

```java
import com.aspose.psd.Image;
import com.aspose.psd.RotateFlipType;

import com.aspose.psd.imageoptions.JpegOptions;
```

## วิธีหมุน PSD – ขั้นตอน 1: โหลดภาพ

สร้างอินสแตนซ์ `Image` ที่ชี้ไปยังไฟล์ PSD ต้นฉบับของคุณ:

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";
Image image = Image.load(sourceFile);
```

## วิธีหมุน PSD – ขั้นตอน 2: หมุนภาพ 270 องศา

ใช้เมธอด `rotateFlip` พร้อมค่า `RotateFlipType.Rotate270FlipNone` เพื่อหมุน 270 องศาโดยไม่พลิกใด ๆ:

```java
image.rotateFlip(RotateFlipType.Rotate270FlipNone);
```

> **เคล็ดลับ:** หากต้องการพลิกภาพแนวนอนหรือแนวตั้ง ให้เลือก `RotateFlipType` อื่น ๆ เช่น `Rotate90FlipX` หรือ `Rotate180FlipY`

## วิธีหมุน PSD – ขั้นตอน 3: แปลง PSD เป็น JPEG และบันทึก

หลังจากหมุนแล้ว คุณสามารถ **แปลง PSD เป็น JPEG** (หรือฟอร์แมตอื่นที่รองรับ) ด้วยคลาสอ็อปชันที่เหมาะสม:

```java
String destName = dataDir + "RotatedImage_out.jpg";
image.save(destName, new JpegOptions());
```

ไฟล์ `RotatedImage_out.jpg` ตอนนี้มีเนื้อหา PSD ดั้งเดิมที่ถูกหมุน 270 องศาและบันทึกเป็น JPEG แล้ว

## ปัญหาที่พบบ่อยและวิธีแก้

| ปัญหา | วิธีแก้ |
|-------|----------|

## คำถามที่พบบ่อย

**Q: Aspose.PSD รองรับฟอร์แมตภาพต่าง ๆ หรือไม่?**  
A: รองรับ PSD, JPEG, PNG, BMP, GIF และฟอร์แมตเรสเตอร์อื่น ๆ อีกหลายประเภท

**Q: ฉันสามารถกำหนดการหมุนแบบกำหนดเองได้หรือไม่, ไม่ใช่แค่การพลิกที่กำหนดไว้?**  
A: ได้! แม้ `RotateFlipType` จะให้มุมที่พบบ่อย คุณสามารถเรียกหลายครั้งหรือใช้เมทริกซ์การแปลงเพื่อหมุนมุมใดก็ได้ตามต้องการ

**Q: ฉันจะเปลี่ยน PSD ที่หมุนแล้วเป็นฟอร์แมตอื่น เช่น PNG อย่างไร?**  
A: แทนที่ `JpegOptions` ด้วย `PngOptions` (หรือคลาสออปชันที่เหมาะสม) ในเมธอด `save`

**Q: จะหาแหล่งสนับสนุนหรือความช่วยเหลือเพิ่มเติมได้จากที่ไหน?**

**Q: มีรุ่นทดลองใช้ฟรีหรือไม่?**  
A: มี คุณสามารถสำรวจ Aspose.PSD ด้วย [free trial](https://releases.aspose.com/)

**Q: จะขอรับลิขสิทธิ์ชั่วคราวได้อย่างไร?**
