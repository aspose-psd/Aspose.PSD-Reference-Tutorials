# บันทึก psd เป็น png – แทนที่ฟอนต์ที่หายไปใน Java

## บทนำ

หากคุณต้องการ **save PSD as PNG** ขณะเปลี่ยนฟอนต์ที่หายไปหรือไม่ต้องการในไฟล์ Photoshop (PSD) การแทนที่ฟอนต์ของ Aspose PSD ทำให้กระบวนการง่ายดาย ในแอปพลิเคชัน Java คุณสามารถโหลด PSD, บอก Aspose ว่าจะใช้ฟอนต์สำรองใด, แล้วส่งออกภาพที่แก้ไขในรูปแบบใดก็ได้ตามต้องการ บทเรียนนี้จะพาคุณผ่านขั้นตอนการทำงานทั้งหมด—ตั้งแต่การตั้งค่าโครงการจนถึงการส่งออก PNG ที่อัปเดต—เพื่อให้คุณสามารถ **handle missing fonts PSD** ได้อย่างมั่นใจโดยไม่ต้องเปิด Photoshop

## คำตอบด่วน
- **ไลบรารีที่จัดการการแทนที่ฟอนต์คืออะไร?** Aspose.PSD for Java  
- **ใช้เวลานานเท่าไหร่ในการทำงานนี้?** ประมาณ 5‑10 นาทีสำหรับสถานการณ์พื้นฐาน  
- **ฟอนต์ใดที่ใช้เป็นฟอนต์สำรองเริ่มต้น?** คุณสามารถตั้งค่า TrueType ฟอนต์ใดก็ได้ เช่น “Arial”  
- **ฉันสามารถบันทึกเป็นรูปแบบอื่นนอกจาก PNG ได้หรือไม่?** ใช่ – รองรับ PSD, JPEG, BMP และอื่น ๆ  
- **ต้องการไลเซนส์สำหรับการใช้งานจริงหรือไม่?** จำเป็นต้องมีไลเซนส์ Aspose.PSD ที่ถูกต้องสำหรับการใช้งานที่ไม่ใช่รุ่นทดลอง  

## การแทนที่ฟอนต์ใน Aspose PSD คืออะไร?
การแทนที่ฟอนต์ของ Aspose PSD คือกระบวนการกำหนดฟอนต์สำรองที่ไลบรารีจะใช้เมื่อพบฟอนต์ที่หายไปหรือไม่รองรับในไฟล์ PSD ซึ่งทำให้เลเยอร์ข้อความแสดงผลอย่างถูกต้องโดยไม่ต้องแก้ไขด้วยมือใน Photoshop และช่วยให้คุณสามารถ **handle missing fonts PSD** ได้โดยอัตโนมัติ

## นำเข้าแพ็กเกจ
`PsdImage` class แสดงเอกสาร Photoshop ในหน่วยความจำ ให้เข้าถึงเลเยอร์, ข้อความ, และความสามารถในการเรนเดอร์ `PsdLoadOptions` ควบคุมวิธีการอ่านไฟล์ รวมถึงฟอนต์สำรอง ส่วน `SaveOptions` (หรือคลาสย่อยตามรูปแบบ) กำหนดวิธีการเขียนภาพ

```java
import com.aspose.psd.Image;

import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageloadoptions.PsdLoadOptions;
import com.aspose.psd.imageoptions.PngOptions;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ
ระบุโฟลเดอร์ที่มีไฟล์ PSD ต้นฉบับ แทนที่ตัวแสดงตำแหน่งด้วยพาธจริงบนเครื่องของคุณ  

`File` object เพียงแค่ชี้ไปที่ไฟล์ PSD ที่คุณต้องการประมวลผล; ไม่จำเป็นต้องตั้งค่าเพิ่มเติมที่นี่  

```java
String dataDir = "Your Document Directory";
```

## ขั้นตอนที่ 2: โหลดภาพพร้อมฟอนต์ทดแทน
สร้างอินสแตนซ์ของ `PsdLoadOptions`, ตั้งค่าฟอนต์ทดแทนเริ่มต้น (เช่น **Arial**) แล้วโหลด PSD Aspose จะใช้ฟอนต์สำรองโดยอัตโนมัติเมื่อพบฟอนต์ที่หายไป  

`PsdLoadOptions` ให้คุณกำหนดพฤติกรรมการโหลด รวมถึงฟอนต์สำรองที่จะแทนที่ฟอนต์ที่หายไประหว่างขั้นตอนการนำเข้า  

```java
PsdLoadOptions psdLoadOptions = new PsdLoadOptions(); 
psdLoadOptions.setDefaultReplacementFont("Arial");

PsdImage psdImage = (PsdImage)Image.load(dataDir + "Cloud_AzPlat_Banner3A_SB_EN_US_160x600_chinese_font.psd", psdLoadOptions);
```

## ขั้นตอนที่ 3: บันทึกภาพที่แทนที่เป็น PNG
หลังจากทำการแทนที่ฟอนต์แล้ว คุณสามารถส่งออกภาพในรูปแบบที่รองรับได้ ที่นี่เราบันทึกเป็น PNG แต่คุณก็สามารถเลือก JPEG, BMP หรือแม้กระทั่งบันทึกกลับเป็น PSD ได้  

เมธอด `save` ของ `PsdImage` รับอ็อบเจกต์ `SaveOptions`; การใช้ `PngOptions` ทำให้ผลลัพธ์เป็น PNG แบบ loss‑less ที่เหมาะกับเว็บหรือการประมวลผลต่อไป  

```java
PngOptions pngOptions = new PngOptions();
psdImage.save(dataDir + "replaced_font.png", pngOptions);
```

## ฉันจะบันทึก PSD เป็น PNG หลังจากแทนที่ฟอนต์ได้อย่างไร?
โหลด PSD พร้อมฟอนต์สำรอง แล้วเรียก `psdImage.save("output.png", new PngOptions())` การบันทึกแบบบรรทัดเดียวนี้จะสร้าง PNG ที่เรนเดอร์เต็มรูปแบบซึ่งสะท้อนฟอนต์ที่ถูกแทนที่ ทำให้คุณสามารถฝังภาพได้ทุกที่โดยไม่ต้องกังวลเรื่องฟอนต์ที่หายไป ตรวจสอบว่าคุณได้ตั้งค่าฟอนต์ทดแทนก่อนบันทึกแล้ว และสามารถปรับการบีบอัด PNG ผ่านอ็อบเจกต์ `PngOptions` เพื่อให้ได้ขนาดไฟล์ที่เหมาะสม
