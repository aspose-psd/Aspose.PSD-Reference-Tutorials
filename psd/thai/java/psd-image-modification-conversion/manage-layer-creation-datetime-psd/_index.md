# สร้างเลเยอร์ PSD ใหม่และจัดการวันที่และเวลาการสร้างใน Java

## บทนำ
เมื่อคุณทำงานกับไฟล์ Photoshop (PSD) อย่างโปรแกรมเมติก การสามารถ **create new PSD layer** objects และติดตาม timestamp การสร้างของพวกมันเป็นการเปลี่ยนเกมอย่างแท้จริง ไม่ว่าคุณจะกำลังสร้างระบบควบคุมเวอร์ชันสำหรับสินทรัพย์การออกแบบ, ทำการแก้ไขเป็นชุดอัตโนมัติ, หรือแค่ต้องการเส้นทางการตรวจสอบสำหรับโครงการร่วม, การรู้วิธีอ่านและตั้งค่าวันที่สร้างของเลเยอร์จะช่วยให้คุณรักษาที่มาที่ไปของการเปลี่ยนแปลงทุกอย่างได้อย่างครบถ้วน ในบทเรียนนี้เราจะเดินผ่านกระบวนการทั้งหมดโดยใช้ Aspose.PSD for Java — ตั้งแต่การโหลด PSD, ดึงวันที่สร้างของเลเยอร์, ตรวจสอบ, จนถึงการเพิ่มเลเยอร์ปรับแต่งใหม่ทั้งหมด

## คำตอบด่วน
- **ไลบรารีที่จัดการไฟล์ PSD ใน Java คืออะไร?** Aspose.PSD for Java  
- **ฉันสามารถอ่านวันที่สร้างของเลเยอร์ได้หรือไม่?** ใช่, โดยใช้ `layer.getLayerCreationDateTime()`  
- **สามารถเพิ่มเลเยอร์ปรับแต่งใหม่ได้หรือไม่?** แน่นอน – `im.addLevelsAdjustmentLayer()` สร้างขึ้น  
- **ฉันต้องการใบอนุญาตสำหรับการใช้งานในผลิตภัณฑ์หรือไม่?** จำเป็นต้องมีใบอนุญาตเชิงพาณิชย์สำหรับการใช้งานที่ไม่ใช่รุ่นทดลอง  
- **เวอร์ชัน Java ที่รองรับคืออะไร?** JDK 8 หรือใหม่กว่า  

## “create new PSD layer” คืออะไร?
การสร้างเลเยอร์ PSD ใหม่หมายถึงการแทรกอ็อบเจ็กต์เลเยอร์ใหม่อย่างโปรแกรมเมติก — เช่น เลเยอร์ปรับแต่ง, ข้อความ, หรือพิกเซล — ลงในเอกสาร PSD ที่มีอยู่ การดำเนินการนี้ทำให้คุณสามารถขยายหรือแก้ไขภาพได้โดยไม่ต้องเปิด Photoshop ด้วยตนเอง

## ทำไมต้องจัดการวันที่และเวลาการสร้างเลเยอร์?
การติดตามวันที่และเวลาการสร้างของแต่ละเลเยอร์ช่วยให้คุณ:
- **ตรวจสอบการแก้ไข** – รู้ได้อย่างแม่นยำว่าเลเยอร์ถูกเพิ่มเมื่อใด.  
- **ซิงโครไนซ์สินทรัพย์** ระหว่างทีมโดยเปรียบเทียบ timestamp.  
- **อัตโนมัติกระบวนการทำงาน** ที่พึ่งพากฎตามเวลา (เช่น ซ่อนเลเยอร์ที่เก่ากว่าหนึ่งเดือน).

## นำเข้าแพ็กเกจ
แรกสุด, นำเข้าคลาสของ Aspose.PSD และยูทิลิตี้ของ Java ที่คุณต้องการ การนำเข้าต่าง ๆ นี้ทำให้คุณเข้าถึงการจัดการภาพ, การจัดการเลเยอร์, และการจัดรูปแบบวันที่.

```java
import com.aspose.psd.Image;
import com.aspose.psd.examples.Utils.Assert;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
import java.text.DateFormat;
import java.text.SimpleDateFormat;
import java.util.Date;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ
ระบุโฟลเดอร์ที่มีไฟล์ PSD ที่คุณต้องการทำงาน แทนที่ตัวแปรตำแหน่งที่เก็บด้วยพาธเต็มบนเครื่องของคุณ.

```java
String dataDir = "Your Document Directory";
```

## ขั้นตอนที่ 2: โหลดไฟล์ PSD
สร้างอินสแตนซ์ของ `PsdImage` โดยโหลดไฟล์เป้าหมาย วัตถุนี้เป็นจุดเริ่มต้นสำหรับการดำเนินการกับเลเยอร์ทั้งหมด.

```java
String sourceName = dataDir + "OneLayer.psd";
PsdImage im = (PsdImage) Image.load(sourceName);
```

## ขั้นตอนที่ 3: เข้าถึงเลเยอร์และวันที่สร้างของมัน
ดึงเลเยอร์แรก (ดัชนี 0) และรับ timestamp การสร้างของมัน นี่คือวันที่คุณจะใช้เปรียบเทียบหรือบันทึกต่อไป.

```java
Layer layer = im.getLayers()[0];
Date creationDateTime = layer.getLayerCreationDateTime();
```

## ขั้นตอนที่ 4: จัดรูปแบบวันที่สร้าง
แปลงอ็อบเจ็กต์ `Date` ดิบให้เป็นสตริงที่มนุษย์อ่านได้ ปรับรูปแบบตามที่คุณต้องการหากต้องการรูปแบบอื่น.

```java
DateFormat dateFormat = new SimpleDateFormat("yyyy/MM/dd HH:mm:ss");
```

## ขั้นตอนที่ 5: ตรวจสอบความถูกต้องของวันที่สร้าง
เพื่อการสาธิต เราเปรียบเทียบวันที่ที่ดึงมาด้วยค่าที่คาดหวัง ในโครงการจริงคุณอาจเปรียบเทียบกับบันทึกในฐานข้อมูลหรือไฟล์กำหนดค่า.

```java
Date expectedDateTime = new Date("2018/7/17 8:57:24");
Assert.areEqual(expectedDateTime, creationDateTime);
```

## ขั้นตอนที่ 6: สร้างเลเยอร์ใหม่
ตอนนี้เราจริง ๆ **create new PSD layer** objects. ที่นี่เราจะเพิ่มเลเยอร์ปรับระดับ (Levels) ซึ่งทำให้คุณปรับช่วงโทนสีได้โดยไม่เปลี่ยนพิกเซลต้นฉบับ.

```java
Date now = new Date();
Layer createdLayer = im.addLevelsAdjustmentLayer();
```

> **เคล็ดลับ:** ตัวแปร `now` บันทึกช่วงเวลาที่คุณเพิ่มเลเยอร์ ซึ่งคุณสามารถเก็บเป็นเมทาดาต้าต่อไปได้หากต้องการ timestamp แบบกำหนดเอง.

## ปัญหาทั่วไปและวิธีแก้
| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|---------|
| `NullPointerException` on `layer.getLayerCreationDateTime()` | ไฟล์ PSD ไม่มีเลเยอร์หรือดัชนีของเลเยอร์อยู่นอกช่วง. | ตรวจสอบว่า `im.getLayers().length > 0` ก่อนเข้าถึง. |
| Date mismatch in validation | `Date` constructor แปลงสตริงตามการตั้งค่าภูมิภาค. | ใช้ `SimpleDateFormat.parse("2018/07/17 08:57:24")` เพื่อการแปลงที่เชื่อถือได้. |
| New layer not visible in Photoshop | เลเยอร์ปรับแต่งอาจถูกซ่อนโดยค่าเริ่มต้น. | เรียก `createdLayer.setVisible(true);` หลังจากสร้าง. |
