## วิธีนำเข้าภาพ PSD ไปยังเลเยอร์
ด้านล่างเป็นขั้นตอนที่ชัดเจนและเป็นลำดับเลขที่อธิบาย **วิธีเพิ่มภาพ** ไปยังเลเยอร์ PSD, **ตั้งค่าพิกัดเลเยอร์**, และ **เติมสีเลเยอร์ PSD**.

### ขั้นตอนที่ 1: นำเข้าแพ็กเกจที่จำเป็น
ก่อนอื่นเรานำเข้าคลาสของ Aspose.PSD ที่ต้องใช้ ซึ่งจะเป็นพื้นฐานสำหรับโค้ดส่วนต่อไป.

```java
import com.aspose.psd.Color;
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.Point;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
```

### ขั้นตอนที่ 2: กำหนดไดเรกทอรีของเอกสาร
ระบุตำแหน่งที่ไฟล์ PSD ต้นฉบับอยู่และที่ที่ผลลัพธ์จะถูกบันทึก.

```java
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยพาธจริงบนระบบไฟล์ของคุณที่เก็บไฟล์ PSD อยู่.

### ขั้นตอนที่ 3: โหลดไฟล์ PSD ของคุณ
เปิดไฟล์ PSD เพื่อให้เราสามารถทำงานกับเลเยอร์ต่าง ๆ ได้.

```java
PsdImage image = (PsdImage) Image.load(dataDir + "sample.psd");
```

ตรวจสอบให้แน่ใจว่า `"sample.psd"` ตรงกับชื่อไฟล์ที่คุณต้องการแก้ไข.

### ขั้นตอนที่ 4: ดึงเลเยอร์เป้าหมาย
เลือกเลเยอร์ที่จะรับภาพใหม่ ที่นี่เราใช้เลเยอร์ที่สอง (index 1).

```java
Layer layer = image.getLayers()[1];
```

หากต้องการเลเยอร์อื่น เพียงเปลี่ยนค่า index ตามที่ต้องการ.

### ขั้นตอนที่ 5: สร้างภาพใหม่เพื่อทำการนำเข้า
ต่อไปเราจะ **add image psd layer** ด้วยการสร้าง `PsdImage` ใหม่ที่เราจะวาดลงไป.

```java
PsdImage drawImage = new PsdImage(200, 200);
```

คุณสามารถปรับความกว้างและความสูงให้ตรงกับรูปภาพต้นฉบับของคุณได้.

### ขั้นตอนที่ 6: เติมพื้นผิวภาพ (ตั้งค่าสีเลเยอร์)
มาทำ **fill psd layer color** ด้วยพื้นหลังสีเหลืองสดใส ตัวอย่างนี้แสดงวิธีตั้งค่าสีทึบก่อนการวาด.

```java
Graphics g = new Graphics(drawImage);
g.clear(Color.getYellow());
```

หากต้องการสีอื่น สามารถแทนที่ `Color.getYellow()` ด้วย `Color` ที่คุณต้องการได้.

### ขั้นตอนที่ 7: วาดภาพบนเลเยอร์ (ตั้งค่าพิกัดเลเยอร์)
นี่คือหัวใจของ **how to add image** – เราวางภาพที่สร้างใหม่ลงบนเลเยอร์ที่เลือกโดยกำหนดพิกัดเฉพาะ.

```java
layer.drawImage(new Point(10, 10), drawImage);
```

คำสั่ง `Point(10, 10)` **sets layer coordinates** (X = 10, Y = 10). ปรับค่าตามต้องการเพื่อวางภาพในตำแหน่งที่ต้องการ.

### ขั้นตอนที่ 8: บันทึกไฟล์ PSD ที่อัปเดต
สุดท้ายให้เขียนการเปลี่ยนแปลงกลับไปยังดิสก์.

```java
image.save(dataDir + "ImportImageToPSDLayer_out.psd");
```

ตั้งชื่อไฟล์ผลลัพธ์ให้มีความหมาย; ตัวอย่างนี้เพิ่ม `_out` เพื่อไม่ให้ไฟล์ต้นฉบับถูกเขียนทับ.

## ปัญหาทั่วไปและวิธีแก้
- **ภาพแสดงเป็นสีขาว** – ตรวจสอบว่าคุณได้เรียก `Graphics.clear()` ก่อนการวาด; มิฉะนั้นแคนวาสอาจเป็นโปร่งใส.  
- **แก้ไขเลเยอร์ผิด** – จำไว้ว่าดัชนีเลเยอร์เริ่มจาก 0. ตรวจสอบดัชนีที่ใช้ใน `getLayers()` อีกครั้ง.  
- **โปรไฟล์สีไม่รองรับ** – Aspose.PSD รองรับโปรไฟล์ส่วนใหญ่ แต่หากสีเปลี่ยนแปลง ให้แปลงภาพต้นฉบับเป็น sRGB ก่อนนำเข้า.
