# บันทึก PSD เป็น PNG ด้วย Aspose.PSD สำหรับ Java

## นำเข้าแพ็กเกจ

การนำเข้าต่อไปนี้จะดึงคลาสหลักของ Aspose.PSD ที่จำเป็นสำหรับการโหลดไฟล์ PSD และกำหนดค่าตัวเลือกการส่งออก PNG.  
```java
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PngOptions;
```

เราจะทำการแยกกระบวนการบันทึกภาพออกเป็นหลายขั้นตอนเพื่อความเข้าใจที่ชัดเจนและครอบคลุม

## วิธีบันทึก PSD เป็น PNG ด้วย Aspose.PSD for Java?

คลาส `PsdImage` แทนเอกสาร Photoshop ในหน่วยความจำ, ส่วน `ImageSaveOptions` ร่วมกับ `SaveFormat` ระบุรูปแบบเอาต์พุตและการตั้งค่าการบีบอัดที่ต้องการ โดยการโหลด PSD และเรียกเมธอด save พร้อมตัวเลือก PNG คุณสามารถแปลงไฟล์ได้ในหนึ่งคำสั่งที่มีประสิทธิภาพ

โหลดไฟล์ PSD ด้วย `new PsdImage("source.psd")` และเรียก `psd.save("output.png", ImageSaveOptions.createSaveOptions(SaveFormat.Png))`. คำสั่งบรรทัดเดียวนี้จัดการการแบนเลเยอร์, การรักษาโปรไฟล์สี, และการบีบอัด PNG โดยอัตโนมัติ สำหรับการทำงานเป็นชุด, ให้วางคำสั่งนี้ภายในลูปที่วนผ่านไฟล์ต้นทางของคุณ

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสารของคุณ

ตั้งค่าเส้นทางสำหรับไดเรกทอรีเอกสารของคุณ, ที่ซึ่งไฟล์ PSD ของคุณตั้งอยู่:

```java
String dataDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: ระบุเส้นทางต้นทางและปลายทาง

กำหนดเส้นทางสำหรับไฟล์ PSD ต้นทางและไฟล์ปลายทางที่ภาพจะถูกบันทึก:

```java
String sourceFile = dataDir + "sample.psd";
String destName = dataDir + "result.png";
```

### ขั้นตอนที่ 3: โหลดภาพ PSD

โหลดภาพ PSD โดยใช้ Aspose.PSD:

```java
Image image = Image.load(sourceFile);
```

### ขั้นตอนที่ 4: บันทึกภาพด้วยตัวเลือก

`PsdImage` เป็นคลาสหลักของ Aspose.PSD ที่แทนเอกสาร Photoshop ในหน่วยความจำ. แคสต์ภาพที่โหลดเป็น `PsdImage` แล้วบันทึกเป็นไฟล์ PNG:

```java
PsdImage psdImage = (PsdImage)image;
psdImage.save(destName, new PngOptions());
```

ทำซ้ำขั้นตอนเหล่านี้สำหรับแต่ละภาพที่คุณต้องการบันทึก, เพื่อให้กระบวนการทำงานอย่างต่อเนื่องกับ Aspose.PSD

## ปัญหาทั่วไปและวิธีแก้

- **OutOfMemoryError on large files** – เปิดการสตรีมโดยใช้ `PsdImage.load(inputStream, true)` เพื่อหลีกเลี่ยงการโหลดไฟล์ทั้งหมดเข้าสู่ RAM.
- **Missing transparency** – ตรวจสอบว่าคุณใช้ `PngOptions` พร้อม `ColorType = PngColorType.Rgba` เพื่อรักษาช่องอัลฟ่า.
- **Incorrect colors** – ยืนยันว่าโปรไฟล์สีของ PSD ต้นทางถูกฝังอยู่; Aspose.PSD จะนำไปใช้โดยอัตโนมัติระหว่างการส่งออก.
