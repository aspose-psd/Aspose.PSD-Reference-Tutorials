## Aspose.PSD 進階影像操作技巧
以下提供快速且實用的指南，說明開發者最常詢問的 **進階影像操作** 任務。

### 如何將 PSD 轉換為 PNG
Aspose.PSD 讓您載入 PSD 檔案後直接儲存為 PNG，並保留透明度與圖層效果。這非常適合用於 Web 資產。

### 如何將 PSD 儲存為 JPEG
當需要壓縮的點陣圖時，您可以將 PSD 匯出為 JPEG。亦可調整品質設定，以在檔案大小與視覺品質之間取得平衡。

### 如何調整影像亮度
API 提供影像調整類別，讓您在不平面化檔案的情況下，針對單一圖層或整張影像調整亮度、對比度與伽瑪值。

### 如何新增圖層效果
您可以以程式方式加入投影、斜角、發光等 Photoshop 圖層效果，讓執行期間動態增強視覺效果。

### 如何將 PSD 轉換為 PDF
將 PSD 匯出為 PDF 可用於列印就緒的文件或保存檔案。轉換過程會在可能的情況下保留向量文字與圖層資訊。

## 常見問題

**Q: 我可以在商業應用程式中使用 Aspose.PSD 嗎？**  
A: 可以。正式上線需購買商業授權，但可使用免費試用版進行評估。

**Q: Aspose.PSD 支援受密碼保護的 PSD 檔案嗎？**  
A: 支援。您可在載入選項中提供密碼以開啟受保護的檔案。

**Q: 哪些 .NET 版本相容？**  
A: 完全支援 .NET Framework 4.5 以上、.NET Core 3.1 以上、.NET 5 與 .NET 6。

**Q: 如何有效處理大型 PSD 檔案？**  
A: 使用 `LoadOptions` 以啟用延遲載入，並逐層處理以降低記憶體使用量。

**Q: 能否批次將多個 PSD 檔案轉換為 PDF？**  
A: 完全可以。遍歷目錄，載入每個 PSD，然後以 PDF 格式呼叫 `Save`——API 會在一次處理中完成轉換。
