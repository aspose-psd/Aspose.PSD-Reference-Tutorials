## 常見問題

**Q: 我可以在商業 Java 應用程式中使用 Aspose.PSD 嗎？**  
A: 可以，只要持有有效的 Aspose 授權。亦提供免費試用供評估使用。

**Q: 此函式庫是否支援含內嵌字型的 AI 檔案？**  
A: 完全支援。Aspose.PSD 能讀取內嵌字型，並在可能的情況下保留文字渲染。

**Q: 若需要一次轉換大量 AI 檔案該怎麼做？**  
A: 使用迴圈載入每個檔案並呼叫 `save` 方法。函式庫已針對批次處理進行最佳化。

**Q: AI 檔案有尺寸限制嗎？**  
A: 函式庫可處理數百 MB 的檔案，唯一限制來自 JVM 的堆積記憶體大小。

**Q: 轉換為 PNG 時如何保留透明度？**  
A: 請使用 `PngOptions`，並將 `ColorType = PngColorType.TrueColorWithAlpha` 設定為透明模式。
