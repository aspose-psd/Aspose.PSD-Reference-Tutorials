## 常見問題

**Q: 我可以轉換受密碼保護的 PSB 檔案嗎？**  
**A:** 可以。Aspose.PSD 允許在載入時提供密碼，以開啟加密的 PSB 檔案。

**Q: 將 PSB 轉換為 JPG 會失去圖層資訊嗎？**  
**A:** 轉換為 JPG 會將圖像平面化，圖層資料會被合併。如需保留圖層，請使用 PDF 或 PSD。

**Q: 使用 Aspose.PSD 轉換 PSB 檔案有尺寸上限嗎？**  
**A:** 函式庫設計能處理多個 GB 的大型 PSB 檔案，只要您的 JVM 擁有足夠的堆積記憶體即可。

**Q: 每種輸出格式需要單獨授權嗎？**  
**A:** 不需要。單一 Aspose.PSD for Java 授權即可涵蓋所有支援的匯出格式，包括 JPG、PDF 與 PSD。

**Q: 如何在批次轉換大量 PSB 檔案時提升效能？**  
**A:** 盡可能重複使用同一個 `PsdImage` 實例，並考慮使用平行串流 (parallel streams) 處理，同時調整記憶體設定以符合需求。
