## 什麼是 psd image processing？
psd image processing 是指透過程式碼以程式化方式建立、修改與渲染 Photoshop (.psd) 檔案。使用 Aspose.PSD，您可以操作圖層、套用填色、加入漸層、圖案與描邊——全部透過純 Java API 完成。

## 為何使用 Aspose.PSD for Java 來新增漸層、圖層與圖案效果？
- **完整的 PSD 相容性** – 可使用所有 Photoshop 功能，無需安裝 Photoshop 本身。  
- **純 Java API** – 無原生程式碼，易於整合至伺服器端或桌面應用程式。  
- **豐富的效果函式庫** – 支援漸層、圖案、描邊、簽名等多種功能。  
- **效能最佳化** – 能有效處理大型檔案。
