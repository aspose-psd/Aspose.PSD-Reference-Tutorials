## 什麼是 Java 中的灰階圖像？  

灰階圖像每個像素佔用一個位元組，表示亮度，範圍從 0（黑）到 255（白）。將全彩 PSD 轉為灰階可聚焦於亮度資訊，適用於藝術效果、電腦視覺前置處理或製作輕量資產。

## 為何使用 Aspose.PSD 優化 Java 中的 PNG？  

PNG 為無損格式，但未優化的檔案可能過大。使用 Aspose.PSD 您可以：

1. 在可能的情況下降低位元深度。  
2. 套用適合圖像內容的壓縮濾鏡。  
3. 移除不必要的中繼資料。  

這些步驟可在保持視覺品質的同時縮小檔案大小，十分適合網頁與行動部署。

## 常見問題

**Q: 如何在 Java 中將 PSD 檔案轉換為 TIFF？**  
A: 使用 `PsdImage` 載入 PSD，然後呼叫 `save("output.tiff", new TiffOptions())`。API 會自動處理色彩配置檔與圖層。

**Q: 我可以在轉換為 TIFF 前使用 Java 繪製形狀嗎？**  
A: 可以。使用 Aspose.PSD 提供的 `Graphics` 物件繪製線條、矩形或自訂路徑，然後將結果儲存為 TIFF。

**Q: 在轉換前可以替換 PSD 中的字型嗎？**  
A: 當然可以。此函式庫允許您以系統字型替代缺失的字型，確保最終 TIFF 中的文字正確呈現。

**Q: 如何使用 Aspose.PSD 將 AI 檔案轉換為圖像？**  
A: 雖然 Aspose.PSD 主要支援 PSD，但您可先將 AI 轉為 PDF（使用 Aspose.Slides 或其他工具），再將 PDF 頁面載入為圖像進行後續處理。

**Q: 在 Java 中優化 PNG 檔案的步驟是什麼？**  
A: 降低位元深度、套用適當的壓縮濾鏡，並移除中繼資料——這些皆可透過 Aspose.PSD 的 PNG 選項完成。
