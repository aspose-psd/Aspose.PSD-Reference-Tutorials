- **為什麼選擇 Aspose.PSD for Java？** 它提供完整的 API，支援 PSD、PNG、JPEG 等多種格式，且不需原生函式庫。

## 常見問答

**Q: 我可以在大型 PSD 檔案上使用 median 濾鏡而不會耗盡記憶體嗎？**  
A: 可以。Aspose.PSD 會逐層理，並提供串流選項以降低記憶體使用量。

**Q: median 濾鏡與 Gaussian 濾鏡有何不同？**  
A: median 濾鏡是非線性的，擅長去除鹽與胡椒雜訊且保留邊緣；相較之下，Gaussian 濾鏡是線性的，會均勻平滑影像。

**Q: 是否可以在同一流程中結合 **apply gaussian filter** 與 median 濾鏡？**  
A: 當然可以。你可以串接濾鏡——先套用 Gaussian 模糊以降低高頻雜訊，接著使用 median 濾鏡進行保邊清理。

**Q: Aspose.PSD 是否支援在智慧物件上執行 **crop psd file** 操作？**  
A: 支援。裁剪可作用於任何點陣圖層，包括嵌入於智慧物件中的層。

**Q: 這些教學需要哪個 Java 版本？**  
A: 需要 Java 8 或更新版本；此函式庫完全相容於 Java 11、17 以及之後的 LTS 版。
