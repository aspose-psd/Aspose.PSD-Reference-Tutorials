## 什麼是 PNG 壓縮，為什麼重要？

PNG 壓縮在不失去 PNG 所具備的無損品質前提下，減少每個像素所儲存的資料量。較小的 PNG 檔案在網站上載入更快、佔用較少頻寬，提升使用者體驗，尤其在行動裝置上更為顯著。

## 常見問題

**Q: PNG 壓縮與 PNG 檔案大小縮減有何不同？**  
A: 壓縮是透過演算法減少資料大小的過程，而 PNG 檔案大小縮減則可能同時包括移除中繼資料、降低位元深度或改變解析度等操作。

**Q: 壓縮圖像後，我可以變更 PNG 背景顏色嗎？**  
A: 可以——Aspose.PSD 允許在任何階段修改背景顏色，即使在壓縮之後，也不需要重新編碼整個檔案。

**Q: 變更位元深度會影響透明度嗎？**  
A: 降低位元深度可能會影響漸層的平滑度，但只要保持 32 位元 PNG，alpha 通道（透明度）仍會被保留。

**Q: PNG 的壓縮上限是多少？**  
A: 由於 PNG 為無損格式，壓縮比例取決於圖像複雜度與現有中繼資料。結合壓縮與 PNG 檔案大小縮減技術可獲得最佳結果。

**Q: 在正式環境使用 Aspose.PSD 是否需要授權？**  
A: 需要——商業部署必須擁有有效的 Aspose.PSD 授權。亦提供免費試用版供評估使用。
