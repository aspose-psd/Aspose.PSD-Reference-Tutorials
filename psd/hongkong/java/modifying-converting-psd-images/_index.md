## 常見問題

**Q: 我可以在已包含調整圖層的 PSD 中新增對角水印嗎？**  
A: 可以。Aspose.PSD 允許您將水印堆疊於現有圖層之上，包括 hue saturation、curves 與 color fill 圖層，且不會將檔案扁平化。

**Q: 是否可以批次處理多個 PSD 檔案以新增相同的水印？**  
A: 當然可以。您可以遍歷目錄，載入每個 PSD，套用水印，並儲存結果——全部在單一的 Java 程式中完成。

**Q: 調整圖層教學（hue saturation、curves、color fill）會影響水印的可見性嗎？**  
A: 調整圖層會影響底層影像資料，但若您在這些調整之後再加入水印，則不會影響已光柵化的水印圖層。您也可以將水印放在獨立的圖層上，以取得完整控制。

**Q: 處理大型 PSD 檔案時，我應該注意哪些記憶體考量？**  
A: 請使用「Control Cache Reallocation」教學中描述的快取重新分配控制，以限制 RAM 使用，特別是在批次處理高解析度檔案時。

**Q: 哪些版本的 Aspose.PSD 相容於 Java 11 及更新版本？**  
A: 所有近期的發行版本（2023‑2025）皆支援 Java 8+，包括 Java 11、17 以及更高版本。請檢查產品發行說明以確認您使用的確切版本。
