## 常見問題

**Q: 我需要付費授權才能使用 Aspose.PSD for Java 嗎？**  
A: 提供免費試用供開發與測試使用，但正式上線需購買商業授權。

**Q: 我可以對同一圖層套用多個效果，例如內陰影與外發光嗎？**  
A: 可以。效果可透過依序加入圖層的效果集合來堆疊。

**Q: 是否可以在不遺失圖層資訊的情況下編輯 PSD 檔案？**  
A: 絕對可以。Aspose.PSD 會在儲存變更時保留所有圖層中繼資料、遮色片與調整圖層。

**Q: 最新的 Aspose.PSD 函式庫支援哪些 Java 版本？**  
A: 完全支援 Java 8、11 以及更新的 LTS 版本。

**Q: 如何處理大型 PSD 檔案以避免記憶體問題？**  
A: 使用 `PsdImage` 類別的 `Load` 重載搭配 `FileStream`，並在適當時機啟用延遲載入。
