# PSD को RAW में बदलें Java का उपयोग करके (असंपीडित छवि फ़ाइलें)

## परिचय
जब Java में Photoshop दस्तावेज़ों (PSD) के साथ काम करने की बात आती है, तो **convert PSD to RAW** कैसे किया जाए और PSD को बिना संपीड़न के निर्यात करना समझना छवि की सटीकता को बनाए रखने के लिए आवश्यक है। इस ट्यूटोरियल में हम देखेंगे कि Aspose.PSD असंपीडित छवि फ़ाइलों को संभालने की प्रक्रिया को कैसे सरल बनाता है, PSD को लोड करने से लेकर उसे RAW‑स्टाइल असंपीडित फ़ाइल के रूप में सहेजने तक। चाहे आप ग्राफ़िक्स‑गहन एप्लिकेशन बना रहे हों या लॉस‑लेस इमेज एक्सपोर्ट की आवश्यकता हो, आपको यहाँ सब कुछ मिलेगा। क्या आप शुरू करने के लिए तैयार हैं? चलिए शुरू करते हैं!

## त्वरित उत्तर
- **“convert PSD to RAW” का क्या मतलब है?** यह PSD डेटा को बिना किसी संपीड़न के सहेजता है, जिससे पिक्सेल डेटा अपनी मूल रूप में रहता है।  
- **यह कार्य कौन सी लाइब्रेरी संभालती है?** Aspose.PSD for Java असंपीडित सहेजने के लिए एक सीधा API प्रदान करती है।  
- **क्या मुझे लाइसेंस चाहिए?** परीक्षण के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **कौन सा Java संस्करण आवश्यक है?** JDK 8 या उससे ऊपर।  
- **क्या सहेजने के बाद मैं फ़ाइल को फिर से संपादित कर सकता हूँ?** हाँ – आप असंपीडित PSD को पुनः लोड करके ड्रॉइंग या लेयरिंग जारी रख सकते हैं।

## “convert PSD to RAW” क्या है?
PSD को RAW में बदलना मतलब Photoshop दस्तावेज़ को **बिना किसी संपीड़न** के निर्यात करना है। परिणामी फ़ाइल पूरी पिक्सेल डेटा को बरकरार रखती है, जो उन स्थितियों के लिए आदर्श है जहाँ इमेज क्वालिटी से समझौता नहीं किया जा सकता—जैसे अभिलेखीय संग्रहण, वैज्ञानिक इमेजिंग, या हाई‑एंड प्रिंटिंग पाइपलाइन।

## बिना संपीड़न के PSD को निर्यात क्यों करें?
- **अधिकतम गुणवत्ता:** संपीड़न आर्टिफैक्ट्स के कारण कोई विवरण नहीं खोता।  
- **पूर्वानुमेय फ़ाइल आकार:** RAW फ़ाइलों का आकार सीधे इमेज आयाम और बिट डेप्थ को दर्शाता है।  
- **सरल डाउनस्ट्रीम प्रोसेसिंग:** अन्य टूल्स पिक्सेल डेटा को बिना डिकम्प्रेस किए पढ़ सकते हैं।

## पूर्वापेक्षाएँ
कोड लिखना शुरू करने से पहले हमें कुछ पूर्वापेक्षाओं की जाँच करनी होगी। चिंता न करें; ये काफी सरल हैं!

### जावा डेवलपमेंट किट (JDK)
- सुनिश्चित करें कि आपके सिस्टम पर JDK 8 या उससे ऊपर स्थापित है। यदि नहीं, तो [Oracle website](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) पर जाएँ और नवीनतम संस्करण डाउनलोड करें।

### एकीकृत विकास वातावरण (IDE)
- IntelliJ IDEA, Eclipse, या NetBeans जैसे एक अच्छे IDE से आपका काम आसान हो जाएगा। यदि अभी तक सेट नहीं किया है, तो एक स्थापित करें!

### Aspose.PSD for Java लाइब्रेरी
- Aspose.PSD for Java लाइब्रेरी डाउनलोड करें। आप नवीनतम रिलीज़ [here](https://releases.aspose.com/psd/java/) से प्राप्त कर सकते हैं। 

### जावा का मूल ज्ञान
- आपको जावा प्रोग्रामिंग और ऑब्जेक्ट‑ओरिएंटेड पैरेडाइम की बुनियादी समझ होनी चाहिए ताकि आप सहजता से अनुसरण कर सकें।

### एक PSD फ़ाइल
- परीक्षण के लिए एक नमूना PSD फ़ाइल तैयार रखें। आप इसे Photoshop में बना सकते हैं या ऑनलाइन एक मुफ्त नमूना डाउनलोड कर सकते हैं। 

अब जब सब कुछ तैयार है, चलिए कोड में डुबकी लगाते हैं!

## पैकेज आयात करें
शुरू करने के लिए, हमें अपने कोड के लिए आवश्यक पैकेज आयात करने होंगे। नीचे उन इम्पोर्ट्स की सूची है जिसकी आपको आवश्यकता होगी:

```java
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.CompressionMethod;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PsdOptions;
```

ये इम्पोर्ट्स आवश्यक क्लासेज़ और मेथड्स को हमारे प्रोजेक्ट में लाते हैं, जिससे हम PSD फ़ाइलों को सहजता से हेरफेर कर सकते हैं। अब हम प्रक्रिया को प्रबंधनीय चरणों में विभाजित करेंगे।

## चरण 1: अपनी फ़ाइल निर्देशिका सेट करना
सबसे पहले, आपको यह निर्दिष्ट करना होगा कि आपका PSD फ़ाइल कहाँ स्थित है और आउटपुट कहाँ सहेजना है। हमारे नमूना कोड में, हम एक वेरिएबल बनाएँगे जो डायरेक्टरी पाथ को रखेगा।

```java
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को वास्तविक पाथ से बदलें जहाँ आपका PSD फ़ाइल (`layers.psd`) संग्रहीत है। ऐसा करने से आपका प्रोग्राम फ़ाइल को खोजने में सक्षम होगा।

## चरण 2: PSD फ़ाइल लोड करना
अब, `Image.load()` मेथड का उपयोग करके PSD फ़ाइल लोड करते हैं, और इसे `PsdImage` प्रकार में कास्ट करते हैं।

```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "layers.psd");
```

यह लाइन `Image` क्लास के `load` मेथड को कॉल करती है, जिससे आपका PSD फ़ाइल मेमोरी में लोड हो जाता है। इसे `PsdImage` में कास्ट करने से Java को बताता है कि यह इमेज एक PSD फ़ाइल है जिसके पास PSD‑विशिष्ट कार्यक्षमताएँ हैं।

## चरण 3: सहेजने के विकल्प कॉन्फ़िगर करना
अगला, हमें फ़ाइल सहेजने के विकल्प सेट करने हैं, विशेष रूप से यह निर्दिष्ट करना है कि हम आउटपुट **असंपीडित** (अर्थात, convert PSD to RAW) चाहते हैं।

```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw);
```

`PsdOptions` क्लास हमें PSD फ़ाइल को सहेजने के विभिन्न विकल्प निर्दिष्ट करने की अनुमति देती है। `setCompressionMethod` को `CompressionMethod.Raw` पर सेट करने से हमारी सहेजी गई फ़ाइल असंपीडित रहती है और उच्च गुणवत्ता बनाए रखती है।

## चरण 4: असंपीडित PSD फ़ाइल सहेजना
अब नई कॉन्फ़िगर की गई PSD इमेज को सहेजने का समय है।

```java
psdImage.save(dataDir + "uncompressed_out.psd", saveOptions);
```

यह लाइन हमारे `PsdImage` इंस्टेंस (`psdImage`) पर save फ़ंक्शन को निष्पादित करती है। यह फ़ाइल को `uncompressed_out.psd` के रूप में निर्दिष्ट डायरेक्टरी में सहेजती है और पहले परिभाषित विकल्प लागू करती है।

## चरण 5: नई बनाई गई छवि को पुनः खोलना
सहेजने के बाद, चलिए हमारे आउटपुट इमेज को पुनः लोड करते हैं ताकि यह सुनिश्चित हो सके कि सब कुछ अपेक्षित रूप से काम किया।

```java
PsdImage img = (PsdImage) Image.load(dataDir + "uncompressed_out.psd");
```

`load` को फिर से कॉल करके, हम एक नया `PsdImage` इंस्टेंस बना सकते हैं जो सहेजी गई फ़ाइल से मेल खाता है। यह चरण महत्वपूर्ण है यदि आप सहेजने के बाद इमेज को संशोधित या प्रदर्शित करना चाहते हैं।

## चरण 6: छवि को ड्रॉ या संशोधित करना
अंत में, आप नई खोली गई इमेज पर ड्रॉ या अन्य परिवर्तन कर सकते हैं।

```java
Graphics graphics = new Graphics(img);
```

यहाँ हम एक `Graphics` ऑब्जेक्ट इनिशियलाइज़ करते हैं, जो हमारे `img` पर विभिन्न ग्राफ़िकल ऑपरेशन्स करने की अनुमति देता है। आप आकार बना सकते हैं, टेक्स्ट जोड़ सकते हैं, या यदि चाहें तो लेयर्स को भी संशोधित कर सकते हैं!

- **वैज्ञानिक इमेजिंग:** विश्लेषण के लिए कच्चा पिक्सेल डेटा निर्यात करें।
