# PSD को कैसे संपादित करें: जावा में टेक्स्ट लेयर बाउंड बॉक्स समायोजित करें

## परिचय
यदि आप प्रोग्रामेटिक रूप से **how to edit PSD** फ़ाइलों को संपादित करने के बारे में सोच रहे हैं—विशेषकर जब आपको **edit Photoshop text layer** गुणों को बदलने की आवश्यकता हो—तो जावा के लिए Aspose.PSD लाइब्रेरी बहुत उपयोगी है। यह ट्यूटोरियल आपको **adjust text bound box** और **retrieve text bound box** जानकारी को **aspose psd java** का उपयोग करके प्राप्त करने के सटीक चरणों के माध्यम से ले जाता है। चाहे आप एक अनुभवी डेवलपर हों या जावा के साथ अभी शुरुआत कर रहे हों, आपको स्पष्ट, संवादात्मक मार्गदर्शन मिलेगा जो PSD हेरफेर को सरल और सुलभ बनाता है। चलिए शुरू करते हैं!

## त्वरित उत्तर
- **जावा में PSD फ़ाइलों को संपादित करने में कौन सी लाइब्रेरी मदद करती है?** Aspose.PSD for Java.  
- **क्या मैं टेक्स्ट लेयर के बाउंड बॉक्स को समायोजित कर सकता हूँ?** Yes—use `getTextBoundBox()` and `setSize()` to modify it.  
- **क्या मुझे Photoshop स्थापित करने की आवश्यकता है?** No, Aspose.PSD works independently of Adobe Photoshop.  
- **मुख्य पूर्वापेक्षाएँ क्या हैं?** JDK, an IDE, and the Aspose.PSD for Java library.  
- **बेसिक इम्प्लीमेंटेशन को पूरा करने में कितना समय लगता है?** About 10‑15 minutes to run the sample code.

## Aspose.PSD के साथ “how to edit psd” क्या है?
प्रोग्रामेटिक रूप से PSD को संपादित करना मतलब फ़ाइल को खोलना, उसकी लेयर्स तक पहुंचना, और आकार, स्थिति, या टेक्स्ट सामग्री जैसी गुणों को बदलना—बिना Photoshop लॉन्च किए। Aspose.PSD एक समृद्ध API प्रदान करता है जो जटिल PSD फ़ॉर्मेट को सारांशित करता है, जिससे आप अपनी आवश्यक लॉजिक पर ध्यान केंद्रित कर सकते हैं।

## जावा के लिए Aspose.PSD क्यों उपयोग करें?
Aspose.PSD for Java एक व्यापक फीचर सेट प्रदान करता है जो PSD हेरफेर को सरल और कुशल बनाता है। यह Photoshop की आवश्यकता को समाप्त करता है, सभी प्रमुख लेयर प्रकारों का समर्थन करता है, बड़े फ़ाइलों के लिए भी तेज़ प्रोसेसिंग प्रदान करता है, और Windows, Linux, और macOS पर्यावरण में लगातार चलता है।

- **Photoshop की आवश्यकता नहीं** – works on any server or desktop environment.  
- **पूर्ण लेयर समर्थन** – raster, vector, and text layers can be read or modified.  
- **उच्च‑प्रदर्शन इंजन** – processes files up to 500 MB in under 2 seconds and handles batch jobs of 1,000 files with less than 200 MB peak memory.  
- **क्रॉस‑प्लेटफ़ॉर्म** – run on Windows, Linux, or macOS with the same code.

## पैकेज आयात करें
पहला कदम वह क्लासेस आयात करना है जिनकी आपको आवश्यकता होगी। इसे एक DIY प्रोजेक्ट शुरू करने से पहले सभी उपकरण इकट्ठा करने जैसा समझें।

```java
import com.aspose.psd.Image;
import com.aspose.psd.Size;
import com.aspose.psd.examples.Utils.Assert;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.TextLayer;
```

ये इम्पोर्ट्स आपको इमेज हैंडलिंग, साइज मैनिपुलेशन, और `TextLayer` क्लास तक पहुंच देते हैं जिसके साथ हम काम करेंगे।

## चरण 1: अपनी फ़ाइल पाथ सेट करें
अपनी PSD फ़ाइल के स्थान को निर्दिष्ट करें। यह प्रदर्शन शुरू होने से पहले मंच तैयार करने जैसा है।

```java
String dataDir = "Your Document Directory"; 
String sourceFileName = dataDir + "LayerWithText.psd";
```

`"Your Document Directory"` को अपने मशीन पर वास्तविक फ़ोल्डर पाथ से बदलें।

## चरण 2: PSD फ़ाइल लोड करें
अब हम PSD को खोलते हैं ताकि हम उसकी लेयर्स के साथ इंटरैक्ट कर सकें।

```java
PsdImage im = (PsdImage) Image.load(sourceFileName);
```

`Image.load` मेथड फ़ाइल को पढ़ता है और एक `PsdImage` ऑब्जेक्ट लौटाता है जो हेरफेर के लिए तैयार है।

## चरण 3: टेक्स्ट लेयर प्राप्त करें
उस विशिष्ट टेक्स्ट लेयर की पहचान करें जिसे आप समायोजित करना चाहते हैं। लेयर्स शून्य‑इंडेक्स्ड होती हैं, इसलिए `[1]` दूसरा लेयर दर्शाता है।

```java
TextLayer textLayer = (TextLayer) im.getLayers()[1];
```

सुनिश्चित करें कि आप सही लेयर को लक्षित कर रहे हैं; अन्यथा आप गलत सामग्री को संशोधित कर सकते हैं।

## चरण 4: लेयर का आकार जांचें
किसी भी परिवर्तन से पहले, वर्तमान आकार की पुष्टि करना एक अच्छा विचार है। यह एक सत्यापन जांच के रूप में कार्य करता है।

```java
Size correctOpticalSize = new Size(127, 45);
Size opticalSize = textLayer.getSize();
Assert.areEqual(correctOpticalSize, opticalSize);
```

यदि आकार मेल नहीं खाते, तो `Assert` एक अलर्ट उठाएगा, जिससे आपको पता चलेगा कि कुछ गड़बड़ है।

## चरण 5: बाउंड बॉक्स का आकार प्राप्त करें
अब हम **टेक्स्ट बाउंड बॉक्स**—वह आयत जो रेंडर किए गए टेक्स्ट को घेरती है—को प्राप्त करते हैं।

```java
Size correctBoundBox = new Size(172, 62);
Size boundBox = textLayer.getTextBoundBox();
Assert.areEqual(correctBoundBox, boundBox);
```

आप इस आकार की तुलना अपनी अपेक्षित आयामों से कर सकते हैं या आगे के समायोजनों की गणना के लिए इसका उपयोग कर सकते हैं।

## Aspose.PSD Java का उपयोग करके टेक्स्ट बाउंड बॉक्स कैसे प्राप्त करें?
टेक्स्ट बाउंड बॉक्स प्राप्त करने के लिए, पहले `Image.load` से PSD फ़ाइल लोड करें और `PsdImage` इंस्टेंस प्राप्त करें। फिर `psdImage.getLayers()` के माध्यम से या इंडेक्स द्वारा लक्ष्य `TextLayer` खोजें। उस लेयर पर `getTextBoundBox()` मेथड कॉल करें; यह पिक्सेल में सटीक चौड़ाई और ऊँचाई के साथ एक `Size` ऑब्जेक्ट लौटाता है, जिसे आप लॉग कर सकते हैं या आगे की गणनाओं में उपयोग कर सकते हैं।

## Aspose.PSD Java के साथ टेक्स्ट बाउंड बॉक्स कैसे समायोजित करें?
वर्तमान बाउंड बॉक्स मिलने के बाद, आप `TextLayer` पर सीधे `setSize(new Size(width, height))` कॉल करके इसे संशोधित कर सकते हैं, जहाँ आप इच्छित चौड़ाई और ऊँचाई मान प्रदान करते हैं। वैकल्पिक रूप से, आप `transform()` मेथड के साथ एक ट्रांसफ़ॉर्मेशन मैट्रिक्स लागू करके टेक्स्ट को स्केल या पुनः स्थित कर सकते हैं, इससे पहले कि लेयर को रास्टराइज़ किया जाए। दोनों तरीकों से दृश्य लेआउट आपके डिज़ाइन आवश्यकताओं के अनुसार अपडेट हो जाता है।
