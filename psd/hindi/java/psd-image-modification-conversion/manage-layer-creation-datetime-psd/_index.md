# नया PSD लेयर बनाएं और जावा में निर्माण तिथि‑समय प्रबंधित करें

## परिचय
जब आप प्रोग्रामेटिक रूप से Photoshop (PSD) फ़ाइलों के साथ काम करते हैं, तो **create new PSD layer** ऑब्जेक्ट बनाना और उनके निर्माण टाइमस्टैम्प को ट्रैक करना वास्तव में एक गेम‑चेंजर है। चाहे आप डिज़ाइन एसेट्स के लिए वर्ज़न‑कंट्रोल सिस्टम बना रहे हों, बैच एडिट्स को ऑटोमेट कर रहे हों, या सिर्फ सहयोगी प्रोजेक्ट्स के लिए ऑडिट ट्रेल की आवश्यकता हो, लेयर निर्माण तिथि को पढ़ना और सेट करना जानने से आप हर बदलाव की पूरी उत्पत्ति बनाए रख सकते हैं। इस ट्यूटोरियल में हम Aspose.PSD for Java का उपयोग करके पूरी प्रक्रिया को समझेंगे—PSD लोड करने से, लेयर की निर्माण तिथि प्राप्त करने, उसे वैध करने, और अंत में एक नया एडजस्टमेंट लेयर जोड़ने तक।

## त्वरित उत्तर
- **Java में PSD फ़ाइलों को संभालने वाली लाइब्रेरी कौन सी है?** Aspose.PSD for Java  
- **क्या मैं लेयर की निर्माण तिथि पढ़ सकता हूँ?** हाँ, `layer.getLayerCreationDateTime()` का उपयोग करके  
- **क्या नया एडजस्टमेंट लेयर जोड़ना संभव है?** बिल्कुल – `im.addLevelsAdjustmentLayer()` एक बनाता है  
- **क्या उत्पादन उपयोग के लिए लाइसेंस चाहिए?** गैर‑ट्रायल डिप्लॉयमेंट के लिए एक व्यावसायिक लाइसेंस आवश्यक है  
- **कौन सा Java संस्करण समर्थित है?** JDK 8 या बाद का  

## “create new PSD layer” क्या है?
नया PSD लेयर बनाना मतलब प्रोग्रामेटिक रूप से एक नया लेयर ऑब्जेक्ट—जैसे कि एडजस्टमेंट, टेक्स्ट, या पिक्सेल लेयर—को मौजूदा PSD दस्तावेज़ में सम्मिलित करना है। यह ऑपरेशन आपको फ़ोटोशॉप को मैन्युअली खोले बिना इमेज को विस्तारित या संशोधित करने की अनुमति देता है।

## लेयर निर्माण DateTime का प्रबंधन क्यों करें?
प्रत्येक लेयर की निर्माण DateTime को ट्रैक करने से आपको मदद मिलती है:
- **समीक्षाओं का ऑडिट** – यह जानने के लिए कि लेयर कब जोड़ी गई।  
- **एसेट्स को सिंक्रनाइज़** करने के लिए टीमों के बीच टाइमस्टैम्प की तुलना करके।  
- **वर्कफ़्लो को ऑटोमेट** करने के लिए जो समय‑आधारित नियमों पर निर्भर होते हैं (जैसे, एक महीने से पुराने लेयर्स को छिपाएँ)।

## पैकेज इम्पोर्ट करें
पहले, आपको आवश्यक Aspose.PSD क्लासेज़ और Java यूटिलिटीज़ इम्पोर्ट करें। ये इम्पोर्ट्स आपको इमेज हैंडलिंग, लेयर मैनिपुलेशन, और डेट फ़ॉर्मेटिंग तक पहुँच प्रदान करते हैं।

```java
import com.aspose.psd.Image;
import com.aspose.psd.examples.Utils.Assert;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
import java.text.DateFormat;
import java.text.SimpleDateFormat;
import java.util.Date;
```

## चरण 1: अपने दस्तावेज़ डायरेक्टरी सेट करें
उस फ़ोल्डर को निर्दिष्ट करें जिसमें वह PSD हो जिसे आप उपयोग करना चाहते हैं। प्लेसहोल्डर को अपने मशीन पर पूर्ण पाथ से बदलें।

```java
String dataDir = "Your Document Directory";
```

## चरण 2: PSD फ़ाइल लोड करें
`PsdImage` इंस्टेंस बनाएं लक्ष्य फ़ाइल को लोड करके। यह ऑब्जेक्ट सभी लेयर ऑपरेशन्स के लिए एंट्री पॉइंट है।

```java
String sourceName = dataDir + "OneLayer.psd";
PsdImage im = (PsdImage) Image.load(sourceName);
```

## चरण 3: लेयर और उसकी निर्माण तिथि तक पहुँचें
पहला लेयर (इंडेक्स 0) प्राप्त करें और उसकी निर्माण टाइमस्टैम्प निकालें। यह वह तिथि है जिसे आप बाद में तुलना या लॉग करेंगे।

```java
Layer layer = im.getLayers()[0];
Date creationDateTime = layer.getLayerCreationDateTime();
```

## चरण 4: निर्माण तिथि को फ़ॉर्मेट करें
कच्चे `Date` ऑब्जेक्ट को मानव‑पठनीय स्ट्रिंग में बदलें। यदि आप अलग फ़ॉर्मेट पसंद करते हैं तो पैटर्न को समायोजित करें।

```java
DateFormat dateFormat = new SimpleDateFormat("yyyy/MM/dd HH:mm:ss");
```

## चरण 5: निर्माण तिथि को वैध करें
डेमॉन्स्ट्रेशन के लिए, हम प्राप्त तिथि को अपेक्षित मान से तुलना करते हैं। वास्तविक प्रोजेक्ट्स में आप इसे डेटाबेस रिकॉर्ड या कॉन्फ़िगरेशन फ़ाइल से तुलना कर सकते हैं।

```java
Date expectedDateTime = new Date("2018/7/17 8:57:24");
Assert.areEqual(expectedDateTime, creationDateTime);
```

## चरण 6: नया लेयर बनाएं
अब हम वास्तव में **create new PSD layer** ऑब्जेक्ट बनाते हैं। यहाँ हम एक Levels एडजस्टमेंट लेयर जोड़ते हैं, जो मूल पिक्सेल को बदले बिना टोनल रेंज को समायोजित करने देता है।

```java
Date now = new Date();
Layer createdLayer = im.addLevelsAdjustmentLayer();
```

> **Pro tip:** `now` वेरिएबल उस क्षण को कैप्चर करता है जब आप लेयर जोड़ते हैं, जिसे आप बाद में यदि कस्टम टाइमस्टैम्प चाहिए तो मेटाडेटा के रूप में स्टोर कर सकते हैं।

## सामान्य समस्याएँ और समाधान
| समस्या | क्यों होता है | समाधान |
|-------|----------------|-----|
| `layer.getLayerCreationDateTime()` पर `NullPointerException` | PSD में कोई लेयर नहीं है या लेयर इंडेक्स सीमा से बाहर है। | `im.getLayers().length > 0` को एक्सेस करने से पहले सत्यापित करें। |
| वैलिडेशन में तिथि असंगति | `Date` कंस्ट्रक्टर स्ट्रिंग को लोकल‑निर्भर तरीके से पार्स करता है। | विश्वसनीय पार्सिंग के लिए `SimpleDateFormat.parse("2018/07/17 08:57:24")` का उपयोग करें। |
| Photoshop में नया लेयर दिखाई नहीं देता | एडजस्टमेंट लेयर डिफ़ॉल्ट रूप से छिपा हो सकता है। | निर्माण के बाद `createdLayer.setVisible(true);` कॉल करें। |
