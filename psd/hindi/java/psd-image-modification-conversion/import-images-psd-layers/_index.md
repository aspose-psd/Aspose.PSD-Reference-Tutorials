## “how to import psd” क्या है?
PSD को इम्पोर्ट करना मतलब है प्रोग्रामेटिकली एक Photoshop डॉक्यूमेंट को लोड करना, उसकी लेयर्स को मॉडिफ़ाई करना (जैसे इमेज जोड़ना), और फिर अपडेटेड फ़ाइल को सेव करना। यह तरीका बैच प्रोसेसिंग, ऑटोमेटेड ग्राफ़िक्स जनरेशन, या बड़े एप्लिकेशन में डिज़ाइन एसेट्स को इंटीग्रेट करने के लिए आदर्श है।

## लेयर्स में PSD इमेज इम्पोर्ट करने के चरण
नीचे एक स्पष्ट, क्रमांकित वॉकथ्रू दिया गया है जो **how to add image** को PSD लेयर में जोड़ना, **set layer coordinates**, और **fill psd layer color** समझाता है।

### चरण 1: आवश्यक पैकेज इम्पोर्ट करें
सबसे पहले, हम Aspose.PSD क्लासेज़ को इम्पोर्ट करते हैं जिनकी हमें आवश्यकता होगी। यह बाकी कोड के लिए मंच तैयार करता है।

```java
import com.aspose.psd.Color;
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.Point;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.Layer;
```

### चरण 2: डॉक्यूमेंट डायरेक्टरी सेट करें
परिभाषित करें कि आपका स्रोत PSD कहाँ स्थित है और परिणाम कहाँ सेव होगा।

```java
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को अपने फ़ाइल सिस्टम पर वास्तविक पाथ से बदलें जहाँ आपके PSD फ़ाइलें स्थित हैं।

### चरण 3: अपना PSD फ़ाइल लोड करें
PSD को खोलें ताकि हम उसकी लेयर्स के साथ काम कर सकें।

```java
PsdImage image = (PsdImage) Image.load(dataDir + "sample.psd");
```

सुनिश्चित करें कि `"sample.psd"` वह फ़ाइलनाम है जिसे आप एडिट करना चाहते हैं।

### चरण 4: टार्गेट लेयर निकालें
उस लेयर को चुनें जो नई इमेज प्राप्त करेगी। यहाँ हम दूसरी लेयर (इंडेक्स 1) का उपयोग कर रहे हैं।

```java
Layer layer = image.getLayers()[1];
```

यदि आपको कोई अलग लेयर चाहिए, तो बस इंडेक्स बदल दें।

### चरण 5: इम्पोर्ट करने के लिए नई इमेज बनाएं
अब हम **add image psd layer** करेंगे एक नई `PsdImage` बनाकर, जिस पर हम ड्रॉ करेंगे।

```java
PsdImage drawImage = new PsdImage(200, 200);
```

आप अपनी स्रोत इमेज के अनुसार चौड़ाई और ऊँचाई को समायोजित कर सकते हैं।

### चरण 6: इमेज सतह को फ़िल करें (लेयर कलर सेट करें)
आइए **fill psd layer color** को एक चमकीले पीले बैकग्राउंड से भरें। यह दर्शाता है कि ड्रॉ करने से पहले सॉलिड कलर कैसे सेट किया जाता है।

```java
Graphics g = new Graphics(drawImage);
g.clear(Color.getYellow());
```

`Color.getYellow()` को अपनी पसंद के किसी भी `Color` से बदलने में संकोच न करें।

### चरण 7: लेयर पर इमेज ड्रॉ करें (लेयर कोऑर्डिनेट्स सेट करें)
यह **how to add image** का मुख्य भाग है – हम नई बनाई गई इमेज को चुनी हुई लेयर पर विशिष्ट कोऑर्डिनेट्स के साथ रखते हैं।

```java
layer.drawImage(new Point(10, 10), drawImage);
```

`Point(10, 10)` कॉल **sets layer coordinates** (X = 10, Y = 10) करता है। इन मानों को बदलकर इमेज को ठीक उसी जगह रखें जहाँ आपको चाहिए।

### चरण 8: अपडेटेड PSD फ़ाइल सेव करें
अंत में, बदलावों को डिस्क पर लिखें।

```java
image.save(dataDir + "ImportImageToPSDLayer_out.psd");
```

आउटपुट फ़ाइल को एक अर्थपूर्ण नाम दें; उदाहरण में मूल फ़ाइल को अनछुआ रखने के लिए `_out` जोड़ा गया है।

## सामान्य समस्याएँ और समाधान
- **इमेज खाली दिख रही है** – ड्रॉ करने से पहले `Graphics.clear()` कॉल करना सुनिश्चित करें; अन्यथा कैनवास ट्रांसपेरेंट रह सकता है।  
- **गलत लेयर मॉडिफ़ाई हो रही है** – याद रखें कि लेयर इंडेक्स 0 से शुरू होते हैं। `getLayers()` में उपयोग किए गए इंडेक्स को दोबारा चेक करें।  
- **असमर्थित कलर प्रोफ़ाइल** – Aspose.PSD अधिकांश प्रोफ़ाइल्स को संभालता है, लेकिन यदि आपको कलर शिफ्ट दिखे, तो इम्पोर्ट करने से पहले स्रोत इमेज को sRGB में कनवर्ट करने की कोशिश करें।
