# Aspose.PSD for Java में फ़ॉन्ट कैसे बदलें

## परिचय

यदि आप एक Java एप्लिकेशन में Photoshop (PSD) फ़ाइलों के साथ काम कर रहे हैं, तो गायब फ़ॉन्ट्स दृश्य लेआउट को बिगाड़ सकते हैं और रेंडरिंग त्रुटियाँ उत्पन्न कर सकते हैं। **फ़ॉन्ट बदलने** का तेज़ और विश्वसनीय तरीका डिज़ाइन की सटीकता बनाए रखने के लिए आवश्यक है। इस ट्यूटोरियल में आप सीखेंगे कि Aspose.PSD for Java का उपयोग करके गायब फ़ॉन्ट्स को कैसे बदलें, परिणाम को PNG में बदलें, और अपनी इमेज कन्वर्ज़न वर्कफ़्लो को सुगम रखें। अंत तक, आप इमेज में फ़ॉन्ट बदलने, PSD को PNG के रूप में सहेजने, और डेवलपर्स द्वारा आम तौर पर सामना किए जाने वाले मुद्दों से बचने में सक्षम होंगे।

## त्वरित उत्तर
- **PSD प्रोसेसिंग में “फ़ॉन्ट बदलना” का क्या अर्थ है?** यह गायब या अनुपलब्ध टाइपफ़ेस को आपके द्वारा निर्दिष्ट फ़ॉलबैक फ़ॉन्ट से बदल देता है।  
- **कौन सी लाइब्रेरी इसे स्वचालित रूप से संभालती है?** Aspose.PSD for Java `PsdLoadOptions.setDefaultReplacementFont` प्रदान करती है।  
- **क्या मैं बदलने के बाद PSD को PNG में भी बदल सकता हूँ?** हाँ – `PngOptions` का उपयोग करें और `psdImage.save` को कॉल करें।  
- **उत्पादन उपयोग के लिए क्या लाइसेंस चाहिए?** गैर‑मूल्यांकन बिल्ड्स के लिए एक वैध Aspose.PSD लाइसेंस आवश्यक है।  
- **कौन सा Java संस्करण समर्थित है?** वर्तमान Aspose.PSD रिलीज़ के साथ कोई भी Java 8+ रनटाइम काम करता है।

## PSD फ़ाइलों में फ़ॉन्ट प्रतिस्थापन क्या है?

जब कोई PSD ऐसे फ़ॉन्ट का संदर्भ देता है जो होस्ट मशीन पर स्थापित नहीं है, तो रेंडरिंग इंजन एक सामान्य फ़ॉन्ट पर फ़ॉलबैक करता है, जिससे अक्सर टेक्स्ट का विसंरेखण हो जाता है। फ़ॉन्ट प्रतिस्थापन आपको एक डिफ़ॉल्ट फ़ॉन्ट (जैसे Arial) निर्धारित करने की अनुमति देता है, जिसे लाइब्रेरी हर बार जब वह कोई गायब टाइपफ़ेस पाती है, उपयोग करती है, जिससे दृश्य आउटपुट सुसंगत रहता है।

## Aspose.PSD के साथ गायब फ़ॉन्ट्स को क्यों बदलें?

- **शून्य‑निर्भरता समाधान** – नेटिव Photoshop या बाहरी टूल्स की आवश्यकता नहीं।  
- **बैच‑तैयार** – समान सेटिंग्स के साथ लूप में दर्जनों फ़ाइलें प्रोसेस करें।  
- **इमेज कन्वर्ज़न तैयार** – प्रतिस्थापन के बाद आप सीधे **PSD को PNG के रूप में सहेज** सकते हैं या **PSD को PNG में बदल** सकते हैं बिना अतिरिक्त चरणों के।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS Java रनटाइम पर काम करता है।

## पूर्वापेक्षाएँ

1. **Aspose.PSD लाइब्रेरी** – [releases page](https://releases.aspose.com/psd/java/) से Aspose.PSD for Java लाइब्रेरी डाउनलोड और इंस्टॉल करें।  
2. **Java विकास पर्यावरण** – JDK 8 या बाद का स्थापित और कॉन्फ़िगर किया हुआ होना चाहिए।  

अब जब हमारे पास आवश्यक चीज़ें हैं, चलिए कोड में डुबकी लगाते हैं।

## पैकेज इम्पोर्ट करें

आवश्यक Aspose.PSD क्लासेज़ को इम्पोर्ट करके शुरू करें। यह आपको इमेज लोडिंग, फ़ॉन्ट‑प्रतिस्थापन विकल्प, और PNG निर्यात क्षमताओं तक पहुँच देता है।

```java
import com.aspose.psd.Image;

import com.aspose.psd.fileformats.png.PngColorType;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageloadoptions.PsdLoadOptions;
import com.aspose.psd.imageoptions.PngOptions;
```

## चरण 1: अपने दस्तावेज़ निर्देशिका को सेट करें

उस फ़ोल्डर को परिभाषित करें जिसमें स्रोत PSD है और जहाँ परिणामी PNG सहेजा जाएगा।

```java
String dataDir = "Your Document Directory";
```

## चरण 2: स्रोत और लक्ष्य फ़ाइलें निर्दिष्ट करें

इनपुट PSD और आउटपुट PNG के पूर्ण पाथ प्रदान करें। यह वर्कफ़्लो को स्पष्ट बनाता है और कोड को रखरखाव में आसान बनाता है।

```java
String sourceFile = dataDir + "sample.psd";
String destName = dataDir + "result.png";
```

## चरण 3: फ़ॉन्ट प्रतिस्थापन सेटिंग्स कॉन्फ़िगर करें

एक `PsdLoadOptions` इंस्टेंस बनाएं और Aspose.PSD को बताएं कि जब वह कोई गायब फ़ॉन्ट पाता है तो कौन सा फ़ॉन्ट उपयोग करना है। इस उदाहरण में हम **Arial** का उपयोग करते हैं, लेकिन आप इसे अपने सिस्टम में स्थापित किसी भी फ़ॉन्ट से बदल सकते हैं।

```java
PsdLoadOptions loadOptions = new PsdLoadOptions();
loadOptions.setDefaultReplacementFont("Arial");
```

## चरण 4: PSD इमेज लोड करें और फ़ॉन्ट बदलें

उपरोक्त विकल्पों के साथ PSD लोड करें। लाइब्रेरी स्वचालित रूप से किसी भी गायब फ़ॉन्ट को आपके द्वारा निर्दिष्ट डिफ़ॉल्ट से बदल देती है।

```java
Image image = Image.load(sourceFile, loadOptions);
PsdImage psdImage = (PsdImage) image;
```

## चरण 5: संशोधित इमेज सहेजें

PNG निर्यात विकल्प कॉन्फ़िगर करें—अल्फा चैनल के साथ ट्रू कलर—ताकि पारदर्शिता बनी रहे। यह चरण फ़ॉन्ट प्रतिस्थापन के बाद **PSD को PNG में बदलने** को भी दर्शाता है।

```java
PngOptions options = new PngOptions();
options.setColorType(PngColorType.TruecolorWithAlpha);
psdImage.save(destName, options);
```

### अभी क्या हुआ?

- PSD को एक फ़ॉलबैक फ़ॉन्ट (Arial) के साथ खोला गया।  
- सभी टेक्स्ट लेयर जो मूल रूप से गायब फ़ॉन्ट्स का संदर्भ देती थीं, अब Arial में रेंडर होती हैं।  
- अंतिम इमेज को PNG के रूप में सहेजा गया, प्रभावी रूप से **PSD को PNG के रूप में सहेजते** हुए दृश्य सटीकता बनी रही।
