# चरण-दर-चरण फ़िल्टर: मीडियन और विईनर फ़िल्टर लागू करें (Java)

## परिचय

यदि आप जावा में शोरयुक्त छवियों को साफ़ करने के लिए **step by step filter** वर्कफ़्लो की तलाश में हैं, तो आप सही जगह पर आए हैं। Aspose.PSD for Java दोनों मीडियन और विईनर फ़िल्टर लागू करना आसान बनाता है, और यह प्रोसेसिंग के बाद आपको **convert PSD to GIF** भी करने देता है। इस ट्यूटोरियल में हम आपको लाइब्रेरी सेटअप से लेकर फ़िल्टर किए गए परिणाम को सहेजने तक सब कुछ दिखाएंगे—ताकि आप अपने एप्लिकेशन में उच्च‑गुणवत्ता वाली इमेज डीनॉइज़िंग को आत्मविश्वास के साथ एकीकृत कर सकें।

## त्वरित उत्तर
- **Median फ़िल्टर क्या करता है?** यह नमक‑और‑काली मिर्च शोर को कम करता है जबकि किनारों को संरक्षित रखता है।  
- **Wiener फ़िल्टर कब उपयोग करना चाहिए?** स्थानीय इमेज वैरिएंस को ध्यान में रखते हुए अनुकूली शोर घटाने के लिए।  
- **कोड चलाने के लिए लाइसेंस चाहिए?** विकास के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **क्या मैं आउटपुट को GIF के रूप में सहेज सकता हूँ?** हाँ—Aspose.PSD आपको **convert PSD to GIF** एक ही चरण में करने देता है।  
- **इम्प्लीमेंटेशन में कितना समय लगता है?** बेसिक सेटअप के लिए आमतौर पर 10 मिनट से कम।

## चरण-दर-चरण फ़िल्टर क्या है?

*step by step filter* दृष्टिकोण इमेज प्रोसेसिंग को स्पष्ट, प्रबंधनीय चरणों में विभाजित करता है—छवि लोड करना, फ़िल्टर विकल्प कॉन्फ़िगर करना, फ़िल्टर लागू करना, और अंत में परिणाम सहेजना। यह व्यवस्थित प्रवाह आपको प्रत्येक भाग को डिबग करने, कोड को पुन: उपयोग करने, और विभिन्न इमेज फ़ॉर्मेट के लिए प्रक्रिया को अनुकूलित करने में मदद करता है।

## क्यों उपयोग करें Aspose.PSD for Java?

- **विस्तृत फ़ॉर्मेट समर्थन** – PSD, PNG, JPEG, GIF और अधिक को संभालता है।  
- **कोई बाहरी निर्भरताएँ नहीं** – शुद्ध Java, किसी भी प्रोजेक्ट में एम्बेड करना आसान।  
- **इन‑बिल्ट फ़िल्टर विकल्प** – Median, Wiener, और अन्य उन्नत फ़िल्टर बॉक्स से बाहर तैयार हैं।  
- **एक‑क्लिक रूपांतरण** – प्रोसेसिंग के बाद सीधे GIF, PNG, या JPEG में निर्यात करें।

## पैकेज इम्पोर्ट करें

छवि हैंडलिंग और फ़िल्टर विकल्पों तक पहुँच प्रदान करने वाले आवश्यक क्लासेस को इम्पोर्ट करके शुरू करें।

```java
import com.aspose.psd.Image;
import com.aspose.psd.RasterImage;

import com.aspose.psd.imagefilters.filteroptions.MedianFilterOptions;
import com.aspose.psd.imageoptions.GifOptions;
```

## चरण-दर-चरण फ़िल्टर: मीडियन फ़िल्टर कैसे लागू करें

### चरण 1: छवि लोड करें

सबसे पहले, API को उस PSD फ़ाइल की ओर इंगित करें जिसे आप साफ़ करना चाहते हैं।

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";

// Load the source image
Image image = Image.load(sourceFile);
```

### चरण 2: Image को RasterImage में कास्ट करें

फ़िल्टर रास्टर डेटा पर काम करता है, इसलिए हम सामान्य `Image` को `RasterImage` में कास्ट करते हैं।

```java
// Cast the image into RasterImage
RasterImage rasterImage = (RasterImage)image;
if (rasterImage == null) {
    return;
}
```

### चरण 3: MedianFilterOptions इंस्टेंस बनाएं

मीडियन कर्नेल का आकार कॉन्फ़िगर करें। मध्यम शोर के लिए **4** का आकार अच्छा काम करता है।

```java
// Create an instance of MedianFilterOptions class and set the filter size
MedianFilterOptions options = new MedianFilterOptions(4);
```

### चरण 4: मीडियन फ़िल्टर लागू करें

अब फ़िल्टर को पूरी छवि की सीमाओं पर लागू करें।

```java
// Apply MedianFilterOptions filter to RasterImage object
rasterImage.filter(image.getBounds(), options);
```

### चरण 5: परिणामी छवि सहेजें (PSD को GIF में बदलें)

फ़िल्टरिंग के बाद, हम आउटपुट को GIF के रूप में सहेजेंगे—**convert PSD to GIF** क्षमता को दर्शाते हुए।

```java
String destName = dataDir + "median_test_denoise_out.gif";
// Save the resultant image as a GIF
image.save(destName, new GifOptions());
```

इन चरणों का पालन करके आपने सफलतापूर्वक मीडियन फ़िल्टर लागू किया और साफ़ की गई छवि को GIF के रूप में निर्यात किया है।

## विईनर फ़िल्टर लागू करना (वैकल्पिक विस्तार)

यदि आपके प्रोजेक्ट को अनुकूली शोर घटाने की आवश्यकता है, तो आप मीडियन फ़िल्टर को विईनर फ़िल्टर से बदल सकते हैं। कोड संरचना समान है; आपको केवल `WienerFilterOptions` को इम्पोर्ट करना है और इच्छित रेडियस के साथ इंस्टैंसिएट करना है।

> **Pro tip:** दोनों फ़िल्टरों के लिए विभिन्न कर्नेल आकारों के साथ प्रयोग करें ताकि शोर हटाने और विवरण संरक्षण के बीच सही संतुलन मिल सके।

## सामान्य समस्याएँ और ट्रबलशूटिंग

| लक्षण | संभावित कारण | समाधान |
|---------|---------------|-----|
| `RasterImage` में कास्ट करने पर `ClassCastException` | इनपुट फ़ाइल रास्टर‑संगत PSD नहीं है | सुनिश्चित करें कि PSD में रास्टर लेयर्स हैं या पहले लेयर्स को रास्टर में बदलें |
| आउटपुट GIF खाली है | गंतव्य पथ गलत है या फ़ोल्डर में लिखने की अनुमति नहीं है | सुनिश्चित करें कि `dataDir` एक मौजूदा लिखने योग्य डायरेक्टरी की ओर इशारा करता है |
| फ़िल्टर का कोई प्रभाव नहीं दिख रहा | शोर स्तर के लिए कर्नेल आकार बहुत छोटा है | फ़िल्टर आकार बढ़ाएँ (उदा., `new MedianFilterOptions(6)`) |
