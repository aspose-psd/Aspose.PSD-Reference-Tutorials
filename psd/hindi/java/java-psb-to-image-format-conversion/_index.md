## अक्सर पूछे जाने वाले प्रश्न

**Q:** क्या मैं पासवर्ड‑सुरक्षित PSB फ़ाइल को कनवर्ट कर सकता हूँ?  
**A:** हाँ। Aspose.PSD आपको लोड ऑपरेशन के दौरान पासवर्ड प्रदान करके एन्क्रिप्टेड PSB फ़ाइलों को खोलने की अनुमति देता है।

**Q:** क्या PSB को JPG में कनवर्ट करने से लेयर जानकारी खो जाएगी?  
**A:** JPG में कनवर्ट करने से इमेज फ्लैटन हो जाती है, इसलिए लेयर डेटा मर्ज हो जाता है। यदि आपको लेयर्स संरक्षित रखने हैं तो PDF या PSD का उपयोग करें।

**Q:** Aspose.PSD का उपयोग करते समय PSB फ़ाइलों के आकार पर कोई सीमा है?  
**A:** लाइब्रेरी बहुत बड़े PSB फ़ाइलों (कई गीगाबाइट) को संभालने के लिए डिज़ाइन की गई है, बशर्ते आपके JVM में पर्याप्त हीप मेमोरी हो।

**Q:** क्या प्रत्येक आउटपुट फॉर्मेट के लिए अलग लाइसेंस चाहिए?  
**A:** नहीं। Aspose.PSD for Java का एक ही लाइसेंस सभी समर्थित एक्सपोर्ट फॉर्मेट्स, जैसे JPG, PDF, और PSD को कवर करता है।

**Q:** कई PSB फ़ाइलों को बैच में कनवर्ट करते समय प्रदर्शन कैसे सुधारें?  
**A:** संभव हो तो उसी `PsdImage` इंस्टेंस को पुनः उपयोग करें, और उचित मेमोरी सेटिंग्स के साथ पैरालल स्ट्रीम्स में फ़ाइलों को प्रोसेस करने पर विचार करें।
