# रनटाइम में जावा का उपयोग करके PSD फ़ाइलों में टेक्स्ट जोड़ें

## परिचय
यदि आपने कभी फ़ोटोशॉप दस्तावेज़ को मैन्युअल रूप से संपादित किया है, तो आपको पता होगा कि लेयर्स कितनी शक्तिशाली हो सकती हैं। क्या होगा अगर आप अपने जावा एप्लिकेशन से **add text to PSD** फ़ाइलों को स्वचालित रूप से जोड़ सकें? Aspose.PSD for Java लाइब्रेरी के साथ, आप रनटाइम में PSD में एक टेक्स्ट लेयर बना सकते हैं, जिससे बैच‑प्रोसेसिंग, डायनामिक ग्राफ़िक्स जेनरेशन और ऑटोमेटेड ब्रांडिंग वर्कफ़्लो के द्वार खुलते हैं। इस ट्यूटोरियल में हम पूरे प्रोसेस को कवर करेंगे, प्रोजेक्ट सेटअप से लेकर अपडेटेड फ़ाइल को सेव करने तक।

## त्वरित उत्तर
- **मुझे कौन सी लाइब्रेरी चाहिए?** Aspose.PSD for Java.  
- **क्या मैं मौजूदा PSD में टेक्स्ट जोड़ सकता हूँ?** हाँ – फ़ाइल को लोड करें, एक `TextLayer` जोड़ें, और सेव करें।  
- **प्रोडक्शन के लिए लाइसेंस चाहिए?** गैर‑इवैल्यूएशन उपयोग के लिए एक कमर्शियल लाइसेंस आवश्यक है।  
- **कौन सा जावा संस्करण समर्थित है?** JDK 8 या उससे ऊपर (हम नवीनतम LTS की सलाह देते हैं).  
- **क्या यह वेब बैक‑एंड के लिए उपयुक्त है?** बिल्कुल – API किसी भी जावा‑आधारित सर्वर वातावरण में काम करती है।

## “add text to PSD” क्या है?
PSD में टेक्स्ट जोड़ना मतलब प्रोग्रामेटिक रूप से फ़ोटोशॉप दस्तावेज़ के अंदर एक नया टेक्स्ट लेयर बनाना। यह लेयर किसी भी अन्य फ़ोटोशॉप टेक्स्ट लेयर की तरह व्यवहार करती है: आप इसे मूव कर सकते हैं, सामग्री संपादित कर सकते हैं, और स्टाइल लागू कर सकते हैं—बिना फ़ोटोशॉप खोले।

## जावा के साथ PSD में टेक्स्ट लेयर क्यों बनाएं?
- **ऑटोमेशन** – मार्केटिंग एसेट्स, वॉटरमार्क, या प्रोडक्ट लेबल्स को बल्क में जेनरेट करें।  
- **संगतता** – हजारों फ़ाइलों में एक ही फ़ॉन्ट, साइज और पोज़िशन सुनिश्चित करें।  
- **इंटीग्रेशन** – अन्य जावा सर्विसेज (ई‑कॉमर्स, रिपोर्टिंग, CI पाइपलाइन) के साथ मिलाकर ग्राफ़िक्स ऑन‑द‑फ़्लाई डिलीवर करें।

## पूर्वापेक्षाएँ
कोड लिखने से पहले सुनिश्चित करें कि आपके पास ये हैं:

1. **Java Development Kit (JDK)** – JDK 8 या नया इंस्टॉल करें। आप इसे [यहाँ डाउनलोड कर सकते हैं](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)।  
2. **Aspose.PSD for Java** – नवीनतम JAR को [Aspose रिलीज़ पेज](https://releases.aspose.com/psd/java/) से प्राप्त करें।  
3. **IDE (वैकल्पिक लेकिन उपयोगी)** – IntelliJ IDEA, Eclipse, या कोई भी पसंदीदा एडिटर।  
4. **बेसिक जावा नॉलेज** – क्लासेज़, ऑब्जेक्ट्स, और फ़ाइल I/O से परिचित हों।  
5. **एक सैंपल PSD** – इस गाइड के लिए हम `OneLayer.psd` का उपयोग करेंगे, जिसे आप अपनी पसंद के फ़ोल्डर में रखें।

## पैकेज इम्पोर्ट करें
पहले उन क्लासेज़ को इम्पोर्ट करें जिनकी आपको PSD फ़ाइलों और टेक्स्ट लेयर्स के साथ काम करने के लिए आवश्यकता होगी।

```java
import com.aspose.psd.Image;
import com.aspose.psd.Rectangle;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.fileformats.psd.layers.TextLayer;
```

ये इम्पोर्ट्स आपको Aspose.PSD की कोर फ़ंक्शनैलिटी तक पहुँच प्रदान करते हैं।

## चरण‑दर‑चरण गाइड

### चरण 1: अपने डॉक्यूमेंट डायरेक्टरी को सेट करें
उस फ़ोल्डर को परिभाषित करें जहाँ आपका स्रोत PSD है और जहाँ आउटपुट सेव होगा।

```java
String dataDir = "Your Document Directory"; 
```

`"Your Document Directory"` को अपनी फ़ाइलों के पूर्ण या रिलेटिव पाथ से बदलें।

### चरण 2: अपने स्रोत PSD फ़ाइल को लोड करें
`Image.load()` का उपयोग करके मौजूदा PSD को मेमोरी में लाएँ।

```java
String sourceFileName = dataDir + "OneLayer.psd"; 
Image img = Image.load(sourceFileName);
```

यदि पाथ सही है, तो `img` अब लोडेड फ़ोटोशॉप डॉक्यूमेंट को दर्शाता है।

### चरण 3: `PsdImage` में कास्ट करें
चूँकि हम फ़ोटोशॉप‑स्पेसिफिक फीचर्स का उपयोग कर रहे हैं, सामान्य `Image` को `PsdImage` में कास्ट करें।

```java
PsdImage im = (PsdImage)img;
```

कास्ट करने से `addTextLayer()` जैसे मेथड्स उपलब्ध हो जाते हैं।

### चरण 4: टेक्स्ट लेयर के लिए रेक्टेंगल निर्धारित करें
नए टेक्स्ट की स्थिति और आकार को परिभाषित करें। रेक्टेंगल (x, y) पोज़िशन और (width, height) साइज को निर्धारित करता है।

```java
Rectangle rect = new Rectangle(
    (int)(im.getWidth() * 0.25),
    (int)(im.getHeight() * 0.25),
    (int)(im.getWidth() * 0.5),
    (int)(im.getHeight() * 0.5)
);
```

अपनी लेआउट आवश्यकताओं के अनुसार गणनाओं को समायोजित कर सकते हैं।

### चरण 5: टेक्स्ट लेयर जोड़ें
परिभाषित रेक्टेंगल के भीतर वास्तविक टेक्स्ट लेयर बनाएं।

```java
TextLayer layer = im.addTextLayer("Added text", rect);
```

`"Added text"` को किसी भी स्ट्रिंग से बदलें जो आप PSD में दिखाना चाहते हैं। यही वह जगह है जहाँ हम **create text layer PSD** प्रोग्रामेटिकली करते हैं।

### चरण 6: अपडेटेड PSD फ़ाइल को सेव करें
संशोधित डॉक्यूमेंट को नई फ़ाइल में लिखें ताकि मूल फ़ाइल ओवरराइट न हो।

```java
String psdPath = dataDir + "ImageWithTextLayer.psd";
im.save(psdPath);
```

एक्ज़ीक्यूशन के बाद, आप टार्गेट फ़ोल्डर में `ImageWithTextLayer.psd` पाएँगे, जिसमें नया टेक्स्ट लेयर शामिल होगा।

## सामान्य समस्याएँ और समाधान
| समस्या | कारण | समाधान |
|-------|--------|-----|
| **`NullPointerException` on `im.addTextLayer`** | PSD सही तरीके से लोड नहीं हुआ (गलत पाथ). | सुनिश्चित करें `sourceFileName` मौजूदा PSD की ओर इशारा कर रहा है. |
| **टेक्स्ट दिखाई नहीं दे रहा** | रेक्टेंगल कैनवास के बाहर है या लेयर छिपी हुई है. | रेक्टेंगल कॉर्डिनेट्स समायोजित करें या `layer.setVisible(true)` से लेयर विज़िबिलिटी चेक करें. |
| **LicenseException** | प्रोडक्शन में वैध लाइसेंस के बिना लाइब्रेरी का उपयोग. | एक कमर्शियल लाइसेंस प्राप्त करें और `License license = new License(); license.setLicense("Aspose.PSD.lic");` के माध्यम से सेट करें. |

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न: क्या मैं कई टेक्स्ट लेयर्स जोड़ सकता हूँ?**  
**उत्तर:** हाँ – प्रत्येक टेक्स्ट के लिए चरण 4 और 5 दोहराएँ।

**प्रश्न: मैं टेक्स्ट की स्टाइल (फ़ॉन्ट, साइज, रंग) कैसे सेट करूँ?**  
**उत्तर:** `TextLayer` क्लास का `getTextData()` मेथड आपको `Font`, `FontSize`, `Color` आदि प्रॉपर्टीज़ बदलने की अनुमति देता है। पूर्ण विवरण के लिए Aspose.PSD API डॉक्यूमेंट देखें।

**प्रश्न: यदि मेरे PSD में पहले से कई लेयर्स हैं तो क्या होगा?**  
**उत्तर:** Aspose.PSD जटिल लेयर स्ट्रक्चर को संभालता है। आप विशिष्ट ग्रुप्स को टार्गेट कर सकते हैं या `addTextLayer` के ओवरलोड्स का उपयोग करके नई टेक्स्ट लेयर को इच्छित इंडेक्स पर डाल सकते हैं।

**प्रश्न: क्या यह तरीका वेब एप्लिकेशन के लिए उपयुक्त है?**  
**उत्तर:** बिल्कुल। जब तक आपका सर्वर जावा चलाता है, आप ऑन‑द‑फ़्लाई PSD जेनरेट या मॉडिफ़ाई कर सकते हैं और क्लाइंट को सर्व कर सकते हैं।

**प्रश्न: अगर समस्या आए तो मदद कहाँ मिल सकती है?**  
**उत्तर:** [Aspose सपोर्ट फ़ोरम](https://forum.aspose.com/c/psd/34) पर जाएँ जहाँ समुदाय और Aspose इंजीनियर्स सहायता प्रदान करते हैं।

## निष्कर्ष
आपने अब देखा कि जावा और Aspose.PSD का उपयोग करके रनटाइम में **add text to PSD** फ़ाइलें कितनी आसानी से बनाई जा सकती हैं। यह तकनीक ग्राफ़िक निर्माण को ऑटोमेट करने, एसेट्स को पर्सनलाइज़ करने, और फ़ोटोशॉप‑लेवल एडिटिंग को किसी भी जावा‑आधारित समाधान में इंटीग्रेट करने की शक्ति देती है। Aspose.PSD API के बाकी हिस्सों को एक्सप्लोर करें—शेप्स, रास्टर लेयर्स, या फ़िल्टर जोड़ें और ऑटोमेशन को और भी समृद्ध बनाएं।
