# PSD को PNG में कैसे बदलें और Aspose.PSD for Java के साथ अनुपातिक रूप से आकार बदलें

## परिचय

यदि आपको **PSD को PNG में बदलने** की आवश्यकता है जबकि मूल अनुपात बनाए रखें, तो आप सही जगह पर आए हैं। इस ट्यूटोरियल में हम एक पूर्ण **Aspose.PSD ट्यूटोरियल** के माध्यम से दिखाएंगे कि **कैसे इमेज का आकार अनुपातिक रूप से बदलें** जावा में, इमेज को कैश करके बेहतर प्रदर्शन प्राप्त करें, और अंत में परिणाम को PNG के रूप में सहेजें। अंत तक आप इस वर्कफ़्लो को किसी भी जावा‑आधारित इमेज‑प्रोसेसिंग पाइपलाइन में एकीकृत कर सकेंगे।

## त्वरित उत्तर
- **क्या Aspose.PSD PSD को PNG में बदल सकता है?** हाँ – बस PSD को लोड करें और `PngOptions` के साथ सहेजें।
- **क्या लाइब्रेरी अनुपातिक आकार बदलने का समर्थन करती है?** बिल्कुल; `resizeWidthProportionally` और `resizeHeightProportionally` का उपयोग करें।
- **क्या पहले इमेज को कैश करना आवश्यक है?** कैशिंग (`image.cacheData()`) बड़े PSD फ़ाइलों के लिए प्रदर्शन सुधारती है।
- **कौन सा जावा संस्करण आवश्यक है?** जावा 8 या नया पूरी तरह समर्थित है।
- **क्या उत्पादन के लिए लाइसेंस आवश्यक है?** हाँ, एक व्यावसायिक लाइसेंस मूल्यांकन वॉटरमार्क को हटाता है।

## “PSD को PNG में बदलना” क्या है और यह क्यों महत्वपूर्ण है?

फ़ोटोशॉप दस्तावेज़ (PSD) को पोर्टेबल नेटवर्क ग्राफ़िक्स (PNG) फ़ाइल में बदलने से आप लेयरड डिज़ाइन को हल्के, वेब‑फ़्रेंडली फ़ॉर्मेट में साझा कर सकते हैं। PNG पारदर्शिता और लॉसलेस क्वालिटी को बनाए रखता है, जिससे यह UI एसेट्स, थंबनेल या किसी भी स्थिति में आदर्श है जहाँ इमेज की सटीकता महत्वपूर्ण होती है।

## जावा में इमेज को अनुपातिक रूप से आकार क्यों बदलें?

**इमेज का आकार अनुपातिक रूप से बदलना** बिना उसके अनुपात को विकृत किए ग्राफ़िक्स को विभिन्न स्क्रीन पर प्राकृतिक दिखाता है। `Aspose.PSD` API अनुपातिक स्केलिंग के लिए समर्पित मेथड्स प्रदान करती है, जो स्वचालित रूप से मिलते‑जुलते ऊँचाई या चौड़ाई की गणना करती है, जिससे मैन्युअल गणना त्रुटियों से बचा जा सके।

## पूर्वापेक्षाएँ

कोड में प्रवेश करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. **जावा डेवलपमेंट किट (JDK)** – जावा 8 या बाद का स्थापित हो।
2. **Aspose.PSD for Java** – आधिकारिक साइट [से यहाँ](https://releases.aspose.com/psd/java/) नवीनतम JAR डाउनलोड करें।
3. एक नमूना PSD फ़ाइल (`sample.psd`) जिसे आप अपने प्रोजेक्ट से संदर्भित कर सकें।

## पैकेज इम्पोर्ट करें

अपने जावा क्लास में आवश्यक इम्पोर्ट जोड़ें। ये क्लासेज़ इमेज लोडिंग, कैशिंग, आकार बदलने और PNG आउटपुट विकल्पों तक पहुंच प्रदान करती हैं।

```java
import com.aspose.psd.Image;
import com.aspose.psd.imageoptions.PngOptions;
```

## चरण 1: PSD इमेज लोड करें

पहले स्रोत PSD फ़ाइल को लोड करें। यदि फ़ाइल बड़ी है, तो इमेज डेटा को कैश करने से बाद के ऑपरेशन्स के दौरान मेमोरी उपयोग कम होता है।

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";

Image image = Image.load(sourceFile);

if (!image.isCached()) {
    // Image caching java – improves performance for large files
    image.cacheData();
}
```

## चरण 2: इमेज को अनुपातिक रूप से आकार बदलें (Java)

अब नई डाइमेंशन तय करें। इस उदाहरण में हम इमेज को उसके मूल आकार के **आधे** तक घटाते हैं जबकि अनुपात बनाए रखते हैं। आप आवश्यकतानुसार लक्ष्य चौड़ाई या ऊँचाई को समायोजित कर सकते हैं।

```java
int newWidth = image.getWidth() / 2;
image.resizeWidthProportionally(newWidth);

int newHeight = image.getHeight() / 2;
image.resizeHeightProportionally(newHeight);
```

> **प्रो टिप:** `resizeWidthProportionally` *या* `resizeHeightProportionally` में से किसी एक का उपयोग करें—लाइब्रेरी स्वचालित रूप से अन्य डाइमेंशन की गणना करती है ताकि इमेज अनुपातिक रहे।

## चरण 3: आकार बदली इमेज को PNG के रूप में सहेजें

अंत में, आकार बदली इमेज को PNG फ़ॉर्मेट में एक्सपोर्ट करें। यह चरण **PSD को PNG में बदलने** वर्कफ़्लो को पूर्ण करता है।

```java
String destName = dataDir + "SimpleResizeImageProportionally_out.png";
image.save(destName, new PngOptions());
```

जब कोड चलाया जाएगा, तो आपको एक PNG फ़ाइल मिलेगी जो मूल PSD की आधी आकार की होगी, समान दृश्य गुणवत्ता के साथ और बिना किसी विकृति के।

## सामान्य समस्याएँ और समाधान

| समस्या | कारण | समाधान |
|-------|-------|-----|
| **OutOfMemoryError** | बड़ा PSD कैश नहीं किया गया | आकार बदलने से पहले `image.cacheData()` कॉल करें |
| **Blank PNG output** | फ़ाइल पथ गलत | `dataDir` और लिखने की अनुमतियों की जाँच करें |
| **Aspect ratio looks off** | दोनों चौड़ाई और ऊँचाई मैन्युअल रूप से सेट की गई | Aspose.PSD द्वारा प्रदान किए गए अनुपातिक मेथड्स का उपयोग करें |
