# PSD को फ़ॉन्ट्स के साथ सहेजें और फ़ॉन्ट कैश को फ़ोर्स करें – Aspose.PSD Java

## परिचय

यदि आपको **फ़ॉन्ट्स के साथ PSD सहेजना** जल्दी से करना है और साथ ही यह सुनिश्चित करना है कि सही टाइपफ़ेस उपलब्ध हों, तो आप सही जगह पर हैं। फ़ॉन्ट कैशिंग इमेज प्रदर्शन को **बहुत बेहतर** बना सकती है, विशेष रूप से जब आप बड़े फ़ोटोशॉप दस्तावेज़ों को बार‑बार प्रोसेस करते हैं। इस ट्यूटोरियल में हम फ़ॉन्ट कैश को फ़ोर्स करने, **PSD इमेज** फ़ाइलों को **लोड** करने, और अंत में Aspose.PSD for Java का उपयोग करके **फ़ॉन्ट्स के साथ PSD सहेजने** के सटीक चरणों को देखेंगे।

## त्वरित उत्तर
- **फ़ॉन्ट कैश को फ़ोर्स करने से क्या होता है?** यह Aspose.PSD को सिस्टम फ़ॉन्ट्स को पुनः‑स्कैन करने के लिए मजबूर करता है ताकि नए स्थापित फ़ॉन्ट्स तुरंत पहचाने जाएँ।  
- **फ़ॉन्ट इंस्टॉल होने के लिए मुझे कितना इंतज़ार करना चाहिए?** नमूना कोड **2 मिनट** के लिए रुकता है, जो आमतौर पर मैन्युअल इंस्टॉलेशन के लिए पर्याप्त होता है।  
- **क्या मैं इसे किसी भी PSD फ़ाइल के साथ उपयोग कर सकता हूँ?** हाँ – जब तक फ़ाइल सुलभ है और आवश्यक फ़ॉन्ट्स सिस्टम पर स्थापित हैं।  
- **क्या इस कोड को चलाने के लिए लाइसेंस चाहिए?** प्रोडक्शन उपयोग के लिए एक अस्थायी या पूर्ण लाइसेंस आवश्यक है; मूल्यांकन के लिए एक फ्री ट्रायल काम करता है।  
- **कौन से Java संस्करण समर्थित हैं?** Aspose.PSD for Java JDK 8 और उसके बाद के संस्करणों के साथ काम करता है।

## “फ़ॉन्ट्स के साथ PSD सहेजना” क्या है और यह क्यों महत्वपूर्ण है?

लेयर्स, टेक्स्ट या फ़ॉन्ट्स में बदलाव करने के बाद PSD फ़ाइल को सहेजना वह अंतिम कदम है जो आपके परिवर्तन को स्थायी बनाता है। जब फ़ॉन्ट कैश अद्यतन नहीं होता, तो सहेजी गई फ़ाइल डिफ़ॉल्ट फ़ॉन्ट्स पर वापस आ सकती है, जिससे दृश्य असंगतियाँ उत्पन्न होती हैं। पहले कैश को फ़ोर्स करके, आप सुनिश्चित करते हैं कि **फ़ॉन्ट्स के साथ PSD सहेजना** ऑपरेशन सही टाइपफ़ेस को एम्बेड करे।

## Aspose.PSD के साथ फ़ॉन्ट कैश को फ़ोर्स क्यों करें?

- **प्रदर्शन वृद्धि** – दोहराए गए फ़ॉन्ट लुक‑अप की आवश्यकता कम होती है।  
- **विश्वसनीयता** – सुनिश्चित करता है कि सहेजी गई PSD फ़ाइलें किसी भी मशीन पर ठीक उसी तरह रेंडर हों जैसा आप चाहते हैं।  
- **सरलता** – एक ही मेथड कॉल (`OpenTypeFontsCache.updateCache()`) भारी काम संभालता है।

## पैकेज इम्पोर्ट करें

पहले, PSD इमेज और फ़ॉन्ट कैश के साथ काम करने के लिए आवश्यक क्लासेस को इम्पोर्ट करें। ये स्टेटमेंट्स अपने Java क्लास के शीर्ष पर रखें:

```java
import com.aspose.psd.Image;
import com.aspose.psd.OpenTypeFontsCache;

import com.aspose.psd.fileformats.psd.PsdImage;
import java.io.Console;

```

## चरण‑दर‑चरण गाइड

### चरण 1: PSD इमेज लोड करें (How to load PSD image)

हम मूल PSD फ़ाइल को लोड करके शुरू करते हैं। यह हमें एक बेसलाइन इमेज ऑब्जेक्ट देता है जिसे हम बाद में फ़ॉन्ट कैश अपडेट होने के बाद फिर से सहेज सकते हैं।

```java
String dataDir = "Your Document Directory";

PsdImage image = (PsdImage)Image.load(dataDir + "sample.psd");
image.save(dataDir + "NoFont.psd");
```

- **व्याख्या:**  
  - `Image.load` फ़ाइल को मेमोरी में पढ़ता है।  
  - `image.save` **NoFont.psd** नाम की एक कॉपी बनाता है जो किसी भी नए फ़ॉन्ट उपलब्ध होने से **पहले** की स्थिति को दर्शाता है।  
  - यह चरण तुलना के लिए उपयोगी है और सुनिश्चित करता है कि बाद में कैश अपडेट वास्तव में आउटपुट को बदलता है।

### चरण 2: फ़ॉन्ट इंस्टॉलेशन के लिए प्रतीक्षा करें (java thread sleep – improve image performance)

अब हम उपयोगकर्ता को आवश्यक फ़ॉन्ट मैन्युअली इंस्टॉल करने के लिए समय देते हैं। वास्तविक दुनिया में आप इस चरण को ऑटोमेट कर सकते हैं, लेकिन यह रुकावट कैश‑रिफ्रेश मैकेनिज़्म को दर्शाती है।

```java
System.out.println("You have 2 minutes to install the font");
Thread.sleep(2 * 60 * 1000);
OpenTypeFontsCache.updateCache();
```

- **व्याख्या:**  
  - `Thread.sleep` (एक **java thread sleep**) निष्पादन को **2 मिनट** (2 × 60 × 1000 ms) के लिए रोकता है।  
  - `OpenTypeFontsCache.updateCache()` Aspose.PSD को सिस्टम के OpenType फ़ॉन्ट्स को पुनः‑स्कैन करने के लिए मजबूर करता है, जिससे कोई भी नया फ़ॉन्ट तुरंत रेंडरिंग के लिए उपलब्ध हो जाता है।  
  - यह रुकावट **इमेज प्रदर्शन को बेहतर** बनाती है क्योंकि अगली लोड से पहले कैश नवीनतम फ़ॉन्ट्स को दर्शाता है।

### चरण 3: अपडेटेड PSD इमेज लोड करें और **फ़ॉन्ट्स के साथ PSD सहेजें**

कैश रिफ्रेश के बाद, हम मूल PSD को फिर से लोड करते हैं। इस बार फ़ॉन्ट जानकारी अद्यतन होती है, और हम सही ढंग से **फ़ॉन्ट्स के साथ PSD सहेज** सकते हैं।

```java
PsdImage image1 = (PsdImage)Image.load(dataDir + "sample.psd");
image1.save(dataDir + "HasFont.psd");
```

- **व्याख्या:**  
  - दूसरा लोड नया स्थापित फ़ॉन्ट पकड़ लेता है।  
  - **HasFont.psd** में सहेजने से फ़ाइल में अब सही फ़ॉन्ट रेंडरिंग शामिल होती है, जिससे यह पुष्टि होती है कि कैश अपडेट सफल रहा।
