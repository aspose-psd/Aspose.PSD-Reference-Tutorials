## 为什么在高级图像处理中使用 Aspose.PSD？

- **Cross‑platform** – 在 .NET 和 Java 项目中使用相同的 API。  
- **No Photoshop installation** – 可在服务器、CI 流水线和容器中运行。

## 使用 Aspose.PSD 的高级图像处理技术
以下是针对开发者常问的最常见 **高级图像处理** 任务的快速实用指南。

### 如何将 PSD 转换为 PNG
Aspose.PSD 允许您加载 PSD 文件并直接保存为 PNG，同时保留透明度和图层效果。这对于面向网页的资源非常理想。

### 如何将 PSD 保存为 JPEG
当您需要压缩的光栅图像时，可以将 PSD 导出为 JPEG。您还可以控制质量设置，以在文件大小和视觉保真度之间取得平衡。

### 如何调整图像亮度
API 提供图像调整类，允许您在不展平文件的情况下，对单个图层或整幅图像进行亮度、对比度和伽马的修改。

### 如何添加图层效果
您可以通过编程方式添加投影、斜面、发光等 Photoshop 图层效果。这使得在运行时实现动态的视觉增强成为可能。

### 如何将 PSD 转换为 PDF
将 PSD 导出为 PDF 对于打印就绪的文档或归档非常有用。转换过程在可能的情况下保留矢量文本和图层信息。

## 常见问题

**Q: 我可以在商业应用中使用 Aspose.PSD 吗？**  
A: 可以。生产环境需要商业许可证，但提供免费试用供评估。

**Q: Aspose.PSD 支持受密码保护的 PSD 文件吗？**  
A: 支持。您可以在加载选项中提供密码来打开受保护的文件。

**Q: 哪些 .NET 版本兼容？**  
A: 完全支持 .NET Framework 4.5 及以上、.NET Core 3.1 及以上、.NET 5 和 .NET 6。

**Q: 我如何高效处理大型 PSD 文件？**  
A: 使用 `LoadOptions` 启用延迟加载，并单独处理图层，以降低内存消耗。

**Q: 能否批量将多个 PSD 文件转换为 PDF？**  
A: 完全可以。遍历目录，加载每个 PSD，并使用 PDF 格式调用 `Save` —— API 在一次处理过程中完成转换。
