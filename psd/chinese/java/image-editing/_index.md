## 常见问题

**Q: 我可以合并不同尺寸的图像吗？**  
A: 可以。Aspose.PSD 会根据您定义的画布大小自动调整或填充图层。

**Q: 合并时可以添加颜色叠加吗？**  
A: 当然。可以在合并前对每个图层应用颜色叠加效果，或在最终合成图像上应用。

**Q: 合并后如何保留 EXIF 或 XMP 元数据？**  
A: 在保存合并文件之前，使用 API 创建或复制 XMP 元数据。

**Q: Aspose.PSD 是否支持除 PSD 之外的图像格式合并？**  
A: 支持。您可以加载 JPEG、PNG、BMP 等格式，并将它们合并为 PSD，或导出为其他光栅格式。

**Q: 对于大型图像集合，有哪些性能考虑？**  
A: 使用流式 API 并及时释放中间对象，以保持低内存占用。
