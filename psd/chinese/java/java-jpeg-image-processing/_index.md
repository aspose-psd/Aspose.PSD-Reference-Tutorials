## 常见问题

**问：我可以在不影响原始图像质量的情况下为 JPEG 添加 EXIF 缩略图吗？**  
答：可以。Aspose.PSD 只将缩略图写入元数据段，主图像数据保持不变。

**问：如何使用 Aspose.PSD 在 Java 中写入 EXIF 数据？**  
答：使用 `PsdImage` 类访问 `ExifData` 属性，创建或修改标签，然后调用 `Save()`。

**问：批量处理时，读取 EXIF 标签的最佳方式是什么？**  
答：遍历文件时使用 `PsdImage`，读取 `ExifData.Tags`，并将所需值存入集合以供后续分析。

**问：是否可以在不损坏文件的情况下修改 JPEG EXIF 标签？**  
答：完全可以。API 仅更新 EXIF 段，保留 JPEG 其余结构不变。

**问：Aspose.PSD 是否支持写入自定义 EXIF 标签？**  
答：支持。可以通过 `ExifTag` 类指定标签 ID 和数值，添加非标准标签。
