## 什么是 Java 中的灰度图像？  

灰度图像每像素存储一个字节，表示亮度，范围从 0（黑）到 255（白）。将全彩 PSD 转换为灰度可让您专注于亮度，这对于艺术效果、计算机视觉算法的预处理或创建轻量化资源都很有用。

## 为什么使用 Aspose.PSD 优化 Java 中的 PNG？  

PNG 是无损的，但未优化的文件可能会过大。使用 Aspose.PSD 您可以：

1. 在可能的情况下降低位深度。  
2. 应用适合图像内容的压缩过滤器。  
3. 剥离不必要的元数据。  

这些步骤在保持视觉质量的同时缩小文件尺寸——非常适合网页和移动端部署。

## 常见问题

**Q: 如何在 Java 中将 PSD 文件转换为 TIFF？**  
A: 使用 `PsdImage` 加载 PSD 并调用 `save("output.tiff", new TiffOptions())`。API 会自动处理颜色配置文件和图层。

**Q: 在转换为 TIFF 之前，我可以使用 Java 绘制形状吗？**  
A: 是的。使用 Aspose.PSD 提供的 `Graphics` 对象绘制线条、矩形或自定义路径，然后将结果保存为 TIFF。

**Q: 在转换之前，是否可以在 PSD 中替换字体？**  
A: 当然可以。该库允许您用系统字体替换缺失的字体，确保文本在最终的 TIFF 中正确渲染。

**Q: 如何使用 Aspose.PSD 将 AI 文件转换为图像？**  
A: 虽然 Aspose.PSD 主要针对 PSD，但您可以先使用 Aspose.Slides 或其他工具将 AI 转换为 PDF，然后将 PDF 页面加载为图像进行进一步处理。

**Q: 在 Java 中优化 PNG 文件的步骤是什么？**  
A: 降低位深度、应用适当的压缩过滤器并剥离元数据——这些都可以通过 Aspose.PSD 的 PNG 选项实现。
