## 常见问题

**Q: 我需要付费许可证才能使用 Aspose.PSD for Java 吗？**  
**A:** 免费试用可用于开发和测试，但生产环境需要商业许可证。

**Q: 我可以在同一图层上应用多个效果，例如内部阴影和外发光吗？**  
**A:** 可以。通过将效果依次添加到图层的效果集合中，可实现堆叠。

**Q: 是否可以编辑 PSD 文件而不丢失图层信息？**  
**A:** 绝对可以。Aspose.PSD 在保存更改时会保留所有图层元数据、蒙版和调整图层。

**Q: 最新的 Aspose.PSD 库支持哪些 Java 版本？**  
**A:** 完全支持 Java 8、11 以及更新的 LTS 版本。

**Q: 如何处理大型 PSD 文件以避免内存问题？**  
**A:** 使用 `PsdImage` 类的 `Load` 重载并传入 `FileStream`，并在适当情况下启用惰性加载。
