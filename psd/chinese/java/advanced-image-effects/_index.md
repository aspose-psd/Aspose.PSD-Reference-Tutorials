## 快速答案
- **主要目的是什么？** To demonstrate how to add gradient, layer, and pattern effects to Java images with Aspose.PSD.  
- **需要哪个库？** Aspose.PSD for Java (latest version).

## 什么是 psd image processing？
psd image processing 是指使用代码以编程方式创建、修改和渲染 Photoshop (.psd) 文件。使用 Aspose.PSD，您可以操作图层、应用填充、添加渐变、图案和描边——全部通过纯 Java API 实现。

## 为什么使用 Aspose.PSD for Java 添加渐变、图层和图案效果？
- **完整的 PSD 兼容性** – works with all Photoshop features without needing Photoshop itself.  
- **纯 Java API** – no native code, easy integration into server‑side or desktop apps.  
- **丰富的效果库** – supports gradients, patterns, strokes, signatures, and more.  
- **性能优化** – handles large files efficiently.

## 前置条件

- 已设置 Maven 或 Gradle 项目。  
- Aspose.PSD for Java JAR（从 Aspose 网站下载或通过 Maven Central 添加）。
