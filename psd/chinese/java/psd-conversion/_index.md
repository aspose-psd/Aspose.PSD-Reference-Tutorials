## 常见问题

**Q: 如何使用 Aspose.PSD for Java 将 PSD 转换为 TIFF？**  
A: 使用 `PsdImage` 类加载 PSD 文件，并使用 `TiffOptions` 调用 `save` 方法。教程 “Convert CMYK PSD to CMYK TIFF” 展示了完整代码。

**Q: 我可以以多线程方式导出图像吗？**  
A: 可以。“Export Images in Multi‑Threaded Environment” 指南演示了如何在独立线程中安全运行转换。

**Q: 如何将 GIF 图像层转换为 TIFF？**  
A: 使用 `Image.load` 加载 GIF，遍历其帧，并使用 `TiffOptions` 将每帧保存为 TIFF 页面。参见 “Convert GIF Image Layers to TIFF” 教程。

**Q: Aspose.PSD 是否支持 Java 图像处理的颜色管理？**  
A: 当然。您可以使用默认配置文件或提供自定义 ICC 配置文件，详见 “Color Conversion using Default Profiles” 与 “Color Conversion using ICC Profiles” 部分。

**Q: 我在哪里可以找到更多关于导出图像的示例？**  
A: “Export Images in Multi‑Threaded Environment” 页面提供了高效导出图像的详细示例。
