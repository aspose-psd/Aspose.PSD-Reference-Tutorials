## 常见问题

**Q: 我可以在商业 Java 应用中使用 Aspose.PSD 吗？**  
A: 可以，需使用有效的 Aspose 许可证。提供免费试用供评估。

**Q: 该库是否支持带嵌入字体的 AI 文件？**  
A: 当然。Aspose.PSD 能读取嵌入的字体，并在可能的情况下保留文本渲染。

**Q: 如果需要转换大量 AI 文件怎么办？**  
A: 使用循环加载每个文件并调用 `save` 方法。该库已针对批处理进行优化。

**Q: AI 文件有大小限制吗？**  
A: 该库可处理高达数百兆字节的文件，唯一限制是 JVM 堆大小。

**Q: 将 AI 转换为 PNG 时如何保留透明度？**  
A: 确保使用 `PngOptions` 并将 `ColorType = PngColorType.TrueColorWithAlpha`。
