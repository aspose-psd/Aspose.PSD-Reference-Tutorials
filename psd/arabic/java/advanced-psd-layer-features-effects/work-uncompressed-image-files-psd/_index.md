# العمل مع ملفات الصور غير المضغوطة في PSD باستخدام Java

## المقدمة
عند التعامل مع مستندات الفوتوشوب (PSD) في Java، تجعل مكتبة **aspose psd java** من الضروري فهم كيفية تعديل وحفظ هذه الملفات الغنية بالصور بفعالية. في هذا الدليل، سنغوص عميقًا في استخدام Aspose.PSD، وهو API قوي يبسط إدارة ملفات PSD، بما في ذلك العمل مع الصور غير المضغوطة. سواء كنت مطورًا يرغب في تعزيز تطبيقك بالرسومات المتقدمة أو تريد فقط معالجة ملفات PSD في Java دون عناء، سيوجهك هذا الدليل خلال كل خطوة. هل أنت مستعد للبدء؟ هيا بنا!

## إجابات سريعة
- **كيف يمكنني تصدير ملف PSD بدون ضغط؟** استخدم `CompressionMethod.Raw` في `PsdOptions` عند حفظ الملف.  
- **أي مكتبة تدعم معالجة PSD غير المضغوط في Java؟** مكتبة **aspose psd java** توفر الدعم الكامل.  
- **هل أحتاج إلى ترخيص للاستخدام في الإنتاج؟** نعم، يلزم الحصول على ترخيص تجاري للنشر في بيئات الإنتاج.  
- **ما نسخة Java المطلوبة؟** JDK 8 أو أعلى تكفي.  
- **هل يمكنني تعديل الطبقات بعد الحفظ؟** بالتأكيد – أعد تحميل الملف المحفوظ واستخدم كائن `Graphics` للرسم أو تعديل الطبقات.

## تصدير PSD بدون ضغط باستخدام aspose psd java
حفظ ملف PSD بدون أي ضغط يحافظ على بيانات البكسل الأصلية، وهو أمر أساسي لسلاسل تحرير عالية الدقة، أو سير عمل أرشيفي، أو عندما تحتاج إلى نتائج بكسل‑مثالية للمعالجة اللاحقة. الخطوات أدناه توضح بالضبط كيفية تحقيق ذلك باستخدام API **aspose psd java**.

## استيراد الحزم
لبدء العمل، نحتاج إلى استيراد الحزم الضرورية لكودنا. إليك قائمة الاستيرادات التي ستحتاجها:
```java
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.CompressionMethod;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PsdOptions;
```
هذه الاستيرادات ستجلب الفئات والطرق اللازمة إلى مشروعنا، مما يتيح لنا تعديل ملفات PSD بسلاسة. لنقسم العملية إلى خطوات قابلة للإدارة.

## الخطوة 1: إعداد مسار الدليل الخاص بالملف
أولاً، عليك تحديد مكان وجود ملف PSD الخاص بك ومكان حفظ الناتج. في مثالنا، سننشئ متغيرًا يحمل مسار الدليل.
```java
String dataDir = "Your Document Directory";
```
استبدل `"Your Document Directory"` بالمسار الفعلي حيث يُخزن ملف PSD الخاص بك (`layers.psd`). بهذه الطريقة يضمن برنامجك معرفة مكان البحث عن الملف.

## الخطوة 2: تحميل ملف PSD
الآن، لنحمّل ملف PSD باستخدام طريقة `Image.load()`، مع تحويله إلى نوع `PsdImage`.
```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "layers.psd");
```
هذا السطر يستدعي طريقة `load` من فئة `Image`، محملاً ملف PSD إلى الذاكرة. بتحويله إلى `PsdImage`، نخبر جافا أن نتعامل مع هذا الملف كملف PSD يمتلك وظائف خاصة بعمليات PSD.

## الخطوة 3: تكوين خيارات الحفظ
بعد ذلك، نحتاج إلى إعداد الخيارات لحفظ ملفنا، مع تحديد أننا نريد الناتج غير مضغوط.
```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw);
```
فئة `PsdOptions` تسمح لنا بتحديد خيارات متعددة لحفظ ملف PSD. ضبط `setCompressionMethod` إلى `CompressionMethod.Raw` يضمن أن الملف المحفوظ غير مضغوط ويحافظ على جودة عالية.

## الخطوة 4: حفظ ملف PSD غير المضغوط
حان الوقت الآن لحفظ صورة PSD التي تم تكوينها حديثًا.
```java
psdImage.save(dataDir + "uncompressed_out.psd", saveOptions);
```
هذا السطر ينفّذ عملية الحفظ على كائن `PsdImage` الخاص بنا (`psdImage`). يحفظ الملف باسم `uncompressed_out.psd` في الدليل المحدد ويطبق الخيارات التي عُرّفت مسبقًا.

## الخطوة 5: إعادة فتح الصورة التي تم إنشاؤها حديثًا
بعد الحفظ، لنعد تحميل الصورة الناتجة للتحقق من أن كل شيء عمل كما هو متوقع.
```java
PsdImage img = (PsdImage) Image.load(dataDir + "uncompressed_out.psd");
```
من خلال استدعاء `load` مرة أخرى، يمكننا إنشاء نسخة جديدة من `PsdImage` تت对应 للملف المحفوظ. هذه الخطوة ضرورية إذا رغبت في تعديل أو عرض الصورة بعد حفظها.

## الخطوة 6: الرسم أو تعديل الصورة
أخيرًا، قد ترغب في الرسم على الصورة التي تم فتحها حديثًا أو تعديلها.
```java
Graphics graphics = new Graphics(img);
```
هنا نقوم بتهيئة كائن `Graphics`، والذي يتيح لنا تنفيذ عمليات رسومية مختلفة على `img`. يمكنك رسم أشكال، إضافة نص، أو حتى تعديل الطبقات إذا رغبت!

## المشكلات الشائعة والحلول
- **FileNotFoundException** – تحقق مرة أخرى من مسار `dataDir` وتأكد من أن اسم ملف PSD يطابق تمامًا.  
- **UnsupportedCompressionException** – تأكد من أنك تستخدم نسخة حديثة من Aspose.PSD تدعم `CompressionMethod.Raw`.  
- **License Not Found** – للاستخدام في الإنتاج، حمّل ترخيص Aspose قبل استدعاء أي من طرق API لتجنب علامات التقييم.
