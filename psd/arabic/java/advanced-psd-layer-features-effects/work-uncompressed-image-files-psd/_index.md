# تحويل PSD إلى RAW باستخدام Java (ملفات الصور غير المضغوطة)

## مقدمة
عند العمل مع مستندات Photoshop (PSD) في Java، فهم كيفية **convert PSD to RAW** وتصدير PSD بدون ضغط أمر أساسي للحفاظ على دقة الصورة. في هذا البرنامج التعليمي سنستكشف كيف يبسط Aspose.PSD عملية التعامل مع ملفات الصور غير المضغوطة، من تحميل PSD إلى حفظه كملف غير مضغوط بنمط RAW. سواء كنت تبني تطبيقًا كثيفًا في الرسومات أو تحتاج إلى تصدير صور بدون فقدان، ستجد كل ما تحتاجه هنا. هل أنت مستعد للغوص في الموضوع؟ لنبدأ!

## إجابات سريعة
- **ماذا يعني “convert PSD to RAW”؟** إنه يحفظ بيانات PSD دون أي ضغط، مع الحفاظ على بيانات البكسل في شكلها الأصلي.  
- **ما المكتبة التي تتعامل مع ذلك؟** توفر Aspose.PSD for Java واجهة برمجة تطبيقات بسيطة لحفظ الملفات غير المضغوطة.  
- **هل أحتاج إلى ترخيص؟** الإصدار التجريبي المجاني يعمل للاختبار؛ يلزم الحصول على ترخيص تجاري للإنتاج.  
- **ما نسخة Java المطلوبة؟** JDK 8 أو أعلى.  
- **هل يمكنني تعديل الملف بعد الحفظ؟** نعم – يمكنك إعادة تحميل PSD غير المضغوط ومتابعة الرسم أو إضافة الطبقات.  

## ما هو “convert PSD to RAW”؟
تحويل PSD إلى RAW يعني تصدير مستند Photoshop **بدون أي ضغط**. يحتفظ الملف الناتج ببيانات البكسل الكاملة، وهو مثالي للسيناريوهات التي لا يمكن فيها التضحية بجودة الصورة—مثل التخزين الأرشيفي، التصوير العلمي، أو خطوط إنتاج الطباعة عالية الجودة.

## لماذا تصدير PSD بدون ضغط؟
- **أقصى جودة:** لا فقدان للتفاصيل بسبب آثار الضغط.  
- **حجم ملف متوقع:** ملفات RAW لها حجم يعكس مباشرة أبعاد الصورة وعمق البت.  
- **معالجة لاحقة مبسطة:** يمكن للأدوات الأخرى قراءة بيانات البكسل دون الحاجة إلى فك الضغط أولاً.

## استيراد الحزم
لبدء العمل، نحتاج إلى استيراد الحزم الضرورية لشفرتنا. أدناه قائمة الاستيرادات التي ستحتاجها:

```java
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.fileformats.psd.CompressionMethod;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PsdOptions;
```

ستجلب هذه الاستيرادات الفئات والطرق الضرورية إلى مشروعنا، مما يتيح لنا التعامل مع ملفات PSD بسلاسة. دعنا نقسم العملية إلى خطوات قابلة للإدارة.

## الخطوة 1: إعداد دليل الملفات
أولاً، تحتاج إلى تحديد موقع ملف PSD الخاص بك ومكان حفظ الناتج. في مثال الشيفرة، سننشئ متغيرًا يحمل مسار الدليل.

```java
String dataDir = "Your Document Directory";
```

استبدل `"Your Document Directory"` بالمسار الفعلي حيث يتم تخزين ملف PSD الخاص بك (`layers.psd`). بهذه الطريقة، يضمن برنامجك معرفة مكان البحث عن الملف.

## الخطوة 2: تحميل ملف PSD
الآن، لنحمّل ملف PSD باستخدام طريقة `Image.load()`، مع تحويله إلى نوع `PsdImage`.

```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "layers.psd");
```

هذا السطر يستدعي طريقة `load` من فئة `Image`، محملاً ملف PSD إلى الذاكرة. بتحويله إلى `PsdImage`، نخبر Java بمعاملة هذه الصورة كملف PSD يحتوي على وظائف محددة تتعلق بعمليات PSD.

## الخطوة 3: تكوين خيارات الحفظ
بعد ذلك، نحتاج إلى إعداد الخيارات لحفظ ملفنا، مع تحديد أننا نريد أن يكون الناتج **غير مضغوط** (أي تحويل PSD إلى RAW).

```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw);
```

تتيح لنا فئة `PsdOptions` تحديد خيارات مختلفة لحفظ ملف PSD. ضبط `setCompressionMethod` إلى `CompressionMethod.Raw` يضمن أن يكون ملفنا المحفوظ غير مضغوط ويحافظ على جودة عالية.

## الخطوة 4: حفظ ملف PSD غير المضغوط
حان الوقت الآن لحفظ صورة PSD التي تم تكوينها حديثًا.

```java
psdImage.save(dataDir + "uncompressed_out.psd", saveOptions);
```

هذا السطر ينفذ وظيفة الحفظ على كائن `PsdImage` الخاص بنا (`psdImage`). يحفظ الملف باسم `uncompressed_out.psd` في الدليل المحدد ويطبق الخيارات التي تم تعريفها مسبقًا.

## الخطوة 5: إعادة فتح الصورة التي تم إنشاؤها حديثًا
بعد الحفظ، لنعد تحميل صورة الناتج للتحقق من أن كل شيء عمل كما هو متوقع.

```java
PsdImage img = (PsdImage) Image.load(dataDir + "uncompressed_out.psd");
```

من خلال استدعاء `load` مرة أخرى، يمكننا إنشاء نسخة جديدة من `PsdImage` تتطابق مع الملف المحفوظ. هذه الخطوة ضرورية إذا كنت ترغب في تعديل أو عرض الصورة بعد حفظها.

## الخطوة 6: الرسم أو تعديل الصورة
أخيرًا، قد ترغب في الرسم على الصورة التي تم فتحها حديثًا أو تعديلها.

```java
Graphics graphics = new Graphics(img);
```

هنا نقوم بتهيئة كائن `Graphics`، الذي يتيح لنا إجراء عمليات رسومية مختلفة على `img` الخاص بنا. يمكنك رسم أشكال، إضافة نص، أو حتى تعديل الطبقات إذا رغبت!

## حالات الاستخدام الشائعة
- **التخزين الأرشيفي:** الحفاظ على العمل الأصلي دون أي فقدان.

- **إنتاج الطباعة:** ضمان أعلى دقة قبل إرسال الملفات إلى الطابعة.
