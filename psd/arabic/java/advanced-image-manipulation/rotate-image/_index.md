# تدوير الصورة 270 درجة باستخدام Aspose.PSD للـ Java

## المقدمة

في هذا **دليل معالجة الصور بلغة Java**، ستكتشف كيفية **تدوير الصورة 270 درجة** بسرعة وموثوقية باستخدام Aspose.PSD للـ Java. سواءً كنت تبني أداة تحرير صور، أو تقوم بأتمتة تحويلات دفعة، أو تحتاج فقط إلى إعادة توجيه طبقة PSD، فإن المكتبة تجعل المهمة سهلة. سنتطرق أيضًا إلى عكس الصور وتحويل PSD المدور إلى JPEG، لتوفر لك سير عمل كامل من البداية إلى النهاية.

## إجابات سريعة
- **ما المكتبة التي تتعامل مع التدوير؟** Aspose.PSD للـ Java  
- **ما زاوية التدوير المستخدمة في المثال؟** 270 درجة  
- **هل يمكنني أيضًا عكس الصورة؟** نعم – استخدم خيارات `RotateFlipType` مثل `Rotate90FlipX`  
- **كيف أحفظ النتيجة؟** في المثال نحفظ كـ JPEG باستخدام `JpegOptions`  
- **هل أحتاج إلى ترخيص للإنتاج؟** يلزم وجود ترخيص صالح لـ Aspose.PSD للاستخدام التجاري  

## ما معنى “تدوير الصورة 270 درجة”؟
تدوير الصورة 270 درجة يعني تحويل الصورة ثلاثة أرباع دورة كاملة باتجاه عقارب الساعة (أو 90 درجة عكس اتجاه العقارب). في كثير من سيناريوهات تحرير الرسوم، تتطابق هذه الاتجاهية مع تخطيط البورتريه الأصلي بعد سلسلة من التحولات.

## لماذا نستخدم Aspose.PSD لهذه المهمة؟
- **دعم كامل لـ PSD** – يعمل مع الطبقات، الأقنعة، وكائنات التعديل.  
- **لا حاجة إلى Photoshop الأصلي** – يعمل على أي بيئة تشغيل Java.  
- **واجهة برمجة تطبيقات بسيطة** – استدعاء طريقة واحدة (`rotateFlip`) يتعامل مع التدوير والعكس.  
- **تحويل صيغ سهل** – تصدير مباشر إلى JPEG، PNG، أو صيغ شائعة أخرى.

## المتطلبات المسبقة

قبل البدء، تأكد من وجود ما يلي:

- مكتبة **Aspose.PSD للـ Java** مثبتة. يمكنك تنزيلها ومراجعة مرجع API الكامل [هنا](https://reference.aspose.com/psd/java/).  
- بيئة تطوير Java (JDK 8 أو أعلى).  
- ملف PSD تجريبي تريد تدويره. حدّث المتغير `sourceFile` في الشيفرة بالمسار الصحيح لملفك.

## استيراد الحزم

ابدأ باستيراد الفئات الضرورية من حزمة Aspose.PSD:

```java
import com.aspose.psd.Image;
import com.aspose.psd.RotateFlipType;

import com.aspose.psd.imageoptions.JpegOptions;
```

## كيفية تدوير PSD – الخطوة 1: تحميل الصورة

أنشئ كائن `Image` يشير إلى ملف PSD المصدر الخاص بك:

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";
Image image = Image.load(sourceFile);
```

## كيفية تدوير PSD – الخطوة 2: تدوير الصورة 270 درجة

استخدم طريقة `rotateFlip` مع `RotateFlipType.Rotate270FlipNone` لتحقيق تدوير بزاوية 270 درجة دون أي عكس:

```java
image.rotateFlip(RotateFlipType.Rotate270FlipNone);
```

> **نصيحة محترف:** إذا كنت بحاجة أيضًا إلى عكس الصورة أفقيًا أو عموديًا، اختر `RotateFlipType` مختلفًا مثل `Rotate90FlipX` أو `Rotate180FlipY`.

## كيفية تدوير PSD – الخطوة 3: تحويل PSD إلى JPEG وحفظه

بعد التدوير، يمكنك **تحويل PSD إلى JPEG** (أو أي صيغة مدعومة أخرى) باستخدام فئة الخيارات المناسبة:

```java
String destName = dataDir + "RotatedImage_out.jpg";
image.save(destName, new JpegOptions());
```

الملف `RotatedImage_out.jpg` الآن يحتوي على محتوى PSD الأصلي مدورًا بزاوية 270 درجة ومُحفظًا كـ JPEG.

| المشكلة | الحل |
|-------|----------|
| **الصورة تظهر مقلوبة رأسًا على عقب** | تأكد من أنك استخدمت `Rotate270FlipNone`. لتدوير 90 درجة باتجاه عقارب الساعة استخدم `Rotate90FlipNone`. |
| **ملف الإخراج معطوب** | تأكد من وجود المجلد الوجهة وأن لديك صلاحيات الكتابة. |
| **استثناء الترخيص** | قم بتثبيت ترخيص Aspose.PSD مؤقت أو دائم قبل تحميل الصورة في بيئة الإنتاج. |

ج: نعم، يدعم Aspose.PSD صيغ PSD، JPEG، PNG، BMP، GIF، والعديد من صيغ الرسوم النقطية الأخرى.

**س: هل يمكنني تطبيق تدويرات مخصصة، وليس فقط العكس المحدد مسبقًا؟**  
ج: بالتأكيد! بينما يوفر `RotateFlipType` الزوايا الشائعة، يمكنك دمج عدة استدعاءات أو استخدام مصفوفات التحويل لتدويرات بأي زاوية.
