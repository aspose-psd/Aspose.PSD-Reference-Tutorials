# قص الصورة في جافا: توسيع وقص الصور باستخدام Aspose.PSD لجافا

## المقدمة

في هذا الدرس ستكتشف كيفية **crop image java** باستخدام مكتبة Aspose.PSD. سواء كنت بحاجة إلى توسيع القماش، قص الحواف غير المرغوب فيها، أو تحويل ملف PSD إلى JPEG، فإن الخطوات أدناه ستوجهك عبر عملية نظيفة وقابلة للتكرار. سنغطي المتطلبات المسبقة، عبارات الاستيراد، وكل خطوة برمجية مع شروحات واضحة حتى تتمكن من تطبيق التقنية في مشاريع العالم الحقيقي.

## إجابات سريعة
- **ما هي المكتبة التي تتعامل مع crop image java؟** Aspose.PSD لجافا.
- **هل أحتاج إلى ترخيص للتطوير؟** نسخة تجريبية مجانية تكفي للاختبار؛ الترخيص التجاري مطلوب للإنتاج.
- **هل يمكنني تحويل PSD إلى JPEG أثناء القص؟** نعم، باستخدام `JpegOptions` مع مستطيل القص.
- **هل Java 8 مدعومة؟** Aspose.PSD تدعم Java 8 والإصدارات الأحدث.
- **كم من الوقت تستغرق عملية التنفيذ؟** عادةً أقل من 10 دقائق لعملية قص أساسية.

## ما هو “crop image java”؟

قص الصورة في جافا يعني اختيار منطقة مستطيلة من الصورة الأصلية والتخلص من الباقي. باستخدام Aspose.PSD يمكنك تعريف هذه المنطقة باستخدام كائن `Rectangle`، ثم حفظ النتيجة بصيغة مختلفة مثل JPEG.

## لماذا تستخدم Aspose.PSD لقص الصور في جافا؟

- **دعم كامل لملفات PSD** – العمل مباشرةً مع ملفات PSD ذات الطبقات دون الحاجة إلى تحويلها أولاً.  
- **معالجة رسومية عالية الأداء** – استخدام فعال للذاكرة مع الصور الكبيرة.  
- **تحويل مدمج** – تصدير سهل إلى JPEG, PNG, BMP، إلخ، مع تطبيق القص أو توسيع القماش.  
- **متعدد المنصات** – يعمل على أي نظام يدعم جافا.

## المتطلبات المسبقة

قبل أن نبدأ، تأكد من وجود ما يلي:

1. **مجموعة تطوير جافا (JDK)** – Java 8 أو أحدث مثبتة.  
2. **Aspose.PSD لجافا** – قم بتحميل المكتبة من الموقع الرسمي **[هنا](https://releases.aspose.com/psd/java/)**.  

> **نصيحة احترافية:** أضف ملف JAR الخاص بـ Aspose.PSD إلى مسار الـ classpath في مشروعك أو إلى تبعيات Maven/Gradle لتجنب حدوث `ClassNotFoundException`.

## استيراد الحزم

أضف الاستيرادات المطلوبة إلى ملف المصدر Java الخاص بك. هذه الفئات تمنحك إمكانية تحميل الصور، معالجة البكسلات، تعريف المستطيلات، وخيارات تصدير JPEG.

```java
import com.aspose.psd.Image;
import com.aspose.psd.RasterImage;
import com.aspose.psd.Rectangle;

import com.aspose.psd.imageoptions.JpegOptions;
```

## الخطوة 1: تعيين دليل المستند الخاص بك

حدد المجلد الذي يحتوي على ملف PSD الأصلي. استبدل العنصر النائب بالمسار الفعلي على جهازك.

```java
String dataDir = "Your Document Directory";
```

## الخطوة 2: تحديد مسارات المصدر والوجهة

عرّف أين ستقرأ ملف PSD ومن أين ستكتب ملف JPEG المقصوص.

```java
String sourceFile = dataDir + "example1.psd";
String destName = dataDir + "jpeg_out.jpg";
```

## الخطوة 3: تحميل الصورة وتخزينها مؤقتًا

حمّل ملف PSD إلى كائن `RasterImage`. التخزين المؤقت يحسن الأداء للعمليات اللاحقة مثل القص.

```java
RasterImage rasterImage = (RasterImage)Image.load(sourceFile);
rasterImage.cacheData();
```

## الخطوة 4: إنشاء مستطيل للقص

أنشئ كائن `Rectangle` يصف المنطقة التي تريد الاحتفاظ بها. يمكن أن تكون الإحداثيات سلبية لـ **توسيع** القماش قبل القص، وهو مفيد لإضافة حد حول الصورة الأصلية.

```java
Rectangle destRect = new Rectangle(-200, -200, 300, 300);
```

> **لماذا نستخدم إحداثيات سلبية؟**  
> القيم السلبية لـ X/Y تحرك منطقة القص إلى اليسار/الأعلى، مما يضيف مساحة فارغة (توسيع) حول المحتوى الأصلي قبل القص النهائي.

## الخطوة 5: حفظ الصورة المقصوصة

أخيرًا، احفظ الصورة الناتجة باستخدام `JpegOptions`. تُظهر هذه الخطوة أيضًا **convert psd jpeg** أثناء تطبيق مستطيل القص.

```java
rasterImage.save(destName, new JpegOptions(), destRect);
```

> **النتيجة:** `jpeg_out.jpg` الآن يحتوي على صورة بحجم 300 × 300 بكسل تم توسيعها بمقدار 200 بكسل من كل جانب ثم قصها إلى المستطيل المحدد.

تهانينا! لقد نجحت في **java image cropping**، توسيع القماش، وتحويل ملف PSD إلى JPEG—كل ذلك في بضع أسطر من الشيفرة.

## حالات الاستخدام الشائعة

- **تحضير الأصول للويب** – قص وإعادة تحجيم لقطات الشاشة أو التصاميم قبل رفعها.  
- **إنشاء صور مصغرة** – استخراج منطقة محددة من PSD كبير لأغراض المعاينة.  
- **معالجة دفعة تلقائية** – تكرار عبر مجلد من ملفات PSD، وتطبيق نفس مستطيل القص على كل منها.

## استكشاف الأخطاء وإصلاحها والنصائح

| المشكلة | الحل المقترح |
|-------|----------------|
| `OutOfMemoryError` عند تحميل PSD كبير | استدعِ `rasterImage.cacheData()` مبكرًا وفكّر في زيادة حجم heap للـ JVM (`-Xmx`). |
| المنطقة المقصوصة غير متمركزة | تحقق من إزاحات X/Y للمستطيل؛ تذكّر أن القيم السلبية توسع القماش. |
| JPEG الناتج يبدو غير واضح | عدّل إعداد جودة `JpegOptions` (مثلاً `new JpegOptions { Quality = 90 }`). |
