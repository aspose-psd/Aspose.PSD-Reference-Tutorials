# تغيير دقة PNG في جافا باستخدام Aspose.PSD

## المقدمة
إذا كنت بحاجة إلى **تغيير دقة PNG في جافا** بسرعة وموثوقية، فأنت في المكان الصحيح. في هذا الدرس سنستعرض الخطوات الدقيقة المطلوبة لضبط دقة ملف PNG باستخدام Aspose.PSD للـ Java. سواءً كنت تبني أداة معالجة دفعية، أو خدمة ويب، أو مجرد تحسين بعض الأصول، فإن النهج نفسه يعمل في كل مكان. افتح بيئة التطوير المفضلة لديك، ولنبدأ!

## إجابات سريعة
- **ما المكتبة التي تتعامل مع دقة PNG؟** Aspose.PSD للـ Java  
- **الطريقة الأساسية؟** `PngOptions.setResolutionSettings`  
- **قيم DPI النموذجية؟** 72 × 96 للصور الجاهزة للويب  
- **هل أحتاج إلى ترخيص؟** النسخة التجريبية تعمل للاختبار؛ الترخيص مطلوب للإنتاج  
- **إصدار JDK المدعوم؟** Java 8 أو أعلى  

## ما هو “تغيير دقة PNG في جافا”؟
تغيير دقة PNG في جافا يعني تعديل بيانات الـ DPI (النقاط في البوصة) الوصفية التي تخبر المتصفحات والطابعات بحجم الصورة المطلوب عرضه. تبقى بيانات البكسل كما هي، لكن علامة الدقة تُحدَّث، مما قد يؤثر على حجم الطباعة والجودة.

## لماذا نستخدم Aspose.PSD لهذه المهمة؟
توفر Aspose.PSD واجهة برمجة تطبيقات عالية المستوى تُجردك من التعامل مع تفاصيل تنسيق PNG منخفضة المستوى، لتتمكن من التركيز على منطق الأعمال بدلاً من تفاصيل الملف. تدعم مئات ميزات PSD، تعمل على أي منصة تدعم Java، ولا تحتاج إلى مكتبات أصلية.

## المتطلبات المسبقة
قبل أن تبدأ، تأكد من وجود ما يلي:

1. **مجموعة تطوير جافا (JDK) 8+** – الشيفرة تعمل على أي JDK حديث.  
2. **Aspose.PSD للـ Java** – حمّله من [رابط التحميل](https://releases.aspose.com/psd/java/).  
3. **بيئة تطوير متكاملة** – IntelliJ IDEA، Eclipse، أو VS Code مع دعم Java.  
4. **ملف PSD تجريبي** – سنقوم بتحويله إلى PNG وتغيير دقته.  
5. **معرفة أساسية بـ Java** – ستفهم المقاطع البرمجية دون الحاجة إلى شرح إضافي.

## استيراد الحزم
الآن بعد أن تم إعداد كل شيء، استورد الفئات التي ستحتاجها. هذه الاستيرادات تمنحك الوصول إلى تحميل الصور، إعدادات الدقة، وخيارات تصدير PNG.

```java
import com.aspose.psd.Image;
import com.aspose.psd.ResolutionSetting;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.PngOptions;
```

## الخطوة 1: إعداد مشروع Java الخاص بك
أنشئ مشروع Java جديد (أو وحدة Maven/Gradle) وأضف ملف JAR الخاص بـ Aspose.PSD إلى مسار البناء. إذا كنت تستخدم Maven، أضف الاعتماد المناسب من مستودع Aspose.

## الخطوة 2: تعريف دليل المستندات الخاص بك
أخبر البرنامج بمكان وجود ملف PSD المصدر وأين يتم كتابة ملف PNG الناتج.

```java
String dataDir = "Your Document Directory"; // Update with your folder path
```

استبدل `"Your Document Directory"` بالمسار المطلق أو النسبي الذي يحتوي على `sample.psd`.

## الخطوة 3: تحميل صورة PSD
استخدم الفئة `PsdImage` لقراءة ملف PSD من القرص.

```java
PsdImage psdImage = (PsdImage) Image.load(dataDir + "sample.psd");
```

تأكد من أن اسم الملف يطابق ملف PSD الفعلي الذي تريد معالجته.

## الخطوة 4: إنشاء وتكوين خيارات PNG
هنا نُجري **تغيير دقة PNG في جافا** فعليًا. نقوم بإنشاء كائن `PngOptions` وتعيين قيم DPI الأفقية والعمودية عبر `ResolutionSetting`.

```java
PngOptions options = new PngOptions();
options.setResolutionSettings(new ResolutionSetting(72, 96)); // 72 DPI horizontal, 96 DPI vertical
```

يمكنك استبدال `72` و `96` بأي قيم تناسب جهازك المستهدف. هذا هو جوهر عملية **تعيين دقة الصورة في جافا**.

## الخطوة 5: حفظ PNG الناتج
أخيرًا، صَدِّر ملف PSD كـ PNG مع بيانات الدقة الجديدة.

```java
psdImage.save(dataDir + "SettingResolution_output.png", options);
```

سيظهر الملف `SettingResolution_output.png` في نفس المجلد، الآن يحمل قيم DPI التي حددتها.

## المشكلات الشائعة والنصائح
- **مسار غير صحيح** – تأكد من أن `dataDir` ينتهي بفاصل ملفات (`/` أو `\`).  
- **DPI غير مدعوم** – معظم المتصفحات تتجاهل قيم DPI فوق 300؛ حافظ على قيم معقولة.  
- **استهلاك الذاكرة** – ملفات PSD الكبيرة قد تستهلك ذاكرة RAM كبيرة؛ فكر في تحرير `psdImage` بعد الحفظ (`psdImage.dispose()`).

## الخاتمة
لقد تعلمت الآن كيفية **تغيير دقة PNG في جافا** باستخدام Aspose.PSD. عبر تعديل `ResolutionSetting`، يمكنك التحكم في كيفية تفسير PNGs من قبل الطابعات وأدوات التصميم دون تغيير بيانات البكسل. استكشف خيارات أخرى مثل مستوى الضغط، عمق اللون، أو المعالجة الدفعية لمزيد من الأتمتة في سير عملك.

للمزيد من الاستكشاف، راجع الوثائق الرسمية لـ [Aspose.PSD Documentation](https://reference.aspose.com/psd/java/).

## الأسئلة المتكررة
### ما الصيغ التي يمكنني تحويل ملفات PSD إليها باستخدام Aspose.PSD للـ Java؟
يمكنك تحويل ملفات PSD إلى صيغ مثل PNG، JPEG، BMP، و TIFF.  
### هل أحتاج إلى ترخيص لاستخدام Aspose.PSD للـ Java؟
نعم، بينما تتوفر نسخة تجريبية مجانية، يلزم وجود ترخيص صالح للاستخدام المستمر بعد التقييم.  
### هل يمكنني تغيير الدقة أكثر من مرة في برنامج واحد؟
بالطبع! يمكنك إنشاء كائنات `PngOptions` مختلفة لإعدادات تصدير متعددة داخل نفس التطبيق.  
### ماذا لو كان ملف PSD تالفًا؟
يتعامل Aspose.PSD مع العديد من المشكلات الشائعة، لكن إذا كان الملف تالفًا بشكل كبير قد لا يتم تحميله. احرص دائمًا على الاحتفاظ بنسخة احتياطية.  
### هل Aspose.PSD مناسب للتطبيقات عالية الأداء؟
نعم، صُممت للتعامل مع ملفات كبيرة بكفاءة وتناسب مهام معالجة الصور ذات المتطلبات العالية.

## أسئلة شائعة
**س: كيف يمكنني برمجيًا تعيين DPI مختلف للمحور الأفقي والرأسي؟**  
ج: استخدم `new ResolutionSetting(horizontalDpi, verticalDpi)` كما هو موضح في مثال خيارات PNG.  

**س: هل يمكنني معالجة عدة ملفات PSD دفعيًا في تشغيل واحد؟**  
ج: نعم—قم بلف خطوات التحميل، تكوين الخيارات، والحفظ داخل حلقة تت iterates على مجموعة ملفاتك.  

**س: هل يؤثر تغيير DPI على حجم الملف؟**  
ج: عادة لا؛ DPI هو بيانات وصفية. يتغير حجم الملف فقط إذا قمت بتعديل الضغط أو أبعاد البكسل.  

**س: هل هناك طريقة لقراءة DPI الحالي لملف PNG موجود؟**  
ج: حمّل PNG باستخدام `Image.load()` وتفقد `image.getResolutionSettings()` (متاح لملفات PNG).  

**س: ما إصدارات JDK المدعومة رسميًا؟**  
ج: تدعم Aspose.PSD للـ Java إصدارات JDK 8 حتى JDK 21.
