## Vanliga frågor

**Q:** Behöver jag en betald licens för att använda Aspose.PSD för Java?  
**A:** En gratis provversion finns tillgänglig för utveckling och testning, men en kommersiell licens krävs för produktionsdistributioner.

**Q:** Kan jag applicera flera effekter, såsom inre skugga och yttre glöd, på samma lager?  
**A:** Ja. Effekter kan staplas genom att lägga till dem sekventiellt i lagrets effektkollektion.

**Q:** Är det möjligt att redigera en PSD-fil utan att förlora lagerinformation?  
**A:** Absolut. Aspose.PSD bevarar all lagermetadata, masker och justeringslager när du sparar ändringarna.

**Q:** Vilka Java-versioner stöds av det senaste Aspose.PSD-biblioteket?  
**A:** Java 8, 11 och nyare LTS‑utgåvor stöds fullt ut.

**Q:** Hur hanterar jag stora PSD-filer för att undvika minnesproblem?  
**A:** Använd `PsdImage`‑klassens `Load`‑överladdning med en `FileStream` och aktivera lazy loading där det är lämpligt.
