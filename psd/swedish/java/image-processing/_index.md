## Vanliga frågor

**Q: Kan jag använda medianfilter på stora PSD‑filer utan att få slut på minne?**  
A: Ja. Aspose.PSD bearbetar lager sekventiellt och erbjuder streamingalternativ för att hålla minnesanvändningen låg.

**Q: Hur skiljer sig ett medianfilter från ett Gaussiskt filter?**  
A: Ett medianfilter är icke‑linjärt och är utmärkt för att ta bort salt‑och‑peppar‑brus samtidigt som det bevarar kanter, medan ett Gaussiskt filter är linjärt och jämnt mjukar.

**Q: Är det möjligt att kombinera **apply gaussian filter** och medianfiltrering i en enda pipeline?**  
A: Absolut. Du kan kedja filtren—applicera en Gaussisk oskärpa först för att minska högfrekvent brus, sedan ett medianfilter för kantbevarande rengöring.

**Q: Stöder Aspose.PSD **crop psd file**‑operationer på smarta objekt?**  
A: Ja. Beskärning fungerar på alla rasterlager, inklusive de som är inbäddade i smarta objekt.

**Q: Vilken Java‑version krävs för dessa handledningar?**  
A: Java 8 eller nyare krävs; biblioteket är fullt kompatibelt med Java 11, 17 och senare LTS‑utgåvor.
