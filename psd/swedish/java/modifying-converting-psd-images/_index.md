## Frequently Asked Questions

**Q: Kan jag lägga till en diagonal vattenstämpel i en PSD som redan innehåller justeringslager?**  
A: Ja. Aspose.PSD låter dig stapla en vattenstämpel ovanpå befintliga lager, inklusive nyans‑mättnad, kurvor och färgfyllnadslager, utan att platta till filen.

**Q: Är det möjligt att batch‑processa flera PSD‑filer för att lägga till samma vattenstämpel?**  
A: Absolut. Du kan loopa igenom en katalog, ladda varje PSD, applicera vattenstämpeln och spara resultatet – allt inom ett enda Java‑program.

**Q: Påverkar justeringslagerhandledningarna (nyans‑mättnad, kurvor, färgfyllnad) vattenstämpelns synlighet?**  
A: Justeringslager påverkar den underliggande bilddata men inte det rasteriserade vattenstämpellagret om du lägger till vattenstämpeln efter dessa justeringar. Du kan också placera vattenstämpeln på ett eget lager för full kontroll.

**Q: Vilka minnesaspekter bör jag tänka på när jag arbetar med stora PSD‑filer?**  
A: Använd cache‑omfördelningskontrollerna som beskrivs i handledningen “Control Cache Reallocation” för att begränsa RAM‑användning, särskilt vid batch‑bearbetning av högupplösta filer.

**Q: Vilka versioner av Aspose.PSD är kompatibla med Java 11 och nyare?**  
A: Alla senaste releaser (2023‑2025) stödjer Java 8+, inklusive Java 11, 17 och senare. Kontrollera produktens release‑noteringar för den exakta version du använder.
