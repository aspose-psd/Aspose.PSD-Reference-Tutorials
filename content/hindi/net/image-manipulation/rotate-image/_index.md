Aspose.PSD कार्यक्षमता तक पहुँचने के लिए आवश्यक नामस्थान शामिल करना सुनिश्चित करें। अपने .NET प्रोजेक्ट की शुरुआत में निम्न पंक्तियाँ जोड़ें:

```csharp
using Aspose.PSD.ImageOptions;
```

