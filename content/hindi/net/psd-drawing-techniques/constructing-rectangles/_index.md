## नामस्थान आयात करें

अपने .NET प्रोजेक्ट में आवश्यक नामस्थान आयात करके प्रारंभ करें। ये नामस्थान आयत बनाने के लिए आवश्यक Aspose.PSD कार्यक्षमता तक पहुंच प्रदान करते हैं।

```csharp
using Aspose.PSD.Brushes;

```

## चरण 1: दस्तावेज़ निर्देशिका प्रारंभ करें

अपनी दस्तावेज़ निर्देशिका के लिए पथ सेट करें जहां आउटपुट छवि सहेजी जाएगी।

```csharp
string dataDir = "Your Document Directory";
```

## चरण 2: आयत बनाना

अब, आइए Aspose.PSD का उपयोग करके आयत बनाने की प्रक्रिया के बारे में गहराई से जानें।

### चरण 2.1: BmpOptions का एक उदाहरण बनाएं

```csharp
string outpath = dataDir + "Rectangle.bmp";
BmpOptions saveOptions = new BmpOptions();
saveOptions.BitsPerPixel = 32;
```

### चरण 2.2: छवि का एक उदाहरण बनाएं

```csharp
using (Image image = new PsdImage(100, 100))
{
    // चरण 2.3: ग्राफ़िक्स क्लास प्रारंभ करें और ग्राफ़िक्स सतह साफ़ करें
    Graphics graphic = new Graphics(image);
    graphic.Clear(Color.Yellow);

    // चरण 2.4: आयत बनाएं
    graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
    graphic.DrawRectangle(new Pen(new SolidBrush(Color.Blue)), new Rectangle(10, 30, 80, 40));

    // चरण 2.5: छवि को बीएमपी फ़ाइल स्वरूप में निर्यात करें
    image.Save(outpath, saveOptions);
}
```
