## नामस्थान आयात करें

अपने .NET प्रोजेक्ट में, Aspose.PSD की कार्यक्षमताओं तक पहुँचने के लिए आवश्यक नामस्थान आयात करें। अपने कोड की शुरुआत में निम्नलिखित पंक्तियाँ जोड़ें:

```csharp
using Aspose.PSD.FileFormats.Psd;
using Aspose.PSD.ImageOptions;
```

अब जब आपने पूर्वापेक्षाएँ कवर कर ली हैं, तो आइए उदाहरण को कई चरणों में विभाजित करें।

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "Your Document Directory";
```

 सुनिश्चित करें कि आप प्रतिस्थापित करें`"Your Document Directory"` आपकी दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

## चरण 2: स्रोत और गंतव्य पथ निर्दिष्ट करें

```csharp
//एक्सस्टार्ट: सेविंगटूडिस्क

string sourceFile = dataDir + @"sample.psd";
string destName = dataDir + "result.png";
```

 यहाँ,`sourceFile` PSD फ़ाइल का पथ है जिसे आप संसाधित करना चाहते हैं, और`destName` परिणामी छवि के लिए गंतव्य पथ है।

## चरण 3: छवि को लोड करें और सहेजें

```csharp
// PSD छवि लोड करें और न मिले फ़ॉन्ट बदलें।
using (Image image = Image.Load(sourceFile))
{
    PsdImage psdImage = (PsdImage)image;
    psdImage.Save(destName, new PngOptions());
}
```

यह कोड स्निपेट PSD छवि को लोड करता है, इसे पीएनजी प्रारूप में परिवर्तित करता है, और इसे निर्दिष्ट गंतव्य पर सहेजता है।

 बधाई हो! आपने .NET के लिए Aspose.PSD का उपयोग करके एक छवि को डिस्क पर सफलतापूर्वक सहेज लिया है। यह ट्यूटोरियल प्रक्रिया की मूलभूत समझ प्रदान करता है, लेकिन व्यापक दस्तावेज़ीकरण में जानने के लिए और भी बहुत कुछ है[यहाँ](https://reference.aspose.com/psd/net/).
