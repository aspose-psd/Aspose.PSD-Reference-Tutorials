## परिचय

जावा के लिए Aspose.PSD का उपयोग करके आनुपातिक रूप से छवियों का आकार बदलने पर हमारी चरण-दर-चरण मार्गदर्शिका में आपका स्वागत है। यदि आप अपने जावा अनुप्रयोगों को कुशल छवि आकार बदलने की क्षमताओं के साथ बढ़ाना चाह रहे हैं, तो आप सही जगह पर आए हैं। इस ट्यूटोरियल में, हम आपको स्पष्ट और संक्षिप्त चरणों का उपयोग करके प्रक्रिया के बारे में बताएंगे।

## आवश्यक शर्तें

ट्यूटोरियल में जाने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:

1. जावा पर्यावरण: सुनिश्चित करें कि आपके सिस्टम पर जावा स्थापित है।
2.  Aspose.PSD लाइब्रेरी: Aspose.PSD लाइब्रेरी डाउनलोड और इंस्टॉल करें। आप आवश्यक संसाधन पा सकते हैं[यहाँ](https://releases.aspose.com/psd/java/).

## पैकेज आयात करें

एक बार जब आपकी आवश्यकताएं व्यवस्थित हो जाएं, तो आवश्यक पैकेजों को अपने जावा प्रोजेक्ट में आयात करें। सुनिश्चित करें कि Aspose.PSD लाइब्रेरी आपके प्रोजेक्ट में सही ढंग से संदर्भित है।

```java
import com.aspose.psd.Image;

import com.aspose.psd.imageoptions.PngOptions;
```

## चरण 1: छवि लोड करें

उस छवि को लोड करके शुरुआत करें जिसका आप अपने जावा एप्लिकेशन में आकार बदलना चाहते हैं।

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";

Image image = Image.load(sourceFile);

if (!image.isCached()) {
    image.cacheData();
}
```

## चरण 2: चौड़ाई और ऊंचाई निर्दिष्ट करें

अपनी संशोधित छवि के लिए नई चौड़ाई और ऊँचाई निर्धारित करें। इस उदाहरण में, हम छवि का आकार उसके मूल आयामों के आधे पर बदल देंगे।

```java
int newWidth = image.getWidth() / 2;
image.resizeWidthProportionally(newWidth);

int newHeight = image.getHeight() / 2;
image.resizeHeightProportionally(newHeight);
```

## चरण 3: संशोधित छवि सहेजें

प्रारूप विकल्प (इस मामले में पीएनजी) निर्दिष्ट करते हुए, परिवर्तित छवि को अपने इच्छित स्थान पर सहेजें।

```java
String destName = dataDir + "SimpleResizeImageProportionally_out.png";
image.save(destName, new PngOptions());
```
