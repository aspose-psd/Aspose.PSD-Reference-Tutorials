## पैकेज आयात करें

अपने जावा प्रोजेक्ट में आवश्यक पैकेज आयात करके शुरुआत करें। निम्नलिखित उदाहरण दर्शाता है कि यह कैसे करना है:

```java
import com.aspose.psd.Image;

import com.aspose.psd.imageoptions.GifOptions;
```

अब, आइए हानिपूर्ण GIF कंप्रेसर के कार्यान्वयन को कई चरणों में विभाजित करें।

## चरण 1: अपना प्रोजेक्ट सेट करें

एक नया जावा प्रोजेक्ट बनाएं और अपने प्रोजेक्ट की निर्भरता में जावा लाइब्रेरी के लिए Aspose.PSD जोड़ें।

## चरण 2: फ़ाइल पथ परिभाषित करें

अपने प्रोजेक्ट में स्रोत PSD फ़ाइल पथ और गंतव्य GIF फ़ाइल पथ निर्दिष्ट करें। यहीं पर संपीड़ित GIF सहेजी जाएगी।

```java
String dataDir = "Your Document Directory";
String sourceFile = dataDir + "sample.psd";
String destName = dataDir + "anim_lossy-200.gif";
```

## चरण 3: छवि लोड करें

 मौजूदा PSD छवि को के एक उदाहरण में लोड करें`RasterImage` क्लास का उपयोग कर रहा हूँ`Image.load` तरीका।

```java
Image image = Image.load(sourceFile);
```

## चरण 4: GIF संपीड़न कॉन्फ़िगर करें

 एक बनाने के`GifOptions` GIF संपीड़न के लिए पैरामीटर सेट करने के लिए ऑब्जेक्ट, जैसे अधिकतम अंतर। इस उदाहरण में, हमने अधिकतम अंतर 200 निर्धारित किया है।

```java
GifOptions gifExport = new GifOptions();
gifExport.setMaxDiff(200);
```

## चरण 5: संपीड़ित GIF सहेजें।

 का उपयोग करके संपीड़ित GIF छवि को सहेजें`image.save` निर्दिष्ट के साथ विधि`GifOptions`.

```java
image.save(destName, gifExport);
```
