## परिचय
इस ट्यूटोरियल में, आप सीखेंगे कि जावा के लिए Aspose.PSD का उपयोग करके दीर्घवृत्त कैसे बनाएं। Aspose.PSD एक शक्तिशाली लाइब्रेरी है जो जावा डेवलपर्स को PSD फ़ाइलों के साथ काम करने और छवियों में आसानी से हेरफेर करने की अनुमति देती है। छवि प्रसंस्करण और ग्राफ़िक डिज़ाइन में दीर्घवृत्त जैसी आकृतियाँ बनाना एक मौलिक कार्य है। इस गाइड का पालन करके, आप Aspose.PSD का उपयोग करके प्रोग्रामेटिक रूप से दीर्घवृत्त बनाने का व्यावहारिक अनुभव प्राप्त करेंगे।
## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:
- जावा प्रोग्रामिंग का बुनियादी ज्ञान।
- आपके सिस्टम पर JDK (जावा डेवलपमेंट किट) स्थापित है।
- एकीकृत विकास पर्यावरण (आईडीई) जैसे इंटेलीजे आईडीईए या एक्लिप्स।
-  जावा लाइब्रेरी के लिए Aspose.PSD। आप इसे यहां से डाउनलोड कर सकते हैं[यहाँ](https://releases.aspose.com/psd/java/).
## पैकेज आयात करें
सबसे पहले, आपको Aspose.PSD से आवश्यक पैकेज आयात करने होंगे:
```java
import com.aspose.psd.Color;
import com.aspose.psd.Graphics;
import com.aspose.psd.Image;
import com.aspose.psd.Pen;
import com.aspose.psd.Rectangle;
import com.aspose.psd.brushes.SolidBrush;
import com.aspose.psd.examples.Utils.Utils;
import com.aspose.psd.fileformats.psd.PsdImage;
import com.aspose.psd.imageoptions.BmpOptions;
```
## चरण 1: अपना जावा प्रोजेक्ट सेट करें
इससे पहले कि आप कोडिंग शुरू करें, सुनिश्चित करें कि आपका जावा प्रोजेक्ट निर्भरता के रूप में शामिल Aspose.PSD के साथ सही ढंग से सेट किया गया है।
## चरण 2: PsdImage का एक उदाहरण बनाएं
वांछित चौड़ाई और ऊंचाई के साथ PsdImage का एक नया उदाहरण आरंभ करें।
```java
Image image = new PsdImage(100, 100);
```
## चरण 3: ग्राफ़िक्स ऑब्जेक्ट को प्रारंभ करें
छवि के साथ काम करने के लिए ग्राफ़िक्स का एक उदाहरण बनाएं और प्रारंभ करें।
```java
Graphics graphics = new Graphics(image);
```
## चरण 4: ग्राफ़िक्स सतह साफ़ करें
ड्राइंग बनाने से पहले, ग्राफ़िक्स सतह को एक विशिष्ट रंग (वैकल्पिक) से साफ़ करें।
```java
graphics.clear(Color.getYellow());
```
## चरण 5: एक बिंदीदार दीर्घवृत्त बनाएं।
लाल रंग वाले पेन ऑब्जेक्ट का उपयोग करें और एक निर्दिष्ट आयत के भीतर एक बिंदीदार दीर्घवृत्त बनाएं।
```java
graphics.drawEllipse(new Pen(Color.getRed()), new Rectangle(30, 10, 40, 80));
```
## चरण 6: एक सतत दीर्घवृत्त बनाएं
एक ठोस नीले ब्रश से एक पेन ऑब्जेक्ट बनाएं और दूसरे आयत के भीतर एक सतत दीर्घवृत्त बनाएं।
```java
graphics.drawEllipse(new Pen(new SolidBrush(Color.getBlue())), new Rectangle(10, 30, 80, 40));
```
## चरण 7: छवि सहेजें
अंत में, उत्पन्न छवि को बीएमपी प्रारूप में एक निर्दिष्ट पथ पर सहेजें।
```java
String outputPath = "Your Document Directory/Ellipse.bmp";
BmpOptions saveOptions = new BmpOptions();
saveOptions.setBitsPerPixel(32);
image.save(outputPath, saveOptions);
```
