## นำเข้าเนมสเปซ

ในโปรเจ็กต์ .NET ของคุณ ให้นำเข้าเนมสเปซที่จำเป็นเพื่อเข้าถึงฟังก์ชันการทำงานของ Aspose.PSD เพิ่มบรรทัดต่อไปนี้ที่จุดเริ่มต้นของโค้ดของคุณ:

```csharp
using Aspose.PSD.FileFormats.Psd;
using Aspose.PSD.ImageOptions;
```

เมื่อคุณครอบคลุมข้อกำหนดเบื้องต้นแล้ว เรามาแบ่งตัวอย่างออกเป็นหลายขั้นตอนกัน

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "Your Document Directory";
```

 ให้แน่ใจว่าคุณเปลี่ยน`"Your Document Directory"` ด้วยเส้นทางจริงไปยังไดเร็กทอรีเอกสารของคุณ

## ขั้นตอนที่ 2: ระบุเส้นทางต้นทางและปลายทาง

```csharp
//ExStart:SavingtoDisk

string sourceFile = dataDir + @"sample.psd";
string destName = dataDir + "result.png";
```

 ที่นี่,`sourceFile` คือเส้นทางไปยังไฟล์ PSD ที่คุณต้องการประมวลผล และ`destName` คือเส้นทางปลายทางสำหรับภาพที่ได้

## ขั้นตอนที่ 3: โหลดและบันทึกรูปภาพ

```csharp
// โหลดรูปภาพ PSD และแทนที่แบบอักษรที่ไม่พบ
using (Image image = Image.Load(sourceFile))
{
    PsdImage psdImage = (PsdImage)image;
    psdImage.Save(destName, new PngOptions());
}
```

ข้อมูลโค้ดนี้จะโหลดรูปภาพ PSD แปลงเป็นรูปแบบ PNG และบันทึกลงในปลายทางที่ระบุ
